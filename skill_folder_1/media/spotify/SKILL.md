# Spotify Terminal Workflow

## Core controls
- Play, pause, next, previous
- Volume and device target

## Playlists
- Create and modify
- Add and remove tracks

## Search and queue
- Find by artist, album, track
- Queue management commands

## Teaching approach
- Control one playback session via terminal.
- Build one short playlist from search.
- Compare with local-first alternatives.
