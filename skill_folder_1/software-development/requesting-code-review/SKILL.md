# Requesting Code Review

## Core topics

1. PR quality
   - Description clarity
   - Linked issue and intent

2. Review process
   - Reviewer choice
   - Check semantics and behavior

3. Response
   - Accept feedback gracefully
   - Ask clarifying questions

## Teaching approach
- Rewrite one weak PR description.
- Identify appropriate reviewers.
- Practice responding to feedback.
