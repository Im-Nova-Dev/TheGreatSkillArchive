# Plan Mode

## Core activities

1. Capture intent
   - Goals and constraints
   - Known unknowns

2. Draft plan
   - Steps and order
   - Risks and mitigations

3. Review
   - Validate assumptions
   - Mark unclear decisions

## Teaching approach
- Write one plan for one coding task.
- Review plan completeness.
- Create plan checklist together.
