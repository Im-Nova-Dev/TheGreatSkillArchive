# Writing Plans

## Core topics

1. Planning fundamentals
   - Goal versus solution
   - Bite-sized tasks

2. Scope control
   - Boundaries and exclusions
   - Acceptance criteria

3. Sequencing
   - Dependencies and milestones
   - Checkpoints

## Teaching approach
- Convert one vague idea into concrete plan.
- Review one plan for missing risks.
- Refactor one plan to smaller steps.
