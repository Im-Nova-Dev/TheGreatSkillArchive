# Sessions & Profiles

```bash
hermes sessions list
hermes sessions browse
hermes sessions export OUT
hermes sessions rename ID T
hermes sessions delete ID
hermes sessions prune --older-than N days
hermes sessions stats

hermes profile list
hermes profile create NAME
hermes profile use NAME
hermes profile delete NAME
hermes profile show NAME
hermes profile alias NAME
hermes profile rename A B
hermes profile export NAME
hermes profile import FILE
```
