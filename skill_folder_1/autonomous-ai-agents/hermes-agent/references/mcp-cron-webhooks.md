# MCP, Cron, and Webhooks

```bash
hermes mcp serve
hermes mcp add NAME
hermes mcp remove NAME
hermes mcp list
hermes mcp test NAME
hermes mcp configure NAME

hermes cron list
hermes cron create SCHED
hermes cron edit ID
hermes cron pause/resume ID
hermes cron run ID
hermes cron remove ID
hermes cron status

hermes webhook subscribe N
hermes webhook list
hermes webhook remove NAME
hermes webhook test NAME
```
