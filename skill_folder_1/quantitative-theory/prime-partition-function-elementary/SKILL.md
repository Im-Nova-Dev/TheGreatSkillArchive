# Prime Partition Function — Elementary Asymptotic Analysis

Teach the 2026 elementary asymptotic analysis of the prime partition function pp(n) — the number of ways to write n as a sum of primes — by Cohen Antonir & Shapira (arXiv:2606.06068).

## Overview
- **Main Result**: log pp(n) ~ 2π √(n / 3 log n)
- **Method**: Three-step elementary recipe (R1–R3) adapted from Erdős's inductive method
- **Significance**: Avoids complex analytic machinery (circle method); short, adaptable proof

## Teaching Components
1. Definitions: pp(n), A-restricted partitions, plane partitions, d-power partitions
2. Three-step framework: recursion → exponential sum bound → elementary asymptotics
3. Upper/lower bound proofs for prime partitions
4. Extensions to d-power and plane partitions
5. Exercises with solutions
