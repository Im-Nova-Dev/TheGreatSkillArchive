# Unified Abstract Regularity Lemma

Teach the 2026 Carenini-Franchi unified abstract regularity lemma (arXiv:2606.06192) that recovers Szemerédi's graph regularity lemma, Green's arithmetic regularity lemma, and the Boolean function regularity lemma as direct corollaries of a single energy-increment argument. Covers the abstract framework (finitely additive measure spaces, local test families, refinement schemes with measurability hypothesis), the three specializations with explicit local test functions, the energy increment proof, and teaching exercises.

**Use when:** teaching regularity lemmas across graph theory, additive combinatorics, and Boolean function analysis; demonstrating how abstraction unifies disparate results; introducing energy-increment proofs; connecting Fourier-analytic, combinatorial, and analytic perspectives.

**Category:** quantitative-theory
