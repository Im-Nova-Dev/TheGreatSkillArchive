# Go Teaching & Learning Paths

Use this skill as the entrypoint for learner-centric Go teaching. For language internals, see `go-reference`; for testing, see `go-testing-deep-dive`; for concurrency pedagogy, see `go-concurrency-patterns-teaching`.

## Entry points
- Choose a learner level: beginner, intermediate, advanced.
- Load the matching teaching aid from `references/`.
- Use official sources: https://go.dev/doc/tutorial, https://go.dev/tour/, https://gobyexample.com/

## Rule of one
Add only material that teaches a learner’s next step. Do not overload this skill with full language reference.
