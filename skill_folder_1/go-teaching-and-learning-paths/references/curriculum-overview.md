# Curriculum Overview

## Beginner
Goals: install Go, run programs, understand basic types, functions, control flow, slices, structs, errors, interfaces, packages, and tests.

## Intermediate
Goals: modules and workspaces, methods, JSON, HTTP, struct embedding, context, testing table-driven style.

## Advanced
Goals: concurrency, profiling, fuzz, build tags, cross-compilation, project structure at scale.
