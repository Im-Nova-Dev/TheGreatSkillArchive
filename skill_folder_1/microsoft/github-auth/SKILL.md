# GitHub Auth

## Core topics

1. Methods
   - SSH keys
   - Personal access tokens
   - Device flow

2. Safety
   - Scope minimization
   - Rotation and revocation

## Teaching approach
- Generate one SSH key and add to GitHub.
- Create one scoped token.
- Rotate one credential safely.
