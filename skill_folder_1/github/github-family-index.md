## How to choose

- Start in `github-cli-teaching-core` when you need drills, demos, or beginner-to-advanced exercises.
- Start in `github-cli-knowledge-and-teaching` when you need cohort schedules, instructor scripts, forms, or logistics.
- Load both when designing a full course.
