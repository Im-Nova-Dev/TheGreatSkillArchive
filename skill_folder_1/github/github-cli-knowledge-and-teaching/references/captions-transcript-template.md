# Captions / Transcript Template

## Format
Speaker label + timestamped lines.

```
[00:00] Instructor: Welcome to session two.
[00:04] Instructor: Today we talk about branching.
[00:09] Learner: Is a branch the same as a fork?
[00:12] Instructor: Good question. A fork is…
```

## Tips
- Timestamps every 30 seconds minimum
- Indicate unclear audio with [inaudible]
- Include commands in code blocks inline
