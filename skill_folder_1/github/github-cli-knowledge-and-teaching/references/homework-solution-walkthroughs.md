# Homework Solution Walkthroughs

## Format
For each assignment, provide a clean walkthrough with narration.

### Example: PR Review Assignment
1. Clone repo and inspect logs.
2. `gh pr list` and choose a PR.
3. `git fetch` + branch out for review.
4. Run tests and read diff.
5. `gh pr review --approve` or `--request-changes`.
6. Write a comment tied to a specific line.

### Walkthrough Structure
- Objective
- Mistakes learners commonly make
- Correct solution step-by-step
- Reflection question at end
