# Course Wiki Setup Guide

## Platform
- GitHub Wiki (built-in) — easiest
- MkDocs / Docusaurus — for richer layouts

## Pages to Create
- Home: course overview and schedule
- Lessons: one page per session with notes
- FAQ: crowd-sourced Q&A
- Resources: curated links
- Alumni: contributions and spotlights

## Ownership
- Maintainer owns
- Co-instructors have write access
- Learners can suggest edits via PRs

## Content Rules
- Keep language simple and searchable
- Link to reference files in skill
- Update after each run with lessons learned
