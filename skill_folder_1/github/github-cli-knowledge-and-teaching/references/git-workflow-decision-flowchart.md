# Git Workflow Decision Flowchart

```
Start
 ↓
Working alone or with others?
 ├─ Alone → any workflow
 └─ With others
      ↓
How often do you release?
 ├─ Weekly → trunk-based + feature flags
 ├─ Monthly → GitHub Flow
 └─ Quarterly → Git Flow
```

## Notes
- Start simple. Add ceremony only when pain appears.
- The best workflow is the one your team actually follows.
