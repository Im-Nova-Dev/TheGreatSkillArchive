# Instructor Certification Test

Pass this before you teach the course.

## Part 1: Quiz
1. Explain the difference between `git reset --soft`, `--mixed`, and `--hard`.
2. What does `git switch` do that `git checkout` does not?
3. How do you recover a lost commit with `reflog`?
4. What is the difference between `git merge` and `git rebase`?
5. How do you squash the last 3 commits interactively?
6. What does `gh pr checkout` do?
7. How do you approve and request changes in one review?
8. When do you use `--allow-unrelated-histories`?

## Part 2: Live Demo
Teach the section “your first PR” from memory without notes.

## Part 3: Troubleshooting
A learner gets `fatal: refusing to merge unrelated histories`. Explain in 3 sentences what to ask before fixing it.

## Scoring
- 0-4: needs more practice
- 5-7: pass with conditions
- 8-10: certified
