# Course Completion Portfolio Guide

## Assemble
- One PR with review history
- One repo with clean commit history
- One automation workflow (Actions)
- One reflection document (markdown)

## Format
- GitHub Pages site
- LinkedIn post with screenshots
- Resume bullet:
  - “Shipped PRs and automation workflows using Git and GitHub from the terminal.”

## Review
- Instructor checks completeness
- Peer review optional but encouraged
