# Instructor Quick-Start Guide

Get ready to teach in 5 minutes.

## Minute 0-1
Open terminal. Confirm `git --version` and `gh --version` work.

## Minute 1-2
Clone the class sample repo:
```
git clone <sample-repo-url>
cd sample-repo
```

## Minute 2-3
Run the scaffold script:
```
bash references/lesson-scaffolding-script.sh
```

## Minute 3-4
Open your preferred editor in the repo directory.

## Minute 4-5
Run `git status` to confirm the demo environment is clean.
