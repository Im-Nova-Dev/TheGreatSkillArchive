# Multilingual Command Cheat Sheet

## English (Baseline)
- `git status` — show repo state
- `git add -A` — stage all
- `git commit -m "msg"` — commit
- `git push` — upload

## Translations (concept only)
| English Term | Plain Meaning |
|--------------|---------------|
| checkout | switch to a version |
| stage | prepare to save |
| commit | save |
| push | upload |
| pull | download + merge |

## Usage
- Keep commands in English
- Translate explanations in lessons
- Glossary in `glossary-terms.md` can be localized
