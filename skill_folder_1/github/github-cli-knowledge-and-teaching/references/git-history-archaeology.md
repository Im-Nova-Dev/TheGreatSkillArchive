# Git History Archaeology Activity

## Objective
Use Git’s built-in time-travel tools to reconstruct what happened.

## Stations
1. **Time Travel**: `git log`, `git show`, checkout old commits
2. **Who Did It**: `git blame`
3. **Lost and Found**: `git reflog` to recover deleted branch
4. **Mystery Bug**: `git bisect` to find the breaking commit

## Story Arc
- “A bug appeared in v0.3.0. Use bisect to find the commit that introduced it.”
- “A teammate deleted the branch. Recover the commits.”
- “Find who changed this line and why.”

## Deliverable
Each learner writes a 3-sentence archaeology report.
