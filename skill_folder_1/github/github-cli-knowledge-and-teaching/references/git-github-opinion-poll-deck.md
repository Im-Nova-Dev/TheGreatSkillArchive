# Git/GitHub Opinion Poll Deck

Use at the start to gauge attitudes.

1. “I feel comfortable opening a terminal.” Strongly agree → Strongly disagree
2. “Merge conflicts are manageable.”
3. “I understand what `git rebase` does.”
4. “I would review a stranger’s PR.”
5. “I know what GitHub Actions are for.”

## Follow-Up
Discuss surprising results. Use as baseline for confidence tracking.
