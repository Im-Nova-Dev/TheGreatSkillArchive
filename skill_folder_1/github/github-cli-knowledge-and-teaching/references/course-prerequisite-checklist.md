# Course Prerequisite Checklist

Before starting, each learner should have:

- [ ] Basic computer literacy (files, folders, copy/paste)
- [ ] Access to a terminal (Linux, macOS, or Windows WSL/Git Bash)
- [ ] A GitHub account created
- [ ] Permission to install software on their machine
- [ ] Stable internet connection
- [ ] A text editor installed (nano, vim, VS Code, etc.)
- [ ] Willingness to type commands and read error messages

## Optional but helpful
- Prior experience with any command line
- Familiarity with markdown

## Assessment tip
Run the `beginner-readiness-test.md` to score readiness.
