# Pair Programming Rubric

## Driver
- [ ] Types commands as navigator instructs
- [ ] Asks when unsure
- [ ] Switches roles when prompted

## Navigator
- [ ] Guides without grabbing the keyboard
- [ ] Explains why, not just what
- [ ] Catches mistakes early

## Both
- [ ] Understand the final solution
- [ ] Can repeat the exercise solo later
- [ ] Give each other one piece of feedback
