# Instructor Shadowing Guide

For new instructors learning to teach the course.

## Before the Session
- Read lesson plan and reference files
- Run all demo commands on a clean machine
- Check sample repo URL

## During the Session
- Observe pacing, tone, and questions
- Note where learners struggle
- Do not interrupt the lead instructor

## After the Session
- Write 5 notes: what worked, what to change, 3 learner questions
- Practice one demo segment you will teach next time
- Submit shadowing log to curriculum lead

## First Solo Teach Checklist
- [ ] Taught at least 3 sessions as assistant
- [ ] Passed instructor certification test
- [ ] Demo commands prepared and tested
- [ ] Backup repo ready
- [ ] Chat / questions protocol reviewed
