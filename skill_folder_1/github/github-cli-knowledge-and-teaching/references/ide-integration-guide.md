# IDE Integration Guide

For learners who use VS Code, JetBrains, or other editors alongside the terminal.

## VS Code
- Source Control tab: visual status, stage, commit
- Keyboard shortcuts: `Ctrl+Shift+G`
- Integrated terminal: run `git` commands inline
- Extensions: GitLens for blame/heatmap

## JetBrains
- VCS operations from menus and shortcuts
- Git tool window
- Built-in log viewer and merge tool

## Bridging IDE and CLI
- Both use the same `.git` folder
- You can switch freely between UI and terminal
- Pro tip: do exercises in terminal, then verify in UI

## Discussion
When do GUI tools help, and when does CLI win?
