# Git Command Lineage Poster

How Git terminology evolved.

| Old Tool | Git Equivalent |
|----------|----------------|
| CVS / SVN checkout | git clone |
| SVN update | git pull |
| SVN commit | git commit |
| Perforce branch | git branch |
| Diff tool | git diff |
| Blame | git blame |
| Histories | git log |
| Tag | git tag |

## Narrative
“Git didn’t invent these ideas. It just made them local, fast, and branching by default.”
