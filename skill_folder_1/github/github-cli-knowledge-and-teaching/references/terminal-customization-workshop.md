# Terminal Customization Workshop

## Goals
Make your terminal pleasant to use for Git work.

## Agenda
- Prompt customization (PS1 / Starship / Oh My Zsh)
- ls colors and `exa` / `eza`
- Aliases recap + add your own
- Git-aware prompt: show branch and dirty state
- Kitty / Alacritty font and theme tips

## Hands-On
1. Set a Git-aware prompt.
2. Add 2 personal aliases.
3. Choose a color scheme that is easy on the eyes.

## Bonus
Show `dircolors` and Git integration for ls.
