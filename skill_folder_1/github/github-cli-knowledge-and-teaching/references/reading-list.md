# Git and GitHub Reading List

Recommended progression for students who want to read more.

## Beginner

- Pro Git (Chacon and Straub), Chapters 1–4
- GitHub Docs: “About GitHub CLI”
- Atlassian Git Tutorial: “Basic Git commands”
- git-scm.com Book: Getting Started

## Intermediate

- Pro Git, Chapters 5–9
- Git from the Bottom Up (John Wiegley)
- GitHub Blog: “How the GitHub CLI can now enable triangular workflows”
- Advanced Git commands and workflows guides

## Advanced

- Pro Git, Chapters 10–12
- Git Internals: blob, tree, commit, object model
- GitHub Actions documentation
- GitHub REST API reference
- GraphQL API documentation

## Video and Talks

- Git Tutorial for Beginners (YouTube)
- Git Fundamentals: Staging area, repository, working directory
- GitHub Skyline CLI extension walkthrough
- Advanced workflow automation with GitHub Actions

## Practice

- Use `pygount` or similar tools to inspect repo sizes
- Read and explain real GitHub Actions workflows
- Practice recursive experimentation with test repos
