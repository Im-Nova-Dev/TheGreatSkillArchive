# Git Show-and-Tell Lesson Script

A portable demo for teaching Git to a new student in 20 minutes.

## Props
- Terminal with Git and `gh`
- A GitHub account
- A note card with branch names and commit hashes

## Demo

1. Initialize repo
2. Create README, stage, commit
3. Show `git status` after each step
4. Create branch, edit file, commit
5. Show `git log --graph --oneline --all`
6. Push with `gh repo create`
7. Open PR, review, merge
8. Show `gh browse` to open the merged PR in the browser

## Narration One-Liners

- “Git status is your map.”
- “Commit = autosave for the whole project.”
- “Branch = alternate timeline.”
- “Push = send local history to GitHub.”
- “PR = request to merge + review surface.”

## Common Questions to Prompt

- “What if I want two fixes in separate commits?”
- “What happens when another person pushes?”
- “Why does a branch have a long name?”
