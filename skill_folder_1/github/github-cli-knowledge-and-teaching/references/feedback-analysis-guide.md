# Feedback Analysis Guide

## Quantitative
- Compute NPS = %Promoters - %Detractors
- Calculate average confidence scores pre/post
- Compare completion rates by session format

## Qualitative
Themes to extract from open responses:
- “I still don’t understand X” → revisit next cohort
- “The exercises helped most” → keep them
- “Too fast / too slow” → adjust pacing
- “I want more real PRs” → expand capstone

## Action Template
| Theme | Evidence | Action |
|-------|----------|--------|
| Pace too fast | 5 comments | Cut exercise volume by 20% |
| Conflicts unclear | 8 comments | Add dedicated conflict session |
