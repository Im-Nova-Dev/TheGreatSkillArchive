# Pair Debugging Exercise

## Setup
One person narrates error output. The other types without looking at the error first.

## Scenario A
`git push` rejected. Fix without telling the driver the cause.

## Scenario B
Repo shows both staged and modified for same file. Diagnose.

## Scenario C
Conflicts in 3 files. Resolve by talking only.

## Debrief
- What communication patterns helped?
- When did you want to take the keyboard?
