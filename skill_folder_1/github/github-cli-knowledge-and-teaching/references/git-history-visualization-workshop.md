# Git History Visualization Workshop

## Goal
Read repo history visually.

## Commands
- `git log --oneline --graph --all`
- `git log --graph --decorate`
- `gitk` or `tig` if available

## Exercises
1. Draw the latest history from a given repo.
2. Predict the result of a merge before running it.
3. Identify where a branch diverged.

## Discussion
What does a healthy history look like?
