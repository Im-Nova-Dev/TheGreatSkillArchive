# Course Trailer Script (60 seconds)

0-10s: Hook
> “What if you could undo almost any mistake in your work?”

10-25s: Problem
> “Most developers fear the terminal. They copy-paste commands they don’t understand. Merge conflicts feel like disasters.”

25-40s: Solution
> “This course teaches Git and GitHub the way they were meant to be used: from the command line. You’ll type every command, break things, fix them, and ship real pull requests.”

40-50s: Social proof
> “Past learners: ‘I finally understand rebase.’ ‘I contributed to open source.’”

50-60s: CTA
> “Enroll now. Stop being afraid of the terminal.”
