# Terminal Text Editor Crash Course

## nano
- Easy for beginners
- Shortcuts shown at bottom
- `Ctrl+O` save, `Ctrl+X` exit

## vim
- Steeper curve, ubiquitous
- Modes: normal, insert, visual
- `i` insert, `Esc` normal, `:wq` save and quit

## VS Code (in terminal)
- `code file` opens in VS Code
- `code .` opens folder
- Integrated terminal inside editor

## Exercise
Open a file in each editor. Add a line. Save and quit.

When would you use each?
