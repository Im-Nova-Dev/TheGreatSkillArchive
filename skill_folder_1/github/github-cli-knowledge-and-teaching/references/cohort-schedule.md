# Git and GitHub Cohort Schedule — 4 Weeks

Assumes two 90-minute sessions per week.

## Week 1

Session 1: Git mental model, init, add, commit, status, log
Session 2: Working tree, staging area, branches, switch, log graph

Homework:
- create three repos
- make five commits across two branches
- run `git log --graph --oneline --all`

## Week 2

Session 1: local merge, fast-forward, merge commit, conflict basics
Session 2: clone, remote, origin, push, pull, fetch

Homework:
- push repo to GitHub
- open a PR from a branch
- merge it

## Week 3

Session 1: PR lifecycle, review, inline comments, approve/request changes
Session 2: CI reading, Actions basics, recovery with reflog

Homework:
- make a PR with an intentional style issue
- receive and act on review feedback

## Week 4

Session 1: triangular workflow, fork, upstream sync
Session 2: releases, tags, final project

Homework:
- contribute to a public repo
- publish a small release

## Progression Checkpoints

Use rubric or warm-up quiz each session.
Final assessment: learner can complete an unassisted feature branch -> PR -> merge cycle.
