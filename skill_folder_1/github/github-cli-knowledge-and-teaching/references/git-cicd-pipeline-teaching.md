# Git + CI/CD Pipeline Teaching Sequence

## Order
1. Push triggers GitHub Actions
2. Workflow file syntax (YAML)
3. Steps: checkout → install → test → deploy
4. Status checks shown on PRs
5. Branch protection requiring checks

## Demo
- Open a PR and watch CI run via `gh run list`
- Break a test and see red status
- Fix, push, see green status
