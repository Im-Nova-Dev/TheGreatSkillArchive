# Proctored Exam Script

## Setup
- 60-minute timed session
- Learners in separate breakout rooms or spaced seats
- Instructor shares problem set via private repo

## Exam Structure
1. **Part A: Diagnosis** (20 min)
   Show 5 repo states. Learner describes what happened and the fix.

2. **Part B: Commands** (25 min)
   Execute 8 tasks in a disposable sandbox: branch, merge, conflict, push, PR.

3. **Part C: Explanation** (15 min)
   Oral or written: explain `git rebase` in 3 sentences.

## Rules
- No notes beyond command reference sheet
- Screen share if remote; instructor monitors
- One question clarification per student
