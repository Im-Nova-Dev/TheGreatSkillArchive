# Command-Line Hero Arc

A learner journey handout showing transformation.

## Act 1: The Uncertain Beginner
- Sees terminal as scary
- Types `git status` and freezes
- Thinks “I broke it”

## Act 2: The Curious Apprentice
- Learns staging area
- Creates first branch without asking
- Edits commit messages with `--amend`

## Act 3: The Confident Contributor
- Opens PR without trembling
- Resolves conflicts solo
- Reviews a peer PR

## Act 4: The Terminal Hero
- Customizes prompt and aliases
- Writes a hook
- Mentors someone else

## Moral
“The terminal doesn’t care how you feel. It only cares what you type. And what you type gets better with practice.”
