# Syllabus Templates

## Syllabus A — 4-Week Intensive

- Week 1: Git fundamentals + exercises
- Week 2: Branching and merging
- Week 3: Remotes, forks, GitHub collaboration
- Week 4: PR workflows, Actions, capstone

## Syllabus B — Weekend Sprint (2 Days)

Day 1 AM: Git basics Day 1 PM: Branching
Day 2 AM: Remotes + PR Day 2 PM: Capstone + review

## Syllabus C — Self-Paced Online

- Module 1: Install + first commits
- Module 2: Branches
- Module 3: Remote workflow
- Module 4: Pull requests
- Module 5: Automation
- Module 6: Final project

## Syllabus D — Designer / Non-Dev Track

- Session 1: Git as “text history”
- Session 2: PR as “design review”
- Session 3: Issues as “tickets”
- Session 4: Actions as “publish preview”
