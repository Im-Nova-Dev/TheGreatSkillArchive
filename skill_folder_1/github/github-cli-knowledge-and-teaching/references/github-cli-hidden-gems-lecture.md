# GitHub CLI Hidden Gems Lecture

## Demo Pack
1. `gh pr checks` — see CI status without leaving terminal
2. `gh pr diff` — inspect PR diff locally
3. `gh issue develop` — create branch off issue
4. `gh repo view --web` — open repo in browser instantly
5. `gh alias set` — create reusable shortcuts
6. `gh gist create` — share snippets
7. `gh secret set` — add secrets without web UI
8. `gh release list` — quick audit of releases
9. `gh extension` — extend gh with plugins

## Teaching Notes
- Show where each saves 3+ clicks
- Ask learners to add their own hidden gem
