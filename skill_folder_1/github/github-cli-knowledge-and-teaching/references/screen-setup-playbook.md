# Screen Setup Playbook

## Terminal
- Font: 14pt minimum (JetBrains Mono, Fira Code, or monospace)
- Theme: dark background with high contrast
- Cursor: block, blinking
- Line height: comfortable, not tight

## Shell
- Prompt shows: path, branch, dirty state
- History size >= 10,000
- Aliases for common git commands

## Multiplexer
- tmux / zellij recommended
- Split panes for repo + tests + notes

## Editor
- Same font family as terminal for consistency
- Line numbers on
- Git integration enabled
