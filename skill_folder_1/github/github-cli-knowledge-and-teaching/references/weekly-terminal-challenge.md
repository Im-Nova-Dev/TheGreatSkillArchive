# Weekly Terminal Challenge

A recurring 20-minute challenge for the week after session.

## Week 1 — Git Minidrama

You’ve just initialized a new project. In 5 commands, create: a README, an initial commit, a feature branch, two commits there, then merge it back with a merge commit.

## Week 2 — Conflict Theater

Open a PR with an intentional conflict and resolve it only via terminal (no resolution UI).

## Week 3 — PR Review Slam

Review a PR using `gh pr review` — leave one approval and one change request. Respond with `gh pr comment`.

## Week 4 — Automation Day

Trigger a GitHub Actions workflow manually with `gh run workflow …` and inspect results.

## Prizes/Recognition

- Fastest valid solution
- Cleanest commit message
- Least amount of swearing on the terminal
