# Instructor Burnout Prevention

## Signs
- Ending sessions drained, not energized
- Dreading learners’ questions
- Falling behind prep

## Prevention
- Teach no more than 2x/week
- Record all sessions; reuse good runs
- Batch prep: create all demos in a single day
- Rotate co-instructors if possible
- Ask for feedback: “What can make this easier for me?”

## Recovery
- Skip one session if needed (find substitute)
- Revisit old recordings to remember impact
- Set OOO hours: no Q outside teaching time
