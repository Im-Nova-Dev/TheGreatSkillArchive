# Remote vs In-Person Workshop Adaptation

## Shared Requirements
- All learners have same tools installed
- Clear audio / video quality
- Instructor can see all learner terminals when possible

## In-Person Advantages
- Co-located help for hardware issues
- Body language cues for confusion
- Shared pasteable commands on projector
- Better whiteboard/diagram drawing

## Remote Advantages
- Learners watch your exact terminal settings
- Chat log captures commands verbatim
- Screen-share + spotlight keeps focus
- Recording available for playback

## Remote Adjustments
- Use breakout rooms for pair exercises
- Type commands into shared doc at same time as terminal
- Poll / emoji check-ins for comprehension
- Shorter intervals: 20 min talk, 10 min breakout

## Hybrid Adjustments
- Treat remote learners as primary, in-person as studio audience
- Repeat questions from in-person for remote side
- Have a dedicated chat moderator for remote Q
