# Terminal Shortcut Challenge Game

## Format
Instructor names a task; learners race to type the shortest command.

## Challenges
1. “List all files including hidden ones.” (`ls -a`)
2. “Go to your home directory.” (`cd ~` or `cd`)
3. “See what’s in the previous directory.” (`ls ..`)
4. “Create three nested directories at once.” (`mkdir -p a/b/c`)
5. “Copy file1 to file2.” (`cp file1 file2`)
6. “Find all `.md` files.” (`find . -name "*.md"`)
7. “Count lines in a file.” (`wc -l file`)

## Scoring
- Shortest correct command wins
- Ties broken by readability / explicitness
