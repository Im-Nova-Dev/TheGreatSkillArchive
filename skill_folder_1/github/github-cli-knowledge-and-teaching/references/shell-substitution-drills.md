# Shell Substitution Drills

Warm-up exercises to make the shell feel friendly.

1. `echo $(date)` — inject command output
2. `echo ${USER}` — variable expansion
3. `echo *` — globbing
4. `echo ~` — home shortcut
5. `echo .` vs `echo ..` — current and parent
6. `ls -d */` — list directories only
7. `cp file{,.bak}` — brace expansion

## Teaching Tip
Start with `echo` so learners see results without risk.
