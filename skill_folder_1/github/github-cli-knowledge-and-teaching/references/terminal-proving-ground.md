# Terminal Proving Ground

## Sandbox Challenges
Run each in a disposable repo created by `lesson-scaffolding-script.sh`.

1. Create a branch, add a file, commit, merge.
2. Create a merge conflict and resolve it.
3. Rename a branch and push it.
4. Undo a commit while keeping the work.
5. Cherry-pick one commit from another branch.
6. Reflog recovery: delete a branch, recover it.

## Completion
- Each challenge = one stamp in passport
- Three incorrect attempts = hint from instructor
