# GitHub CLI Alias Building Guide

## Why
Save keystrokes for common `gh` commands.

## Examples
```
gh alias set issueslist 'issue list --label "good first issue"'
gh alias set prview 'pr view --web'
gh alias set prready 'pr review --approve'
```

## Activity
Add three aliases that fit YOUR workflow.
