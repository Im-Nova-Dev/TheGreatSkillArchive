# Alumni Network Setup Guide

## Purpose
Keep graduates connected and practicing after the cohort ends.

## Setup
1. **Community Chat**: Slack or Discord workspace
   - Channels: general, contributions, jobs, tips
2. **GitHub Org**: Create an org for alumni.
   - Each member gets a personal repo sandbox.
   - Shared public repos for open-source practice.
3. **Weekly Digest**
   - One Git tip
   - One contribution opportunity
   - One member spotlight
4. **GitHub Circle**: Run a monthly contributor call.
5. **Badging**: Award digital badges via repo README additions.
6. **Contribution Calendar**: Track merged PRs of alumni on a public calendar.

## Retention Ideas
- Hack days (competitively harmless)
- “Explain Like I’m Five” posts channel
- Pair programming biweekly
- Interview prep sessions
- Book club (Git Pro book)
