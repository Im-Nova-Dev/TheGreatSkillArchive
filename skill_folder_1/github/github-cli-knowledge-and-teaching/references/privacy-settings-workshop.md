# Privacy Settings Workshop

## Goal
Make learners aware of what they expose and how to control it.

## Agenda
1. Profile visibility (public / private)
- 2. Email privacy on GitHub
- 3. Contribution privacy
- 4. Private vs public repos
- 5. SSH key exposure considerations
- 6. Audit log and sessions

## Exercise
Check each setting and write a one-line note on what they changed.
