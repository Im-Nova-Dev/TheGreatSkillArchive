# Git Quiz Bowl

## Format
Teams compete with buzzers or raised hands.

## Sample Questions
1. “What command shows staged vs unstaged files?” (`git status`)
2. “What creates a new branch?” (`git switch -c` or `git branch`)
3. “How do you see last commit only?” (`git log -1`)
4. “What moves commits from one branch to another safely?” (`git cherry-pick`)
5. “What lists remote names?” (`git remote -v`)

## Rules
- 10 seconds to answer
- Bonus: explain why the answer is correct
- Cheat sheet not allowed during bowl
