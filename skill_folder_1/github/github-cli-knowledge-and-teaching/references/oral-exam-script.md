# Oral Exam Script

## Setup
5-10 minute 1-on-1 with each learner at course end.

## Questions (Pick 5)
1. “Walk me through what happens when you type `git push`.”
2. “What does `git status` show when a file is staged?”
3. “What is a merge conflict? What do you do first?”
4. “Why would you use `git rebase` instead of `git merge`?”
5. “Your teammate force-pushed. What now?”
6. “How do you undo a commit without losing the changes?”
7. “What is the difference between `git pull` and `git fetch`?”
8. “What is GitHub Actions?”

## Scoring Rubric
- 0-2: needs more practice
- 3-4: competent
- 5: ready to teach others
