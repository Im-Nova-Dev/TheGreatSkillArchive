# Expanded PR Grading Rubric

## Dimensions
1. **Purpose Clarity** (1-4)
2. **Description Quality** (1-4)
3. **Commit Structure** (1-4)
4. **Tests / Verification** (1-4)
5. **Review Responsiveness** (1-4)
6. **Docs / Comments** (1-4)

## Scoring
- 1: missing or harmful
- 2: present but weak
- 3: solid, minor gaps
- 4: exemplary, easy to review and merge

## Instructor Notes
- Focus feedback on the weakest dimension, not all at once
- Celebrate strong commits explicitly
- Use “reviewer perspective” framing
