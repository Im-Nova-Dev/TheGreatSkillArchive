# Certificate & Badge Designs

## Certificate Text

> THIS CERTIFIES THAT
> ___{Learner Name}___
> successfully completed the
> GitHub in the Terminal course
> demonstrating competency in Git workflows,
> pull requests, and PR reviews from the CLI.
> Date: ___{Date}___. Instructor: ___{Instructor}___.

## Badge Conditions

- **Git Foundations**: created, staged, committed, pushed without assistance.
- **Conflict Resolver**: resolved a merge conflict independently.
- **PR Pilot**: opened, reviewed, and merged a PR via terminal.
- **Automation Explorer**: triggered and read a GitHub Actions run.

## SVG Outline (Badge)
- Circle outline, 12-point “Git” icon or terminal prompt
- Three small stars representing the badges above

## ASCII Badge Alternative

   _______
  /       \   [GIT FOUNDATIONS]
