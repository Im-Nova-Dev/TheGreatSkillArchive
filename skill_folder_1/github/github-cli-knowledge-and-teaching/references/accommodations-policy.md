# Accommodations Policy

## Commitment
We strive to make every session accessible.

## Requests
- DM instructor ahead of time with specific needs
- We will try to meet each request within reason

## Common Accommodations
- Extended time on exercises
- Captions / interpreter for live sessions
- Large-print handouts
- Breaks every 30 minutes if needed
- Alternate pairing arrangements

## Equipment
- Loaner laptops available if needed
- Quiet breakout rooms for focus
