# Git Aliases Competition

## Goal
Design the most useful or creative Git alias.

## Rules
- Must be a valid alias (`git config --global alias.XXX ...`)
- Must have a clear practical use case
- Optional: combine with `gh` for bonus points

## Categories
- Most Practical
- Most Creative
- Best Combo (`git` + `gh`)
- Most Beginner-Friendly

## Sample Aliases
- `git config --global alias.wip "commit -m 'wip' --no-verify"`
- `git config --global alias.prune "fetch --prune"`
- `git config --global alias.last "log -1 HEAD"`

## Judging
Peer vote + instructor tiebreak.
