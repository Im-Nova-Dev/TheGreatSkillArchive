# Pairing Rotation Schedule

## Purpose
Ensure learners pair with multiple partners over the course.

## Format
Print or share a grid.

| Day | Pair A | Pair B |
|-----|--------|--------|
| 1 | 1-2 | 3-4 |
| 2 | 1-3 | 2-4 |
| 3 | 1-4 | 2-3 |
| 4 | 2-3 | 1-4 |
| 5 | 2-4 | 1-3 |
| 6 | 3-4 | 1-2 |

## Rules
- Switch roles halfway (driver → navigator)
- Both must explain the solution before moving on
- Report back to the group: “one thing we learned together”
