# Git Vocabulary Bingo

## How to Play
- Distribute 5x5 grids with Git terms
- Instructor calls definitions or scenarios
- First to get bingo explains the term to win

## Variations
- No caller — learners draw cards and act out definitions
- Teams of two: one defines, one guesses
