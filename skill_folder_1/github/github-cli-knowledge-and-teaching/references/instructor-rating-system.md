# Instructor Rating System

## Peer Review
After each session, a co-instructor observes and rates:
- Clarity 1-5
- Pacing 1-5
- Learner engagement 1-5
- Technical accuracy 1-5

## Learner Reviews
Post-session anonymous survey (3 questions):
1. What’s one thing that worked?
2. What’s one thing to change?
3. Any confusion or frustration?

## Self-Assessment
Instructor writes 3 notes before and after each session.

## Calibration
Monthly: instructors compare ratings and align standards.
