# Git Workflow Comparison Chart

| Feature | GitHub Flow | Git Flow | Trunk-Based |
|---------|-------------|----------|-------------|
| Branches | Short-lived feature branches | Long-lived develop | Short-lived + feature flags |
| Main branch deployable | Always | Yes (release branches separate) | Yes |
| Release cadence | Continuous | Scheduled | Continuous |
| Best for | Small teams, SaaS | Large teams, scheduled releases | High-frequency deploys |

## Teaching tip
Let learners try each in a sandbox for 10 minutes, then vote on favorite.
