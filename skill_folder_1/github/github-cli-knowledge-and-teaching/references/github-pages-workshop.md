# GitHub Pages Workshop

## Concept
Host a site directly from a repo.

## Sources
- Docs in `/docs` folder on main
- Custom workflow deploying built assets
- Branch `gh-pages` as source

## Commands
- Create `docs/index.html`
- Settings → Pages → Source: `main` → `/docs`

## Demo
1. Add `docs/index.html`
2. Commit and push
3. View site at `https://<user>.github.io/<repo>`
