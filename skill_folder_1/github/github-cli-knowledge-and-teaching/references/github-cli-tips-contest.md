# GitHub CLI Tips Contest

Format: 15-minute in-class contest.

## Rules
- Use only `gh` — no web browser
- Each tip earns points
- Fastest unique tip wins tiebreaker

## Tip Bank
- `gh pr create --title "…" --body "…"`
- `gh pr view --web` to open in browser
- `gh issue list --label "good first issue"`
- `gh run list --workflow …`
- `gh api repos/{owner}/{repo}/releases`
- `gh pr checkout <id>`
- `gh pr review --approve`
- `gh pr merge --squash --delete-branch`
- `gh gist create`
- `gh auth status`
- `gh secret set`
- `gh label list`
- `gh repo fork --clone`
- `gh alias set`
- `gh extension list`

## Scoring
- 1 pt per tip
- 2 pts if it includes a flag or pipe
- 3 pts if it replaced a previously common multi-step process
