# Async Video Recording Scripts

## Episode 1: “What Did I Just Do?”
- Angle: terminal + mic narration
- Runtime: ~5 minutes
- Script:
  1. “When I type `git status`, Git is telling me what my project is thinking.”
  2. Show unstaged file → stage → commit → log.
  3. “This is the Git rhythm.”
  4. Speed-run a typical editing → commit loop.

## Episode 2: “Branching in 90 Seconds”
- Show branch diagram BEFORE and AFTER.
- Use `git switch -c`, two commits, then `git merge`.
- Commentary: “Branching is free. Merging is a promise.”

## Episode 3: “Merge Conflict Demystified”
- Inject conflict intentionally.
- Read both sides of the conflict markers.
- Resolve on screen, commit.
- Key teaching line: “Conflicts are a normal conversation.”

## Episode 4: “Pull Request Workflow”
- gh CLI screencast: `gh pr create` → `gh pr view` → check CI → `gh pr merge -s`.
- Vocalize the checklist.

## Production Notes
- 720p minimum, terminal font 14pt+
- Dark terminal theme for visual consistency
- Include 1 captioning option if possible
