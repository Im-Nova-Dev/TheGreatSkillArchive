# Course Localization Guide

## Priorities
- Translate concept explanations only, not jokes
- Keep commands in English (universal in coding)
- Localize examples to local open-source communities when possible

## Translation Guidance
- Consistent terminology: match original glossary
- Keep markdown structure identical
- Flag culturally-specific references

## Checklist
- [ ] `glossary-terms.md` translated
- [ ] Slide text translated
- [ ] Quiz strings localized
- [ ] Instructor notes in original or target language
