# Course Intro Script

## Opening (5 min)

> “Welcome. By the end of this course you will not just know Git commands — you will think in Git. You’ll look at a project and see branches, commits, and PRs as a story. Let’s start with why version control exists: because text changes over time, and you deserve to remember every version.”

## Icebreaker
Ask: “What’s the most confusing Git moment you’ve had so far?” Write answers anonymously; use them as recurring examples.

## Course Map
- Week 1: Git foundations
- Week 2: Branching
- Week 3: GitHub collaboration
- Week 4: Pull requests and automation

## Rules of Engagement
- Hands-on: you type, I narrate
- Questions stop the class
- Pair exercises every session
- No “git experts” flexing
