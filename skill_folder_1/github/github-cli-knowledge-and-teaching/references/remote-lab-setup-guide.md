# Remote Lab Setup Guide

For learners whose local machine can’t install Git or the terminal.

## Option 1: GitHub Codespaces
- Free tier available
- Preinstall Git and VS Code in browser
- Run exercises directly in the browser terminal

## Option 2: GitPod
- Open repo in GitPod
- Preconfigured workspace

## Option 3: Replit
- Create a “Bash” repl
- Run `git` commands in the shell pane

## Option 4: Shared JupyterHub Terminal
- If your organization runs JupyterHub, launch a terminal session

## Option 5: Instructor-Hosted SSH
- Instructor provides temporary SSH access to a machine with Git
- Works for corporate / school networks with restrictions

## Setup Checklist for Remote Learners
- [ ] Log into chosen platform
- [ ] Configure `git config --global user.name` and `user.email`
- [ ] Test `git init` and `git commit -m "hello"`
- [ ] Clone the class sample repo
- [ ] Verify commands in lesson plan match the environment
