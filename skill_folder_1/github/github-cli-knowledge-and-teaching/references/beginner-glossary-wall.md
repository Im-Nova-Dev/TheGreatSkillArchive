# Beginner Glossary Wall

## Print or Project

| Term | Simple Definition |
|------|-------------------|
| **Repository** | A project folder with history |
| **Commit** | A saved snapshot |
| **Branch** | A parallel timeline |
| **Merge** | Combining timelines |
| **Conflict** | Git needing human help |
| **Remote** | A version hosted online |
| **Clone** | Download a remote repo |
| **Pull** | Download and update |
| **Push** | Upload your commits |
| **PR** | Proposal to change code |
| **HEAD** | “You are here” marker |
| **Staging area** | Your packing box before shipping |
| **Working tree** | The files you see |
| **Fork** | Your personal copy |
| **GitHub Actions** | Automated CI |
| **Release** | A snapshot ready for users |
| **Upstream** | The original repo |
| **Origin** | Your clone’s default remote |
| **Tag** | A bookmark |
| **Diff** | A before/after comparison |
