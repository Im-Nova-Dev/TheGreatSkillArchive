# Live Captioning Workflow

## Goal
Make live sessions accessible.

## Tools
- OBS + VTT caption generator
- Zoom / Teams built-in captions
- Manual backup: Google Docs voice typing

## Procedure
1. Start captions at beginning of session.
2. Assign a chat moderator to fix obvious typos.
3. Save transcript after session.
4. Upload alongside video.

## Instructor Notes
- Speak slightly slower than normal
- Pause for questions before moving on
- Repeat learner questions verbatim so captions capture them
