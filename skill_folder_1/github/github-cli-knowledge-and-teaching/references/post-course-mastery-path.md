# Post-Course Mastery Path

## After the Basics
- Interactive rebase fluency
- `git bisect` and debugging workflows
- Large repo strategies (partial clone, sparse checkout)
- Submodules and subtree basics

## GitHub Advanced
- Actions matrices and caching
- Dependabot and security advisories
- GitHub Apps and webhooks
- GitHub CLI extensions

## Community & Career
- Maintain a blog on a Git topic
- Contribute to one open-source project per month
- Give a talk or write a post teaching something you learned
- Join GitHub Community Exchange or local Meetup

## Suggested Reading
- Pro Git book (free online)
- Flight rules for Git
- GitHub skills paths
