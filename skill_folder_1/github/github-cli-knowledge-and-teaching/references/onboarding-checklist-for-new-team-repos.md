# Onboarding Checklist for New Team Repos

## First-Time Setup
- [ ] Clone repo: `git clone <url>`
- [ ] Read README and CONTRIBUTING
- [ ] Run setup / install
- [ ] Run tests

## Git Config
- [ ] Set user.name and user.email
- [ ] Configure signing key if required
- [ ] Set credential helper

## GitHub Setup
- [ ] Join org team
- [ ] Enable branch protection rules on main
- [ ] Understand CODEOWNERS and review requirements

## First Contribution
- [ ] Assign yourself an issue
- [ ] Create a branch with issue number
- [ ] Open a PR early (WIP is okay)
- [ ] Request reviews
- [ ] Merge when approved
