# Course Readiness Self-Survey

## Before You Start

1. Can you open a terminal and run a command? (yes/no)
2. Have you typed `git status` before? (yes/no)
3. Do you know what a branch is? (yes / vague / no)
4. What scares you most about Git?
5. What do you hope to build / contribute to?

## Scoring Guide
- 0-1 “yes” answers: extra practice recommended in session zero
- 3+ “yes” answers: ready for standard track

## Outcome
- Share results with instructor for tailored coaching
