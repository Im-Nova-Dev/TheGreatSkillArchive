# Parent / Manager Communication Template

## For Parents (K-12 / younger learners)
Subject: {Student} is learning Git and GitHub!

{Student} is learning version control and collaboration tools used by professional developers. They are:
- Creating code projects with Git
- Opening pull requests on GitHub
- Learning to debug and review code

No prior experience required — we start from the basics.

## For Managers (Corporate Training)
Subject: Git/GitHub training progress update

Team members are learning terminal-based Git workflows including:
- Branching, merging, and conflict resolution
- Pull request creation and review
- GitHub Actions basics

Expected business impact: fewer merge conflicts, faster onboarding, standardized workflow.

## Progress Sharing
- Weekly summary of topics covered
- Optional: share learner’s GitHub profile link
