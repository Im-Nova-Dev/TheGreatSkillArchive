# Two-Factor Authentication Setup Guide

## Why
Passwords alone are weak. 2FA prevents account takeover.

## Methods
- TOTP app (Authy, 1Password, Google Authenticator)
- Security key (YubiKey, Touch ID)
- SMS (least recommended)

## Setup
1. Settings → Password and authentication
2. Enable two-factor auth
3. Save recovery codes
4. Test by signing out and back in
