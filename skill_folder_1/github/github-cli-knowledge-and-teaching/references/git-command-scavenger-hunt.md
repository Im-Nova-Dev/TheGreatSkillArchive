# Git Command Scavenger Hunt

## Objective
Find the right Git command for each scenario.

## Scenarios
1. “Show me which files I changed.”
2. “Save my changes as a snapshot.”
3. “See my saved snapshots.”
4. “Create an alternate timeline.”
5. “Switch to that alternate timeline.”
6. “Combine two timelines.”
7. “Upload my timeline to GitHub.”
8. “Download the latest from GitHub.”
9. “Find a commit I lost.”
10. “Temporarily save my work and switch tasks.”

## Answers
1. `git status` 2. `git commit -m "..."` 3. `git log --oneline`
4. `git branch new` 5. `git switch new` 6. `git merge new`
7. `git push` 8. `git pull` 9. `git reflog`
10. `git stash`
