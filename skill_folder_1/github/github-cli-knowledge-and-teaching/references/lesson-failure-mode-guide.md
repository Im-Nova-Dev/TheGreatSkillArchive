# Lesson Failure Mode Guide

## Symptoms & Fixes

**Symptom: Learners still confused after second explanation**
- Stop teaching, switch to analogy + 2-min coaching
- Do not reword the same sentence louder

**Symptom: Everyone typing but nothing works**
- You moved too fast; demo the next step as a group before letting individuals try

**Symptom: Some finish in 5 min, others in 30**
- Provide optional advanced extensions to quick finishers
- Pair quick finishers with struggling learners (not to take over — to narrate)

**Symptom: Confidently wrong answers in group**
- Gently correct with evidence from a real terminal session
- Do not shame; normalize confusion

**Symptom: Setup failures block the room**
- Have a documented fallback repo.zip with `.git` pre-initialized
- Keep a helper station for late joiners

**Symptom: Energy drops at midpoint**
- Stand up demo, not sit
- Insert a 5-min quiz or competition

## Pre-Flight Checklist
- [ ] Test all commands on a clean machine
- [ ] Verify sample repo URLs are reachable
- [ ] Have backup plan if internet fails
- [ ] Instructor terminal font >= 14pt
