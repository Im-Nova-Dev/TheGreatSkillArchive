# GitHub Profile Review Assignment

## Task
Review your own GitHub profile and a classmate’s profile.

## Checklist
- [ ] Profile photo and bio present
- [ ] Pinned repos showcase best work
- [ ] README in each pinned repo is descriptive
- [ ] Contributions graph looks active
- [ ] No accidental commits with secrets

## Peer Review
Exchange with a classmate. Each writes:
- One strength
- One suggestion
- One question

## Outcome
Learners treat their GitHub profile as a portfolio.
