# Lesson Recording Checklist

## Equipment
- [ ] External microphone or headset
- [ ] Screen capture at 720p minimum
- [ ] Terminal font >= 14pt
- [ ] Dark theme for legibility

## Pre-Flight
- [ ] Close Slack, email, personal apps
- [ ] Turn off notifications
- [ ] Test audio levels
- [ ] Backdrop / background clean

## Recording
- [ ] Start with 5-second slate: date, session, topic
- [ ] Type slowly; narrate every step
- [ ] Show error states too — don’t just edit them out
- [ ] Pause between exercises to let recording breathe

## Post-Production
- [ ] Trim silence and mistakes
- [ ] Add captions or transcript
- [ ] Export MP4 and upload to channel
- [ ] Link in course portal
