# Homework Assignment Tracker

## Usage
Instructor maintains one copy per track. Name the file by cohort, e.g. `track-2026-06.py`.

## Template

```
COHORT = "June 2026"
ASSIGNMENTS = {
    "pr-review-1": {
        "title": "Open a PR and review one peer PR",
        "due": "2026-06-10",
        "check": ["pr created", "review submitted"],
        "weight": 1,
    },
    "conflict-resolve": {
        "title": "Create and resolve a merge conflict",
        "due": "2026-06-12",
        "check": ["conflict markers seen", "clean merge commit"],
        "weight": 1,
    },
}
```
