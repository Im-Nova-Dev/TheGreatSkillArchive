# Trailer Recording Script

Visual + audio guidance for recording the promo video.

## Shot List
1. Terminal typing `git status` (close-up)
2. Branch graph animating (`git log --graph`)
3. PR creation in terminal (`gh pr create`)
4. Smiling learner/actor saying “I get it now”
5. Call-to-action screen

## Narration
Use the 60-second course trailer script. Speak clearly, at a comfortable pace. Add captions.
