# Plain Language Contributor Agreement

This document explains the spirit of contributing in our community. It is not legally binding; use the official CLA if required.

## We Ask You
- Write clear commit messages and PR descriptions
- Respect review feedback
- Not introduce copyrighted code without permission
- Follow our code of conduct

## We Promise
- Review contributions in a timely way
- Credit you by name in release notes if desired
- Give constructive, specific feedback
- Not merge without at least one review for non-trivial changes

## How We Work Together
- Branches are cheap, create them freely
- PRs are conversations, not confrontations
- Merges can be reverted — no panic
- If stuck, ask — we all were beginners once
