# Terminal Typing Tutor Setup Guide

## Tools
- `speedtype` or `gtypist`
- Custom word list: git and shell commands

## Word List Examples
- `git status`
- `git switch -c feature`
- `git push -u origin main`
- `gh pr create --title`

## Practice Routine
- 5 minutes at start of each session
- Track WPM and accuracy
- Celebrate improvements
