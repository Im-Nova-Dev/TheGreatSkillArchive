# Course Completion Badges Definition

## Badges
- **Git Initiated** — completed session zero
- **Branch Creator** — created and merged a branch
- **Conflict Resolver** — resolved a merge conflict
- **PR Pilot** — opened and merged a PR via terminal
- **Reviewer** — submitted a PR review
- **Automator** — triggered and read an Actions run
- **Mentor** — helped another learner

## Display
- Markdown snippet for README
- SVG badge image
- Credly / blockchain alternative
