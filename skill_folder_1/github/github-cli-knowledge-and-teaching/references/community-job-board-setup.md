# Community Job Board Setup

## In Chat (Slack / Discord)
- Channel: #jobs
- Format required: Role + Company + Remote/On-site + Link
- No recruiters without maintainer approval

## Shared Doc
- Columns: Date | Role | Company | Link | Referrer
- Pin the most recent positions

## Alumni Referral Program
- “I referred a learner” earns small thank-you (badge, shout-out)

## GitHub Repo Alternative
- Create `jobs` repo with markdown entries
- Each job is an issue with labels by type (full-time, contract, open-source)
