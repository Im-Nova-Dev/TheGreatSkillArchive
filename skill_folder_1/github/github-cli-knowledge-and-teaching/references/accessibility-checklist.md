# Accessibility Checklist

## Before the Workshop
- [ ] Printed handouts use 12pt+ font
- [ ] Slides have high contrast and avoid red/green-only cues
- [ ] Captions or transcripts for all videos
- [ ] Live captioning available if possible
- [ ] Instructor speaks clearly, not too fast

## During the Workshop
- [ ] Repeat questions from chat before answering
- [ ] Describe visuals verbally ("note the red branch in the graph")
- [ ] Allow extra time on exercises
- [ ] Provide keyboard shortcuts list with mnemonics
- [ ] Check in with learners who seem disengaged

## Materials
- [ ] Offer markdown version in addition to PDF
- [ ] Provide dark-mode alternative
- [ ] Avoid text-only inside images
- [ ] Offer plain-text quiz alternatives
