# Issue Triage Workshop

## Goal
Teach learners to read, classify, and act on GitHub issues.

## Agenda
- What is triage?
- Labels and milestones
- Exercise: categorize 10 sample issues
- Demo: `gh issue list`, `gh issue edit`, `gh issue close`
- Role-play: act as maintainer of a busy repo

## Sample Issues (Mix)
- Bug report with reproduction
- Feature request with no details
- Duplicate issue
- Question (“how do I…?”)
- Spam / off-topic

## Outcome
- Comfortable reading issues
- Can label and close appropriately
- Understands “good first issue” curation
