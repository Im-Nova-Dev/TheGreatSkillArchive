# GitHub Terms of Service — Simplified for Learners

## What you own
You own the content you post (code, issues, PRs).

## What GitHub owns
The platform, the UI, the infrastructure.

## Rules
- Don’t post illegal or abusive content
- Don’t spam or scrape excessively
- Respect others’ intellectual property

## Takeaways
- Public repos = public code
- Private repos = still subject to platform terms
- GitHub can remove content that violates rules
