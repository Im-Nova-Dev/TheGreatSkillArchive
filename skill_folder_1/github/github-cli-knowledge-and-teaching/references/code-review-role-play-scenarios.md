# Code Review Role-Play Scenarios

## Scenario 1: The Overeager Newcomer
Role-play PR that changes 40 files for a one-line fix.

## Scenario 2: The Absent Author
PR with no description; reviewer must decide whether to nitpick or wait.

## Scenario 3: The Nitpicker
Review with 60 line comments on a small change; learner practices responding calmly.

## Scenario 4: The Late Fix
PR opened after merge; practice revert vs follow-up PR choice.
