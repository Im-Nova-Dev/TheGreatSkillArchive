# Shell History as Study Log Worksheet

## Concept
Your bash/zsh history remembers what you practiced.

## Exercise
1. Run `history | grep git | tail -n 20`.
2. List the commands you used most.
3. Which ones did you have to repeat because of typos?
4. Which ones took the longest?

## Reflection
- “Which Git command do I now type without thinking?”
- “Which command do I still hesitate on?”
