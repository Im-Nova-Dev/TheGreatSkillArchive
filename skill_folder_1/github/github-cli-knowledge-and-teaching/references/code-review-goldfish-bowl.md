# Code Review Goldfish Bowl

## Setup
One learner sits in the “bowl” and reviews a PR out loud. Everyone else watches and listens.

## Flow
1. Instructor selects a PR in the sample repo.
2. Reviewer narrates: what they check, what they comment.
3. Observers write down one thing they would do differently.
4. After 10 minutes, swap reviewer.

## Outcomes
- See review in action
- Hear different review styles
- Build shared vocabulary
