# Capstone Judging Rubric

## Scoring Dimensions
| Dimension | 1 | 2 | 3 | 4 |
|-----------|---|----|----|---|
| PR Quality | Minimal description | Some context | Clear description + linked issue | Excellent: steps to reproduce, tests, screenshots |
| Git History | Messy / unclear commits | Some structure | Logical commit flow | Clean, narrative commits |
| Collaboration | Solo, no review | Peer review received | Responded to feedback | Iterated and improved PR |
| Automation | None | Static CI | CI + status check | CI + deploy / release |
| Reflection | None | One-sentence | Short paragraph | Insightful lessons learned |

## Use
- Judge fills rubric per presentation
- Totals map to: <8 needs work, 8-12 solid, >13 excellent
