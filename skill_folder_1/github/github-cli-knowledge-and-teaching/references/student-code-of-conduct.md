# Student Code of Conduct

## Respect
- Listen when others speak
- Ask, don’t tell
- No “git gurus” — we are all learning

## Practice
- Try before asking for the answer
- Share your mistakes openly
- Give credit when borrowing code

## Safety
- Do not share secrets or credentials
- Report suspicious links or behavior
- Respect privacy of classmates’ repos

## Accountability
- Show up on time
- Complete exercises honestly
- Admit when you’re stuck
