# Learning Sprint Templates

## 24-Hour Sprint
- Hour 0-2: install, configure, first commit
- Hour 2-6: branches and merges
- Hour 6-12: remotes, forks, PRs
- Hour 12-20: real PR to an open-source repo
- Hour 20-24: reflection, cleanup, blog post draft

## 3-Day Intensive

Day 1:
- Morning: Git fundamentals
- Afternoon: Branching lab

Day 2:
- Morning: GitHub UI + CLI
- Afternoon: First PR

Day 3:
- Morning: PR review, Actions
- Afternoon: Capstone presentations

## Rules
- Breaks every 90 minutes
- No skipped exercises
- End with a tangible artifact (PR or repo)
