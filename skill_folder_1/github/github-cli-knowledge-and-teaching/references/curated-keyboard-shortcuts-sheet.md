# Curated Keyboard Shortcuts Sheet

## Terminal
- `Ctrl + C` — cancel current input
- `Ctrl + L` — clear screen
- `Ctrl + R` — reverse search history
- `Ctrl + A / E` — start / end of line
- `Ctrl + U / K` — delete before / after cursor
- `Tab` — autocomplete

## Git
- `git add -p` — stage hunks
- `git commit --amend` — fix last commit
- `git rebase -i HEAD~3` — reword / squash / reorder
- `git cherry-pick <sha>` — apply a single commit
- `git stash; git stash pop` — temporary save

## GitHub CLI
- `gh pr create`
- `gh pr checkout <id>`
- `gh pr view`
- `gh pr review --approve`
- `gh issue list --label "good first issue"`

## Bash / Zsh
- `!!:gs/old/new/` — last command substitution
- `history | grep git` — find past command
