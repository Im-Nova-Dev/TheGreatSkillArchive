# Homework Late Policy

## Philosophy
Learning beats deadlines. The goal is practice, not punishment.

## Rules
- Late work accepted without penalty within 7 days
- After 7 days: 10-minute check-in to unblock
- No zeros — incomplete work gets a “not yet” and a retry path
- Excuses are not required; just submit when ready

## Communication
- Learners DM instructor when late
- Instructor responds within 24h
- If stuck, provide hint, not answer
