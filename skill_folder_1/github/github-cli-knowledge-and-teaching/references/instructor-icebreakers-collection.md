1. “Git horror stories” — everyone shares one
2. “Two truths and a lie” about version control
3. “Alias auction” — bid keystrokes for creative aliases
4. “Commit message charades” — act out a message, others guess
5. “Branch naming brainstorm” — fastest 10 branch names wins
6. “Terminal speed round” — fastest `git log` navigation
7. “Pair interview” — learn one fact about your partner in 2 minutes
