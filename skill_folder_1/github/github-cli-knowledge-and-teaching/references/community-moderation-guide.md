# Community Moderation Guide

For alumni Slack/Discord and class chat.

## Roles
- **Maintainer**: full access, sets tone
- **Moderator**: handles reports, enforces rules
- **Member**: participates, reports issues

## Rules
- Be respectful and beginner-friendly
- No elitism (“git gurus” not welcome)
- Announce job posts only in jobs channel
- Do not share credentials or tokens

## Escalation Path
1. Moderator warns
2. Moderator removes message
3. Maintainer bans (24h or permanent)

## Welcome Message
> Welcome to the Git/GitHub learning community. Rules are simple: be kind, ask questions, share what you learn. Use `/topic` for resume-style posts. DM a moderator for anything private.
