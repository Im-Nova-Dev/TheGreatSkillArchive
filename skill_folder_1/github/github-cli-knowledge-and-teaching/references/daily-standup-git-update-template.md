# Daily Standup Git Update Template

For teams using Git in daily syncs.

```
Yesterday:
- Branch: ______
- PRs: opened ___, reviewed ___, merged ___

Today:
- Branch: ______
- Plan: PR, review, or bugfix

Blockers:
- Waiting on review from: ___
- Conflicts / issues: ___
```

“One Git win from yesterday. One Git goal for today.”
