# Beginner Readiness Test — Git and GitHub

Use this before the first lesson to decide the starting session.

Format: yes/no questions in plain language.

## Questions

1. Can you open a terminal?
2. Have you ever typed a command in a terminal?
3. Do you know what a folder is?
4. Do you know the difference between saving a file and copying a file?
5. Have you used GitHub in a browser before?
6. Can you log in to GitHub?
7. Do you know what a “file” is on a computer?
8. Have you ever zipped or unzipped a file?
9. Do you know how to copy and paste text in the terminal?
10. Are you comfortable editing a text file?

## Scoring

- 8–10 yes: start with Session 1
- 5–7 yes: start with terminal warm-up, then Session 1
- 0–4 yes: do a 30-minute terminal basics prereq before Git lessons

## Notes

- “Yes” can mean “I think so” — probe one or two follow-ups verbally.
- This test is about readiness, not intelligence.
- Record the result; it helps decide which drills to emphasize.
