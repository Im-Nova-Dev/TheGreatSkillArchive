# Custom Terminal Prompt Workshop

## Goals
Add Git awareness to your prompt.

## Tools
- Starship (cross-shell)
- Oh My Zsh themes
- Pure prompt
- Custom PS1 / Starship.toml

## Hands-On
1. Install and configure Starship.
2. Enable `git_branch` and `git_status` modules.
3. Choose a theme and font that displays emoji or symbols well.

## Discussion
How does a prompt change your workflow?
