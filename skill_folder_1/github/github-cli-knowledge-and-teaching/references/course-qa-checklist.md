# Course QA Checklist

- [ ] All lesson commands tested on clean machine
- [ ] Videos render properly
- [ ] PDFs generated and readable
- [ ] Links resolved (no 404s)
- [ ] Spelling / grammar pass
- [ ] Accessibility: alt text, contrast, captions
- [ ] Setup script runs without errors
- [ ] Backup / fallback plan documented
- [ ] Instructor rehearsal completed
- [ ] Learner communication drafted
