# Instructor Shadow Evaluation Rubric

For observers rating a lead instructor.

| Dimension | Excellent | Good | Needs Work |
|-----------|-----------|------|------------|
| Clarity | Always clear | Mostly clear | Frequent rewording |
| Pacing | Balanced | Slightly fast/slow | Hard to follow |
| Engagement | Learners active | Some active | Most passive |
| Accuracy | No errors | Minor errors | Multiple corrections |
| Recovery | Smooth when stuck | Recovered okay | Lost the room |

## Notes
Observer adds one specific praise and one specific improvement.
