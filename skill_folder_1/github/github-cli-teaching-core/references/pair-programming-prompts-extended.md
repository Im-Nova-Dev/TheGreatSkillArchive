# Extended Pair Programming Prompts

1. “Navigate to the same ref without saying it out loud. You have 10 seconds. Unspoken telepathy wins.”
2. “One person handles conflict resolution, the other narrates. Switch halfway.”
3. “One person is the author, one is the reviewer — collaborate on a single PR.”
4. “Solve the same rebase two ways: merge vs rebase. Compare graphs.”
5. “Create a repo from scratch without looking at notes. Time each other.”
6. “Find something in `git log` and describe it in one line to your partner.”
7. “Race: who can create, rename, push, and PR a branch fastest?”
8. “Debug mode: one person reads error output, the other types.”
