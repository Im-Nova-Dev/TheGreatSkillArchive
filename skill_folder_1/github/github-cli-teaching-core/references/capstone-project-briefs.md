# Capstone Project Briefs

## Capstone A — Solo Contribution

Goal: make a visible, merged PR to any open-source repo.

Milestones:
- Find 1 issue labeled “good first issue”
- Reproduce it
- Open a PR with tests/docs
- Iterate through reviews
- PR merged

Deliverable: Screenshot + short reflection.

## Capstone B — Team Internal Repo

Goal: ship a small feature in a 2-3 person team.

Milestones:
- Feature branch split
- Multiple commits across people
- PR opened, reviewed, approved
- Merged to main
- Release tagged

Deliverable: Link to tag + PR.

## Capstone C — Automation Play

Goal: add a GitHub Actions workflow to a real project.

Milestones:
- Multi-job workflow
- Matrix build (if applicable)
- Artifact upload or deployment step

Deliverable: `gh run view` output + workflow file.

## Capstone D — Documentation PR

Goal: fix or improve documentation on GitHub.com/docs or a community docs repo.

Deliverable: PR link + before/after diff summary.
