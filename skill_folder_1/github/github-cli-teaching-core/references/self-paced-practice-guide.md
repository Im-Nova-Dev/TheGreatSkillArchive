# Self-Paced Git Practice Guide

Use this as a solo progression from zero to comfortable.

## Week 1 — Foundations

Day 1:
- Install Git, set identity
- Create repo, commit a file, inspect with `git status`
- Practice undoing with `git restore`, `git reset --soft`, `git revert`

Day 2:
- Create branches, switch between them
- Visualize history with `git log --graph --oneline --all`

Day 3:
- Practice merge conflict simulation
- Create two branches, edit same file, merge one into main, merge second

Day 4:
- Push to GitHub, create PR, merge, delete branch

Day 5:
- Review someone else’s PR or revisit yesterday’s PR
- Write a review using `gh pr review`

## Week 2 — Habit Building

- Rewrite messy local history with interactive rebase
- Fix upstream PR feedback and push amendments
- Practice `git cherry-pick` by copying a commit between branches
- Restore from `git reflog` after a deliberate mistake

## Week 3 — Real Collaboration

- Contribute to a small open-source repo
- Report and fix a real bug via issue + PR
- Keep a fork in sync with upstream for 1 week

## Practice Tip

After every exercise, ask:
- What did I just do?
- What changed locally?
- What changed on GitHub?
- What would I do differently next time?
