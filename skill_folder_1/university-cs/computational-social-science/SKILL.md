# Computational Social Science

## Core topics

1. Data
   - Trace data and sampling
   - Bias and representativeness

2. Methods
   - Network analysis
   - Text and topic models

3. Causality
   - Correlation versus causation
   - Experiment basics

## Teaching approach
- Analyze one simple social dataset.
- Map one network relationship.
- Explain one causal inference limit.
