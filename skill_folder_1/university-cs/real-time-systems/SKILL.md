# Real-Time Systems

## Core topics

1. Real-time concepts
   - Hard versus soft
   - Timing requirements

2. Scheduling
   - RMS and EDF
   - Feasibility analysis

3. Implementation
   - Interrupt handling
   - Priority inversion

## Teaching approach
- Compare hard and soft real-time.
- Analyze one schedule for feasibility.
- Design one task set and schedule.
