# Fuzz Testing

## Core topics

1. Fuzzing approaches
   - Random and mutation
   - Coverage guidance

2. Corpus and input
   - Seed management
   - Grammar-based generation

3. Triage and CI
   - Bug reduction
   - Integration

## Teaching approach
- Run one simple fuzzer.
- Explain one coverage-guided concept.
- Triage one crash report.
