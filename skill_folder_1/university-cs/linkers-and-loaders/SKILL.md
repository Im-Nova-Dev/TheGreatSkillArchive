# Linkers and Loaders

## Core topics

1. Linking
   - Object files and symbols
   - Relocation

2. Libraries
   - Static and dynamic
   - Shared objects

3. Runtime
   - Position-independent code
   - Dynamic loading

## Teaching approach
- Inspect one binary with nm and ldd.
- Explain one symbol resolution failure.
- Compare static and dynamic linking tradeoffs.
