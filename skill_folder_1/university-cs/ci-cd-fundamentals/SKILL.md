# CI/CD Fundamentals

## Core topics

1. Pipeline design
   - Build, test, package stages
   - Gates and manual approvals

2. Environments
   - Development, staging, production
   - Parity and configuration

3. Deployment modes
   - Rolling, blue-green, canary
   - Feature flags

4. Observability
   - Artifact storage
   - Release notes and rollbacks

## Teaching approach
- Wire one existing project to a simple pipeline.
- Add one stage and justify its cost.
- Simulate one deployment failure and recover.
