# Algorithms Core Teaching

## Core topics

1. Techniques
   - Sorting and searching
   - Graph traversal

2. Problem solving
   - Dynamic programming
   - Greedy and divide-and-conquer

3. Communication
   - Complexity reasoning
   - Explaining tradeoffs

## Teaching approach
- Solve one problem with two algorithms.
- Prove one greedy property intuitively.
- Explain one algorithm choice for one learner.
