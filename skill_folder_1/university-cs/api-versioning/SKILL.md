# API Versioning

## Core topics

1. Strategies
   - URI versioning
   - Header and content versioning

2. Compatibility
   - Breaking versus nonbreaking
   - Deprecation policy

3. Rollout
   - Migration steps
   - Monitoring clients

## Teaching approach
- Design one versioning policy.
- Propose one safe migration between versions.
- Review one API change for compatibility.
