# User Experience Fundamentals

## Core topics

1. Usability
   - Learnability and efficiency
   - Error prevention and recovery

2. Design process
   - User research
   - Prototyping and testing

3. Accessibility
   - Inclusive design principles
   - Assistive technology

## Teaching approach
- Conduct one heuristic evaluation.
- Design one accessibility improvement.
- Review one interface for usability issues.
