# IO Systems

## Core topics

1. IO models
   - Block and character devices
   - Memory mapped IO

2. Scheduling
   - IO scheduler role
   - Elevator concepts

3. Performance
   - DMA and interrupts
   - Caching and prefetch

## Teaching approach
- Compare sync and async IO concepts.
- Review one IO scheduler choice.
- Explain one throughput bottleneck.
