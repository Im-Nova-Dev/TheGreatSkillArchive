# Secure Coding

## Core topics

1. Inputs and outputs
   - Injection and validation
   - Encoding and escaping

2. Auth and sessions
   - Identity and access flaws
   - Session handling

3. Hardening
   - Secrets management
   - Logging and monitoring

## Teaching approach
- Audit one code path for risks.
- Fix one vulnerable example.
- Add one logging improvement.
