# Virtualization and Containers

## Core topics

1. Virtualization
   - Type 1 and type 2 hypervisors
   - VM image lifecycle

2. Containers
   - Isolation mechanisms
   - Image primitives
   - Run and inspect containers

3. Orchestration
   - Service definitions
   - Networking and storage

## Teaching approach
- Run same workload in VM and container.
- Compare startup time and resource usage.
- Explain security boundary differences.
