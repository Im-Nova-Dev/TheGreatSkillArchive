# Instruction Set Architecture

## Core topics

1. Instruction design
   - Formats and encoding
   - Addressing modes

2. Calling conventions
   - Parameter passing
   - Register usage

3. Evolution
   - RISC and CISC
   - SIMD and extensions

## Teaching approach
- Decode one instruction for one ISA.
- Compare two instruction formats.
- Explain one calling convention choice.
