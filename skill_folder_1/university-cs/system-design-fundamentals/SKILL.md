# System Design Fundamentals

## Core topics

1. Requirements
   - Functional and non-functional
   - Estimations: QPS, storage, bandwidth

2. Scaling
   - Vertical vs horizontal
   - Stateless services and read replicas

3. Storage
   - SQL vs NoSQL choices
   - Indexing and denormalization
   - Caching: client, CDN, application, database

4. Communication
   - Synchronous and asynchronous
   - REST and GraphQL concepts
   - gRPC and protobuf basics

5. Reliability
   - Health checks and retries
   - Circuit breakers
   - Observability

## Teaching approach
- Use one target system per session.
- Ask for constraints before proposing architecture.
- Draw one diagram and then critique it.
