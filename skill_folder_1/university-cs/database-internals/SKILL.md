# Database Internals

## Core topics

1. Storage
   - Pages and files
   - Buffer management

2. Indexing
   - B-trees and variants
   - Hash indexes

3. Query processing
   - Execution plans
   - Join algorithms

## Teaching approach
- Compare one storage engine design.
- Analyze one query execution plan.
- Explain one index choice tradeoff.
