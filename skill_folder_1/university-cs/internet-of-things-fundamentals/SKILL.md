# Internet of Things Fundamentals

## Core topics

1. Device
   - Edge hardware
   - Embedded software

2. Connectivity
   - Communication models
   - Protocols

3. Scale
   - Device management
   - Data handling

## Teaching approach
- Design one IoT deployment.
- Evaluate one connectivity choice.
- Review one security model.
