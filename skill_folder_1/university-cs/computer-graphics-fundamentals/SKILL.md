# Computer Graphics Fundamentals

## Core topics

1. Geometry
   - Coordinate spaces and transforms
   - Primitives

2. Rasterization
   - Fragment processing
   - Depth and stencil

3. Shading
   - Lighting models
   - Texture mapping

## Teaching approach
- Implement one simple renderer component.
- Explain one shading model.
- Compare rasterization with ray tracing.
