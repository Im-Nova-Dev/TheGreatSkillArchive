# Practical Projects

## Core topics

1. Planning
   - Problem definition
   - Scope and milestones

2. Execution
   - Implementation strategy
   - Testing and refinement

3. Review
   - Deployment
   - Retrospective

## Teaching approach
- Plan one project from requirements.
- Structure one implementation roadmap.
- Conduct one project retrospective.
