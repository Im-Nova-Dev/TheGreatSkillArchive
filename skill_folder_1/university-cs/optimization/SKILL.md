# Optimization

## Core topics

1. Continuous
   - Convexity idea
   - Gradient and descent

2. Discrete
   - Integer programming idea
   - Relaxation concept

3. Practical
   - Solver usage concept
   - Warm start and constraints

## Teaching approach
- Formulate one problem with variables and objective.
- Explain convex versus nonconvex risk.
- Compare exact versus heuristic approach.
