# Container Runtime

## Core topics

1. Isolation
   - Namespaces
   - Cgroups

2. Images and storage
   - Layers and registry
   - Volumes

3. Networking and security
   - Bridge and overlay
   - Hardening

## Teaching approach
- Inspect namespaces and cgroups for one container.
- Explain one storage driver concept.
- Plan one container security baseline.
