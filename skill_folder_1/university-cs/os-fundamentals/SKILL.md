# OS Fundamentals

## Core topics

1. Processes and threads
   - Lifecycle and states
   - Scheduling concept

2. Memory and storage
   - Virtual memory
   - File and block IO

3. Concurrency
   - Synchronization
   - System calls

## Teaching approach
- Trace one process lifecycle.
- Explain one scheduler decision.
- Model one synchronization scenario.
