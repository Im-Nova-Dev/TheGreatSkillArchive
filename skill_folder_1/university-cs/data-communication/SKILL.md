# Data Communication

## Core topics

1. Transmission
   - Signals and bandwidth
   - Modulation and encoding

2. Errors
   - Detection and correction
   - Retry and recovery

3. Networking
   - Topologies
   - Multiplexing

## Teaching approach
- Compare one analog versus digital example.
- Design one simple error detection scheme.
- Review one communication system diagram.
