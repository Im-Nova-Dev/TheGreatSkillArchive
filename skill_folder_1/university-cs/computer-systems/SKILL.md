# Computer Systems

## Core topics

1. Execution model
   - Processor and instruction concept
   - Memory hierarchy

2. Memory
   - Cache and locality
   - Virtual memory

3. Systems programming
   - System calls and linking
   - Performance analysis

## Teaching approach
- Trace one simple program execution at system level.
- Explain one cache hit versus miss scenario.
- Propose one performance-oriented code change.
