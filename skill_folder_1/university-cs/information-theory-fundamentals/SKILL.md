# Information Theory Fundamentals

## Core topics

1. Entropy
   - Uncertainty and information
   - Binary entropy

2. Source coding
   - Compression concept
   - Prefix codes and Kraft inequality

3. Channel coding
   - Capacity concept
   - Redundancy and error correction

4. Applications
   - Compression and communication
   - ML connection briefly

## Teaching approach
- Compute entropy for one toy distribution.
- Compare statistical and dictionary compression.
- Explain error correction by example.
