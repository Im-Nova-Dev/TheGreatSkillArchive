# Retrieval Augmented Generation

## Core topics

1. Retrieval
   - Chunking and embedding
   - Indexing strategies

2. Generation
   - Context injection
   - Prompt integration

3. Evaluation
   - Relevance and faithfulness
   - Failure modes

## Teaching approach
- Build one small retrieval pipeline.
- Evaluate one hallucination risk.
- Compare one hybrid search design.
