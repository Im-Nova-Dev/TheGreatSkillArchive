# Software Architecture

## Core topics

1. Styles
   - Layered, modular, event-driven
   - Microservices and monolith

2. Quality attributes
   - Performance
   - Reliability
   - Maintainability

3. Decisioning
   - ADRs
   - Tradeoff analysis

## Teaching approach
- Draw one architecture for one system.
- Evaluate one style against one requirement.
- Write one architecture decision record.
