# Reliability Patterns

## Core topics

1. Fault tolerance
   - Retries and backoff
   - Timeouts and deadlines

2. Isolation
   - Bulkheads
   - Circuit breaking

3. Resilience practices
   - Graceful degradation
   - Chaos testing

## Teaching approach
- Design one resilient API interaction.
- Add one timeout and retry config.
- Explain one failure mode handled by one pattern.
