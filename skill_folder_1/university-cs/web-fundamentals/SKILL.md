# Web Fundamentals

## Core topics

1. Platform basics
   - URLs and HTTP
   - Request and response

2. Browser
   - DOM and rendering
   - Scripting

3. Features
   - Storage and APIs
   - Security and accessibility

## Teaching approach
- Trace one page load.
- Inspect one request with dev tools.
- Review one page for accessibility issues.
