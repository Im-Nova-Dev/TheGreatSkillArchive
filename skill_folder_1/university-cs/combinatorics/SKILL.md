# Combinatorics

## Core topics

1. Counting
   - Permutations and combinations
   - Binomial coefficients

2. Principles
   - Inclusion-exclusion
   - Pigeonhole principle

3. Recurrence
   - Recurrences
   - Generating functions

## Teaching approach
- Count arrangements for one example.
- Solve one recurrence.
- Explain one combinatorial proof idea.
