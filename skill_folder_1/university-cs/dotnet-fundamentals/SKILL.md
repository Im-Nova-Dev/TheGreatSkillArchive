# .NET Fundamentals

## Core topics

1. Runtime
   - CLR and assemblies
   - Memory management

2. Language
   - C# essentials
   - LINQ and async

3. Practice
   - Dependency injection
   - Minimal API structure

## Teaching approach
- Create one small service.
- Refactor one callback to async.
- Review one dependency graph.
