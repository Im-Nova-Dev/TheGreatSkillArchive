# System Design

## Core topics

1. Requirements
   - Functional and nonfunctional
   - Constraints

2. Scale and resilience
   - Scaling approaches
   - Availability and consistency

3. Design flow
   - APIs and data models
   - Subsystems

## Teaching approach
- Evaluate one system prompt for ambiguity.
- Design one subsystem boundary.
- Recommend one scaling approach with justification.
