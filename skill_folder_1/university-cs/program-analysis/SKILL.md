# Program Analysis

## Core topics

1. Static analysis
   - Control and data flow
   - Taint analysis

2. Dynamic analysis
   - Instrumentation
   - Runtime behavior observation

3. Advanced
   - Symbolic execution
   - Abstract interpretation

## Teaching approach
- Run one static analyzer.
- Instrument one program for coverage.
- Explain one analysis report.
