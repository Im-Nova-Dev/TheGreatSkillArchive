# Parallelism

## Core topics

1. Models
   - Thread and task parallelism
   - Data parallelism

2. Performance
   - Speedup and efficiency
   - Dependencies

3. Correctness
   - Synchronization
   - Scheduling

## Teaching approach
- Decompose one task for parallelism.
- Compute one expected speedup.
- Identify one synchronization hazard.
