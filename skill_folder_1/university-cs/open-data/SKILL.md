# Open Data

## Core topics

1. Principles
   - Access and reuse
   - Licensing

2. Practice
   - Formats and APIs
   - Quality

3. Community
   - Collaboration
   - Governance

## Teaching approach
- Review one dataset for openness.
- Recommend one open license.
- Design one open data API.
