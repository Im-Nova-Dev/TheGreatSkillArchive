# Database Systems Teaching

## Core topics

1. Relational model
   - Tables and keys
   - SQL

2. Internals
   - Indexes and execution
   - Transactions

3. Practice
   - Normalization
   - DB selection

## Teaching approach
- Model one domain relationally.
- Rewrite one query and explain plan.
- Choose one database type for one workload.
