# Licensing and Open Source Legal

## Core topics

1. License types
   - MIT, Apache, GPL, BSD
   - Copyleft versus permissive

2. Copyright
   - Ownership and authorship
   - Work for hire

3. Compliance
   - Attribution and notices
   - Dependency licensing

## Teaching approach
- Read one license and summarize restrictions.
- Audit one project for license compatibility.
- Explain one compliance requirement to teammate.
