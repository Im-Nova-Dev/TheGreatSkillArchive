# Game Development Fundamentals

## Core topics

1. Core loop
   - Update and render
   - Fixed timestep

2. Rendering
   - Sprites and cameras
   - Layers and sorting

3. Input
   - Keyboard, mouse, gamepad
   - Buffered input concept

4. Game systems
   - State management
   - Collision detection
   - Simple physics

## Teaching approach
- Build one tiny prototype in engine or HTML canvas.
- Add one mechanic each session.
- Evaluate feel and pacing together.
