# Web Security Fundamentals

## Core topics

1. Injection and scripting
   - SQL injection concept
   - Cross-site scripting prevention

2. Session and cookies
   - Secure cookie flags
   - Session fixation

3. Mitigations
   - Input validation
   - CSP and CORS

## Teaching approach
- Identify one vulnerable pattern.
- Fix one injection example.
- Configure one secure header set.
