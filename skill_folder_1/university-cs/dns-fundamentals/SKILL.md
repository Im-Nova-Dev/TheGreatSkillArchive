# DNS Fundamentals

## Core topics

1. Resolution
   - Recursive and authoritative
   - Record types

2. Configuration
   - Zones and delegation
   - TTL behavior

3. Security
   - DNSSEC concept
   - DoH and privacy

## Teaching approach
- Trace one lookup end to end.
- Configure one basic zone conceptually.
- Debug one resolution issue.
