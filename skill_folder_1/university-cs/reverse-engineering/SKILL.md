# Reverse Engineering

## Core topics

1. Static analysis
   - Disassembly
   - Control flow

2. Dynamic analysis
   - Debugging
   - Tracing

3. Practice
   - Firmware
   - Reassembly

## Teaching approach
- Identify one function boundary.
- Trace one execution path.
- Explain one anti-reverse technique.
