# Open Source Contribution

## Core topics

1. Finding work
   - Good first issue ideas
   - Reading project health signals

2. Submitting
   - Issue quality
   - PR size and description

3. Review
   - Responding to review
   - Iteration discipline

## Teaching approach
- Turn one vague complaint into one issue.
- Review one PR and identify one helpful comment.
- Trace one contribution from idea to merge.
