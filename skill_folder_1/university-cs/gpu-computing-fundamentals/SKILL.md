# GPU Computing Fundamentals

## Core topics

1. Hardware model
   - SM and warps
   - Thread hierarchy

2. Memory
   - Global and shared memory
   - Latency hiding

3. Practice
   - Kernel concept
   - Profiling

## Teaching approach
- Explain one SM scheduling model.
- Profile one kernel occupancy.
- Map one algorithm to GPU structure.
