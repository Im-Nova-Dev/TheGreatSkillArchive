# Networking Operations

## Core topics

1. Core services
   - DNS design
   - DHCP and IPAM

2. Traffic management
   - Firewalls and NAT
   - Load balancing

3. Troubleshooting
   - Packet capture and flow logs
   - Path analysis tools

## Teaching approach
- Review one production network diagram.
- Reproduce and resolve one latency issue.
- Document one change and rollback plan.
