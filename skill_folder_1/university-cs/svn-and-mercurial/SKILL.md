# SVN and Mercurial

## Core topics

1. SVN
   - Centralized model
   - Branches and locking

2. Mercurial
   - Workflow concepts
   - Branching and history

3. Comparison
   - Differences from Git
   - Migration considerations

## Teaching approach
- Compare one centralized workflow with Git flow.
- Map one branching naming scheme.
- Explain one use case for centralized VCS.
