# Code Quality

## Core topics

1. Clarity
   - Naming and structure
   - Simple logic

2. Architecture
   - Cohesion and coupling
   - Duplication control

3. Culture
   - Reviews and standards
   - Debt management

## Teaching approach
- Review one module for quality signals.
- Classify one debt as fixable versus manageable.
- Propose one concrete quality improvement.
