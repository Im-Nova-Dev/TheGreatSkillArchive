# Workload Characterization

## Core topics

1. Metrics
   - Throughput and latency
   - Percentiles and tails

2. Patterns
   - Read-write split
   - Bursty versus steady

3. Usage
   - Capacity planning
   - System design inputs

## Teaching approach
- Characterize one service behavior.
- Convert observations into requirements.
- Discuss one implication for cache or queue choice.
