# Human-AI Interaction

## Core topics

1. Trust
   - Transparency and expectations
   - Failure communication

2. Interface design
   - Feedback on model output
   - Edit and accept patterns

3. Safety
   - Confidence thresholds
   - Escalation paths

## Teaching approach
- Evaluate one AI-assisted workflow.
- Redesign one interaction to clarify uncertainty.
- Balance automation and control.
