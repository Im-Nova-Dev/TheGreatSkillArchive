# Programming Language Fundamentals

## Core topics

1. Core concepts
   - Syntax and semantics
   - Binding and scope

2. Abstractions
   - Functions and modules
   - Types

3. Runtime
   - Evaluation model
   - Error handling

## Teaching approach
- Compare two languages through one small program.
- Explain one scoping decision.
- Critique one syntax design.
