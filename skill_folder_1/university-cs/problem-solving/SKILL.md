# Problem Solving

## Core topics

1. Decomposition
   - Breaking down ambiguity
   - Prioritization

2. Reasoning
   - Hypothesis-driven
   - Tradeoff analysis

3. Validation
   - Testing ideas
   - Decision making

## Teaching approach
- Decompose one ambiguous problem.
- Propose one hypothesis.
- Evaluate one candidate solution.
