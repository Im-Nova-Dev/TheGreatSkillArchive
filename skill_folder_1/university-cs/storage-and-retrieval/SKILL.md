# Storage and Retrieval

## Core topics

1. On-disk structures
   - Files and blocks
   - Indexing

2. Structures
   - B-trees and variants
   - LSM-trees

3. Performance
   - Reads versus writes
   - Compression

## Teaching approach
- Compare B-tree and LSM-tree usage.
- Evaluate one access pattern.
- Choose one storage structure for one workload.
