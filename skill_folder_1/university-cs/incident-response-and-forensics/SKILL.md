# Incident Response and Forensics

## Core topics

1. Response lifecycle
   - Preparation and detection
   - Containment and eradication

2. Evidence
   - Logs and artifacts
   - Integrity and chain of custody

3. Analysis
   - Timeline creation
   - Attestation and attribution

## Teaching approach
- Review one simplified incident report.
- Extract timeline from one log set.
- Write one postmortem with evidence.
