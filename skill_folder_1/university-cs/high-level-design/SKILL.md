# High Level Design

## Core topics

1. Requirements
   - Functional and nonfunctional
   - Constraints

2. Architecture
   - Service boundaries
   - Data flow and contracts

3. Evaluation
   - Capacity estimation
   - Tradeoffs

## Teaching approach
- Diagram one high level architecture.
- Identify one main tradeoff.
- Present one design decision clearly.
