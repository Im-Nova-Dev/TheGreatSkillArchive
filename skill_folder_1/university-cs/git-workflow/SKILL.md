# Git Workflow

## Core topics

1. Workflows
   - Trunk-based and feature branches
   - Forking model

2. Collaboration
   - PR-based teamwork
   - Review and merge

3. Hygiene
   - Commit conventions
   - Branch cleanup

## Teaching approach
- Review one branching model for team fit.
- Convert one workflow into smaller steps.
- Recommend one commit message format.
