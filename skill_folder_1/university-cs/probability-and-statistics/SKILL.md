# Probability and Statistics

## Core topics

1. Probability
   - Events and independence
   - Random variables

2. Distributions and expectation
   - Common distributions
   - Mean and variance

3. Inference
   - Estimation
   - Hypothesis testing

## Teaching approach
- Compute one probability using rules.
- Interpret one distribution.
- Evaluate one simple hypothesis test.
