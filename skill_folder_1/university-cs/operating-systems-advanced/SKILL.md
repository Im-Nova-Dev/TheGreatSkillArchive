# Advanced Operating Systems

## Core topics

1. Kernel architecture
   - Monolithic and microkernel
   - Modules and interfaces

2. Scheduling
   - Policy and mechanisms
   - Fairness and latency

3. Memory
   - Page replacement and THP
   - Memory zones

4. Storage
   - VFS and filesystems
   - Journaling and consistency

## Teaching approach
- Read one kernel config diff.
- Trace one system call end to end.
- Measure scheduler behavior under load.
