# Clean Code

## Core topics

1. Clarity
   - Naming and intent
   - Small functions

2. Structure
   - Single responsibility
   - Dependency direction

3. Maintenance
   - Error handling
   - Change-friendly design

## Teaching approach
- Refactor one messy function.
- Review one code sample for naming.
- Add one clarifying comment where needed.
