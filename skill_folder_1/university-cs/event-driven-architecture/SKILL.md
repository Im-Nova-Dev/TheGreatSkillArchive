# Event-Driven Architecture

## Core topics

1. Model
   - Events and event streams
   - Producer and consumer roles

2. Topology
   - Topics and partitions
   - Fanout and routing

3. Advanced
   - Sagas and idempotency
   - Ordering and replay

## Teaching approach
- Model one workflow as events.
- Define one schema and naming convention.
- Propose retry handling for one consumer.
