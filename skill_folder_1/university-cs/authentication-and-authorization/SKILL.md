# Authentication and Authorization

## Core topics

1. Authentication
   - Password storage and verification
   - MFA and WebAuthn

2. Protocols
   - OAuth 2.0 and OpenID Connect
   - Session design and cookies

3. Access control
   - RBAC and roles
   - ABAC and policies

## Teaching approach
- Design one auth flow for a small app.
- Identify one trust boundary mistake.
- Trace one login and one authorization decision.
