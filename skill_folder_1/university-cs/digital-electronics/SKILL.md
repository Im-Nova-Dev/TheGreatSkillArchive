# Digital Electronics

## Core topics

1. Logic
   - Gates and Boolean algebra
   - Combinational design

2. Sequential logic
   - Flip-flops
   - Counters and registers

3. Design
   - FSM
   - Timing

## Teaching approach
- Simplify one Boolean expression.
- Design one small sequential circuit.
- Trace one FSM transition.
