# Approximation and Randomized Algorithms

## Core topics

1. Randomization
   - Probability in algorithm design
   - Expected runtime

2. Monte Carlo and Las Vegas
   - One-sided vs two-sided error
   - Repeating for confidence

3. Selection and sampling
   - Randomized select
   - Reservoir sampling

4. Hashing
   - Universal hashing concept
   - Collision probability

5. Approximation
   - Greedy approximation examples
   - Polynomial-time approximation schemes concept

## Teaching approach
- Use dice or coin examples.
- Compare randomized vs deterministic approach.
- Explain when randomness is better.
