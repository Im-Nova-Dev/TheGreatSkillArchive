# Code Comprehension

## Core topics

1. Reading strategy
   - Entry points and call graphs
   - Tracing execution

2. Understanding
   - Design detection
   - Dependency mapping

3. Legacy and docs
   - Documentation from code
   - Incremental understanding

## Teaching approach
- Explore one unfamiliar codebase.
- Diagram one module structure.
- Document one existing component clearly.
