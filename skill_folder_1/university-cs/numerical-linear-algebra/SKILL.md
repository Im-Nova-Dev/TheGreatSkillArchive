# Numerical Linear Algebra

## Core topics

1. Matrices and systems
   - Linear systems
   - Gaussian elimination numerically

2. Decompositions
   - LU, QR, SVD
   - Use cases per decomposition

3. Eigenproblems
   - Power method concept
   - Applications in PCA

4. Conditioning
   - Condition number
   - Stability in floating point

## Teaching approach
- Solve one small system numerically.
- Compare exact and iterative answers.
- Explain conditioning with one example.
