# Concurrent Programming

## Core topics

1. Execution models
   - Threads and processes
   - Preemptive versus cooperative

2. Synchronization
   - Locks and mutexes
   - Condition variables and semaphores

3. Correctness
   - Memory ordering
   - Deadlock avoidance

## Teaching approach
- Fix one race condition.
- Design one thread-safe data structure.
- Debug one deadlocked program.
