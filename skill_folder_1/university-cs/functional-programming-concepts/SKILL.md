# Functional Programming Concepts

## Core topics

1. Core ideas
   - Pure functions
   - Immutability and value semantics
   - Referential transparency

2. Composition
   - Higher-order functions
   - Map, filter, reduce
   - Partial application and currying

3. Types and effects
   - Algebraic data types idea
   - Error handling with result types

4. Practical application
   - Functional patterns in Python and JavaScript
   - When to prefer iterative style

## Teaching approach
- Convert one loop-heavy function to map and filter.
- Replace in-place mutation with returned value style.
- Explain where IO breaks functional reasoning.
