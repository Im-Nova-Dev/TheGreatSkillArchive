# Security Foundations

## Core topics

1. Security properties
   - Confidentiality, integrity, availability
   - Authentication and authorization

2. Threats
   - Threat modeling
   - STRIDE concepts

3. Access control
   - DAC, MAC, RBAC
   - Least privilege

4. Cryptography
   - Symmetric and asymmetric encryption
   - Hashing and MACs
   - TLS conceptually

5. Common defenses
   - Input validation
   - Sanitization and escaping
   - Rate limiting and logging

## Teaching approach
- Start with threat stories.
- Map story to missing control.
- Explain why control matters.
