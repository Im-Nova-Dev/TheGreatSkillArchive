# Functional Programming

## Core topics

1. Core concepts
   - Pure functions and immutability
   - Higher-order functions

2. Patterns
   - Recursion and composition
   - Algebraic data types

3. Advanced
   - Monads and functors
   - Type classes concept

## Teaching approach
- Refactor one imperative example to functional style.
- Explain one monad concept with one concrete example.
- Compare functional versus imperative approach.
