# Data Engineering Fundamentals

## Core topics

1. Data models
   - Relational and document models
   - Dimensional modeling

2. Storage formats
   - Row, column, and object storage
   - Parquet, Avro, JSON tradeoffs

3. Pipelines
   - ETL and ELT
   - Batch vs streaming

4. Orchestration
   - DAGs and scheduling

5. Quality
   - Schema validation
   - Duplicate and null handling

## Teaching approach
- Design one small pipeline together.
- Inspect each stage data format.
- Add one quality check per iteration.
