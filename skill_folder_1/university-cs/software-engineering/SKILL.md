# Software Engineering

## Core topics

1. Lifecycle
   - Requirements and design
   - Implementation and testing

2. Process
   - Models
   - Delivery practices

3. Quality
   - Maintainability
   - Metrics

## Teaching approach
- Breakdown one feature into requirements.
- Evaluate one design choice.
- Review one change for maintainability.
