# Concurrency Models

## Core topics

1. Models
   - Threads and coroutines
   - Actors and CSP

2. Communication
   - Message passing
   - Shared memory

3. Tradeoffs
   - Overhead and scalability
   - Correctness

## Teaching approach
- Model one problem in two concurrency models.
- Explain one model choice for one workload.
- Compare correctness reasoning across models.
