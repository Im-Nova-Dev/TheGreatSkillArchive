# Data Modeling

## Core topics

1. Concepts
   - Entities and relationships
   - Integrity

2. Forms
   - Normalization and denormalization
   - Tradeoffs

3. Evolution
   - Schema changes
   - Data contracts

## Teaching approach
- Model one domain from requirements.
- Evaluate one normalization versus performance choice.
- Critique one existing schema.
