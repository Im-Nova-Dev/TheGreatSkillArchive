# Observability Patterns

## Core topics

1. Signals
   - Metrics, logs, traces
   - When to use each

2. Quality
   - Cardinality and sampling
   - Structure and context

3. Action
   - Alerts and dashboards
   - Incident triage

## Teaching approach
- Review one dashboard for signal quality.
- Rewrite one noisy alert.
- Trace one incident using three telemetry sources.
