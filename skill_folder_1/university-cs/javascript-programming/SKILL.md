# JavaScript Programming

## Core topics

1. Language
   - Syntax and types
   - Functions and closures

2. Asynchronous flow
   - Promises and async/await
   - Event loop concept

3. Environment
   - Browser versus Node
   - Module usage

## Teaching approach
- Convert one callback chain to async/await.
- Trace one async execution order.
- Refactor one module for reuse.
