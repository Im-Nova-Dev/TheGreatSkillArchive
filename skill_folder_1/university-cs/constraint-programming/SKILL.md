# Constraint Programming

## Core topics

1. CSP model
   - Variables and domains
   - Constraints

2. Solving
   - Propagation
   - Backtracking

3. Optimization
   - Objectives
   - Heuristics

## Teaching approach
- Model one scheduling problem in CP.
- Compare constraint versus search approach.
- Solve one small puzzle with propagation idea.
