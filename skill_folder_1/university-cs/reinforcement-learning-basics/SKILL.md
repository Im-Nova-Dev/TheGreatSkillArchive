# Reinforcement Learning Basics

## Core topics

1. Core concepts
   - Agent and environment
   - State, action, reward

2. MDP intuition
   - Policy and value
   - Bellman equation concept

3. Methods
   - Q-learning concept
   - Exploration and exploitation

4. Applications
   - Games and robotics
   - Recommendation and control

## Teaching approach
- Use gridworld or simple control task.
- Compare greedy and exploratory policies.
- Visualize value estimates over time.
