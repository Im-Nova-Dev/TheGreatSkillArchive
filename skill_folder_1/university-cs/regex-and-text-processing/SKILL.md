# Regex and Text Processing

## Core topics

1. Regex building blocks
   - Literals and character classes
   - Groups and alternation
   - Quantifiers and anchors

2. Advanced features
   - Lookahead and lookbehind
   - Flags and modes

3. Tooling
   - grep, sed, tr, cut, sort, uniq, awk
   - When to use regex versus structured parser

4. Capture and substitution
   - Group references
   - Replacement workflows

## Teaching approach
- Extract structured info from one messy file.
- Rewrite one sed command to grep pipeline.
- Optimize one big regex for readability.
