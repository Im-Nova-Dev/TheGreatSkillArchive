# Cache Hierarchy

## Core topics

1. Structure
   - Levels and sizes
   - Latency and bandwidth

2. Mapping
   - Direct mapped and associative
   - Conflict misses

3. Optimization
   - Data layout and access order
   - False sharing avoidance

## Teaching approach
- Measure access patterns in one example.
- Refactor one loop for locality.
- Explain one unexpected slowdown.
