# Parallel Computing Fundamentals

## Core topics

1. concurrency models
   - Threads and processes
   - Green threads and event loops

2. Synchronization
   - Locks and mutexes
   - Semaphores and barriers
   - Atomic operations

3. Memory models
   - Sequential consistency
   - Happens before relationships

4. Parallel patterns
   - Fork-join
   - Map-reduce
   - Pipelines

5. Performance
  _ scalability and Amdahl law
   - Monitoring and hotspots

## Teaching approach
- Start with one threaded program.
- Introduce one race condition and one fix.
- Measure speedup on a small task.
