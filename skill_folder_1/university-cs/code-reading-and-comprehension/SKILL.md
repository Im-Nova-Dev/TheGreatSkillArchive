# Code Reading and Comprehension

## Core topics

1. Orientation
   - Entry points
   - Module boundaries

2. Tracing
   - Data and control flow
   - State transitions

3. Comprehension
   - Summary and mental model
   - Risk identification

## Teaching approach
- Read one new file end to end.
- Summarize one module in own words.
- Mark one confusing section for follow-up.
