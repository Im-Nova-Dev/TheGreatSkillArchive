# Software Project Management for Engineers

## Core topics

1. Planning
   - Estimation and milestones
   - Risk and dependency mapping

2. Delivery
   - Scope and change control
   - Progress reporting

3. Coordination
   - Team processes
   - Incident planning

## Teaching approach
- Create one project breakdown for one small feature.
- Assess one risk and mitigation plan.
- Propose one team coordination improvement.
