# Error Detection and Correction

## Core topics

1. Detection
   - Parity and checksums
   - CRC concept

2. Correction
   - Hamming codes concept
   - Redundancy tradeoffs

3. Usage
   - Network and storage
   - Retry versus FEC

## Teaching approach
- Compute one simple checksum.
- Compare one detection and correction scheme.
- Recommend one strategy for one channel.
