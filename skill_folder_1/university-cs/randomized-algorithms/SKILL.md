# Randomized Algorithms

## Core topics

1. Concepts
   - Randomization in algorithm design
   - Expectation and tail bounds

2. Data structures
   - Skip lists
   - Universal hashing

3. Applications
   - Quickselect
   - Load balancing with balls and bins

## Teaching approach
- Compare deterministic and randomized versions.
- Analyze one simple probability bound.
- Measure empirical behavior on example.
