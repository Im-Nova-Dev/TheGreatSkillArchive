# API Design Fundamentals

## Core topics

1. Styles
   - REST conventions
   - RPC and GraphQL

2. Design
   - Resources and actions
   - Error and status design

3. Evolution
   - Versioning
   - Backward compatibility

## Teaching approach
- Design one API for one use case.
- Review one endpoint for clarity.
- Propose one versioning policy.
