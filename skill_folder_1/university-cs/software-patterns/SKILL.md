# Software Patterns

## Core topics

1. Creational
   - Builder and prototype
   - Singleton usage

2. Structural
   - Adapter and decorator
   - Composite

3. Behavioral
   - Strategy and observer
   - Iterator and command

## Teaching approach
- Identify one pattern in existing code.
- Refactor one section toward a pattern.
- Discuss when pattern adds complexity.
