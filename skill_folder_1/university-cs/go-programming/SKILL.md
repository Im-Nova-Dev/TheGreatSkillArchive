# Go Programming

## Core topics

1. Language

   - Methods and interfaces

2. Concurrency
   - Goroutines and channels
   - Patterns

3. Practice

## Teaching approach
- Implement one simple concurrent server.
- Compare one channel versus mutex solution.
- Structure one Go module cleanly.
