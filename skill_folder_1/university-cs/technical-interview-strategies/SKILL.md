# Technical Interview Strategies

## Core topics

1. Problem solving
   - Reading and restating
   - Example-driven method

2. Communication
   - Explaining approach
   - Discussing tradeoffs

3. Preparation
   - Patterns and drills
   - Behavioral stories

## Teaching approach
- Solve one interview problem without coding first.
- Communicate one solution clearly.
- Prepare one behavioral story.
