# Human-Computer Interaction

## Core topics

1. Cognition
   - Attention and memory limits
   - Error and frustration

2. Interaction
   - Input and output modes
   - Responsiveness

3. Evaluation
   - Heuristic evaluation
   - Usability testing

## Teaching approach
- Audit one interface for cognitive load.
- Redesign one flow to reduce steps.
- Plan one small usability test.
