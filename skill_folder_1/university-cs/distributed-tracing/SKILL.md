# Distributed Tracing

## Core topics

1. Model
   - Traces and spans
   - Context propagation

2. Instrumentation
   - Sampling strategies
   - Baggage and metadata

3. Analysis
   - Finding slow spans
   - Correlating logs and metrics

## Teaching approach
- Trace one request across services.
- Identify one slow path visually.
- Propose one instrumentation improvement.
