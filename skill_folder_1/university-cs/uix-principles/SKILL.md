# UI Principles for Engineers

## Core topics

1. Structure
   - Layout and spacing
   - Visual hierarchy

2. Feedback
   - Error and loading states
   - Success and empty states

3. Accessibility
   - Color and contrast
   - Focus and keyboard support

## Teaching approach
- Review one UI for clarity.
- Improve one screen for one state.
- Compare two implementations.
