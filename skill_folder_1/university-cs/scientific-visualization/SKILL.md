# Scientific Visualization

## Core topics

1. Data types
   - Scalar and vector fields
   - Meshes and volumes

2. Techniques
   - Volume rendering
   - Flow visualization

3. Communication
   - Presentation quality
   - Reproducibility

## Teaching approach
- Choose one visualization technique for one dataset.
- Review one figure for clarity.
- Explain one rendering decision.
