# Competitive Programming

## Core topics

1. Problem solving
   - Reading and parsing constraints
   - Choosing data structures fast

2. Common patterns
   - Two pointers and sliding window
   - Prefix sums and difference arrays
   - Graph BFS and DFS patterns

3. Implementation
   - Template setup
   - Fast I/O and bounds checking

4. Contest strategy
   - Problem order and skip rules
   - Partial scoring
   - Debug time limits

## Teaching approach
- Solve one problem end to end.
- Analyze why one approach fails.
- Focus on correctness before speed.
