# Product Management for Engineers

## Core topics

1. Discovery
   - User interviews
   - Problem versus solution

2. Planning
   - Roadmaps and milestones
   - MVP scope

3. Metrics
   - North star metric
   - Leading and lagging indicators

## Teaching approach
- Turn one feature request into problem statement.
- Define metrics for one shipped feature.
- Write one lightweight product brief.
