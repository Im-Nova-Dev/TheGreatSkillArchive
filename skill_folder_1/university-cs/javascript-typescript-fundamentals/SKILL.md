# JavaScript and TypeScript Fundamentals

## Core topics

1. JavaScript
   - Syntax and async
   - Event loop

2. TypeScript
   - Type annotations
   - Structural typing

3. Practice
   - Modules and config
   - Error handling

## Teaching approach
- Convert one callback chain to async/await.
- Type one JavaScript module in TypeScript.
- Explain one event loop phase.
