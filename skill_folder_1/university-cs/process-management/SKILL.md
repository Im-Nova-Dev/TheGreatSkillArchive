# Process Management

## Core topics

1. Lifecycle
   - States and transitions
   - Creation and exit

2. Control
   - Fork and exec model
   - Signals

3. Practical
   - Process groups
   - Zombie and orphan handling

## Teaching approach
- Trace one shell command execution.
- Explain one fork failure scenario.
- Use one shell to inspect processes.
