# Embedded Linux

## Core topics

1. System bring-up
   - Boot process concept
   - Root filesystem concept

2. Build and toolchain
   - Cross-compilation
   - Yocto and Buildroot overview

3. Linux specifics
   - Device tree
   - Kernel and modules

## Teaching approach
- Diagram one embedded Linux boot flow.
- Compare Yocto and Buildroot.
- Configure one simple root filesystem requirement.
