# Load Balancing

## Core topics

1. Types
   - Layer 4 and layer 7
   - Proxy servers

2. Algorithms
   - Round robin
   - Least connections

3. Operations
   - Health checks
   - Traffic management

## Teaching approach
- Design one load-balanced topology.
- Choose one algorithm for one workload.
- Review one health check policy.
