# Ruby Fundamentals

## Core topics

1. Language
   - Blocks and iterators
   - Classes and modules

2. Style
   - Duck typing
   - Metaprogramming intuition

3. Ecosystem
   - Rails model basics
   - Testing habits

## Teaching approach
- Refactor one loop to idiomatic Ruby iteration.
- Explain one mixin versus inheritance choice.
- Sketch one small Rails-style interface.
