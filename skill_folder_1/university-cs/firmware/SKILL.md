# Firmware

## Core topics

1. Memory and boot
   - Memory map
   - Boot sequence

2. Execution
   - Interrupts
   - Peripheral access

3. Updates
   - OTA
   - Secure boot

## Teaching approach
- Map one MCU memory layout.
- Trace one boot sequence.
- Review one OTA update flow.
