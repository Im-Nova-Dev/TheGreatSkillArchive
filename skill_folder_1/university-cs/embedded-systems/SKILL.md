# Embedded Systems

## Core topics

1. Hardware
   - MCU architecture
   - Memory and peripherals

2. Software
   - Interrupts
   - Communication interfaces

3. Constraints
   - Power management
   - Real-time behavior

## Teaching approach
- Diagram one embedded system block diagram.
- Trace one interrupt flow.
- Evaluate one power management tradeoff.
