# Computer Vision Fundamentals

## Core topics

1. Image processing
   - Pixels, channels, color spaces
   - Filtering and transforms

2. Features
   - Edges and corners
   - Descriptors conceptually

3. Detection and segmentation
   - Object detection flow
   - Semantic segmentation

4. Deep learning
   - CNN architecture intuition
   - Transfer learning

## Teaching approach
- Start with one small image dataset.
- Apply one filter and explain result.
- Visualize feature maps where possible.
