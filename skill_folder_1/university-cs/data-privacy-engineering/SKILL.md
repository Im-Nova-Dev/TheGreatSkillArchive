# Data Privacy Engineering

## Core topics

1. Principles
   - Privacy by design
   - Data minimization

2. Techniques
   - Anonymization methods
   - Differential privacy

3. Compliance
   - Data subject rights
   - Impact assessments

## Teaching approach
- Review one system for privacy risks.
- Apply one anonymization technique.
- Design one consent flow.
