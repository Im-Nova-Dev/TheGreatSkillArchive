# Data Engineering

## Core topics

1. Pipelines
   - ETL and ELT
   - Batch versus streaming

2. Storage
   - Warehouses and lakes
   - Partitioning

3. Quality and governance
   - Validation
   - Catalog and lineage

## Teaching approach
- Design one simple data pipeline.
- Choose one storage architecture.
- Identify one data quality risk.
