# Mobile Computing Fundamentals

## Core topics

1. Platforms
   - iOS and Android concepts
   - App architecture

2. Constraints
   - Power and bandwidth
   - Sensor and location

3. UX
   - Interaction patterns
   - Offline behavior

## Teaching approach
- Compare platform concepts.
- Design one offline mobile experience.
- Identify one power usage issue.
