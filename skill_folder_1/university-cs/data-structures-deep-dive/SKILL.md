# Data Structures Deep Dive

## Core topics

1. Linear and associative
   - Arrays and linked lists
   - Hash tables

2. Hierarchical
   - Trees and heaps
   - Tries

3. Application
   - Graphs
   - Selection reasoning

## Teaching approach
- Implement one structure from scratch.
- Compare two structures for one workload.
- Explain one worst-case behavior.
