# Deep Learning

## Core topics

1. Networks
   - Perceptron to MLP
   - Activation and layers

2. Training
   - Loss and gradients
   - Backpropagation and optimizers

3. Architectures
   - CNN for images
   - RNN and attention
   - Transformer idea

## Teaching approach
- Build one small network from scratch conceptually.
- Trace one forward and backward pass.
- Compare CNN and transformer for one task.
