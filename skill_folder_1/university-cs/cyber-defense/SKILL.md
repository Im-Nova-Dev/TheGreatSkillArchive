# Cyber Defense

## Core topics

1. Defense model
   - Defense in depth
   - Zero trust ideas

2. Controls
   - Network segmentation
   - Secure baselines

3. Response
   - Detection and containment
   - Recovery planning

## Teaching approach
- Design one simple defense model.
- Review one configuration for hardened posture.
- Run one simple incident exercise.
