# Deployment Patterns

## Core topics

1. Strategies
   - Blue-green and canary
   - Rolling and recreate

2. Safety
   - Rollback design
   - Health checks and timeouts

3. Environments
   - Promotion model
   - Configuration drift

## Teaching approach
- Choose one deployment strategy for one release.
- Design one rollback path.
- Review environment gaps.
