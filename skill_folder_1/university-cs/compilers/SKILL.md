# Compilers

## Core topics

1. Analysis
   - Lexing and parsing
   - Semantic checks

2. IR and optimization
   - Intermediate representations
   - Optimization passes

3. Code generation
   - Instruction selection
   - Register allocation

## Teaching approach
- Parse one simple grammar.
- Review one optimization on IR.
- Trace one code generation step.
