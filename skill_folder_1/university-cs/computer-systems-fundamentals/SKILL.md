# Computer Systems Fundamentals

## Core topics

1. Execution
   - Fetch-decode-execute
   - Instruction concepts

2. Memory
   - Hierarchy concept
   - Caches and locality

3. Links and loads
   - Linking and loading
   - Performance analysis

## Teaching approach
- Trace one simple program execution.
- Explain one cache concept.
- Review one performance issue.
