# Programming Language Concepts

## Core topics
- Type systems and soundness
- Evaluation strategies and order
- Memory models and runtime behavior
- Syntax and semantics tradeoffs

## Teaching approach
- Trace one expression in two languages.
- Explain one type error prevented by design.
