# Probabilistic Graphical Models

## Core topics

1. Models
   - Directed and undirected graphs
   - Conditional independence

2. Inference
   - Exact versus approximate
   - Factors and messages

3. Learning
   - Parameter and structure learning
   - Data requirements

## Teaching approach
- Model one small domain with one graph.
- Identify one independence relation.
- Explain one inference difficulty.
