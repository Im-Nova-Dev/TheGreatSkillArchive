# Stream Processing

## Core topics

1. Stream concepts
   - Events and ordering
   - Windowing

2. State and correctness
   - State management
   - Exactly-once semantics

3. Architecture
   - Frameworks
   - Lambda and kappa

## Teaching approach
- Model one real-time aggregation.
- Choose one windowing strategy.
- Compare lambda versus kappa design.
