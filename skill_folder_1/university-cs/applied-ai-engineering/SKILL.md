# Applied AI Engineering

## Core topics

1. Model selection
   - Capability versus cost
   - Latency and memory tradeoffs

2. Prompt engineering
   - Instruction design
   - Few-shot and chain of thought

3. RAG
   - Chunking and retrieval
   - Context budget

## Teaching approach
- Design one prompt for one task.
- Add one retrieval step.
- Measure latency and answer quality.
