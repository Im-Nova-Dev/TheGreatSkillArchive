# Docker and Containers

## Core topics

1. Containers vs VMs
   - Isolation and namespaces
   - Layers and copy-on-write

2. Images
   - Dockerfile instructions
   - Caching and builds
   - Multi-stage builds

3. Runtime
   - Ports and volumes
   - Environment variables
   - Logs and exec

4. Compose
   - Services and dependencies
   - Networking between services
   - Profiles and overrides

## Teaching approach
- Containerize an existing app.
- Inspect layers and image size.
- Run the same image locally and in CI.
