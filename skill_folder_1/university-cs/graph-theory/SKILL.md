# Graph Theory

## Core topics

1. Representation
   - Adjacency list and matrix
   - Graph properties

2. Traversal
   - BFS and DFS
   - Connected components

3. Algorithms
   - Shortest paths
   - Spanning trees

## Teaching approach
- Represent one real-world network as a graph.
- Run one traversal algorithm.
- Explain one shortest path solution.
