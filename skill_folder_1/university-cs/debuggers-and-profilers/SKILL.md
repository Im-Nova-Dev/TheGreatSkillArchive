# Debuggers and Profilers

## Core topics

1. Debugging
   - Breakpoints and stepping
   - Stack inspection

2. Profiling
   - CPU and memory
   - Sampling versus instrumentation

3. Analysis
   - Flame graphs
   - Bottleneck identification

## Teaching approach
- Debug one crash using a debugger.
- Profile one program.
- Explain one flame graph result.
