# LLM Application Design

## Core topics

1. Prompt architecture
   - Instruction design
   - Context management

2. Integration
   - Retrieval and tools
   - Structured output

3. Operations
   - Evaluation and safety
   - Cost and latency

## Teaching approach
- Design one small LLM workflow.
- Evaluate one prompt for failure modes.
- Explain one latency tradeoff.
