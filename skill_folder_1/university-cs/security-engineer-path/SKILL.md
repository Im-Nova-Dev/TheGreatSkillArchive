# Security Engineering Path

## Core topics

1. Reviews
   - Threat model reviews
   - Architecture review

2. Hardening
   - Configuration baselines
   - Network and host controls

3. Response
   - Detection and triage
   - Containment and lessons learned

## Teaching approach
- Run one architecture review.
- Harden one sample host.
- Draft one incident report.
