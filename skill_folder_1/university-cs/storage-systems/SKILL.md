# Storage Systems

## Core topics

1. Storage types

   - Object storage

2. Distributed storage
   - Replication
   - Erasure coding

3. Performance
   - Caching and tiering

## Teaching approach
- Compare one local and distributed storage strategy.
- Evaluate one durability model.
- Review one storage performance metric.
