# Assembly Fundamentals

## Core topics

1. Execution
   - Registers and flags
   - Addressing modes

2. Control flow
   - Calls and returns
   - Stack usage

3. Practice
   - Reading disassembly
   - Tracing execution

## Teaching approach
- Read one compiled function in assembly.
- Trace one function call.
- Write one tiny assembly example.
