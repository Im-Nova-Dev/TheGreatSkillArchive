# Hardware Security Fundamentals

## Core topics

1. CPU and memory
   - Rings and privilege levels
   - DEP and ASLR

2. Boot and trust
   - Secure boot flow
   - Measured boot

3. Crypto hardware
   - TPM and secure enclaves
   - Key storage outside OS

4. Side channels
   - Timing and power analysis


## Teaching approach
- Trace one boot sequence.
- Identify one trust boundary.
- Explain why software checks alone are insufficient.
