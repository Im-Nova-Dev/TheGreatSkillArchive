# Systems Programming

## Core topics

1. Execution model
   - Processes and threads
   - Memory model

2. System interface
   - System calls
   - File descriptors

3. IPC
   - Shared memory and signals
   - Sockets concept

## Teaching approach
- Write one small C or Rust program using syscalls.
- Trace it with strace or ltrace.
- Explain one race if present.
