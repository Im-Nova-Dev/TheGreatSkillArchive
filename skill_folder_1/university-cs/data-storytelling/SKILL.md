# Data Storytelling

## Core topics

1. Narrative
   - Structure
   - Audience fit

2. Visualization
   - Chart choice
   - Annotation

3. Communication
   - Executive framing
   - Handling uncertainty

## Teaching approach
- Reframe one analysis as a story.
- Critique one chart for clarity.
- Draft one concise data insight summary.
