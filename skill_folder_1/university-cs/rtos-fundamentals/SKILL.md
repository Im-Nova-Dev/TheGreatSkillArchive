# RTOS Fundamentals

## Core topics

1. Tasks
   - Creation and lifecycle
   - Scheduling

2. Synchronization
   - Queues and semaphores
   - Mutexes

3. Timing
   - Timers and ticks
   - ISR considerations

## Teaching approach
- Design one simple RTOS task set.
- Compare schedulers.
- Identify one resource-sharing hazard.
