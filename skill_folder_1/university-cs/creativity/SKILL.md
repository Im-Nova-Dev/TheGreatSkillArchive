# Creativity

## Core topics

1. Ideation
   - Divergent thinking
   - Analogies

2. Techniques
   - Brainstorming
   - Constraint-based creativity

3. Evaluation
   - Idea screening
   - Development

## Teaching approach
- Brainstorm one solution space.
- Combine two existing ideas.
- Evaluate one novel approach.
