# Rust Fundamentals

## Core topics

1. Ownership and borrowing
   - Move and copy
   - Borrow checker

2. Types
   - Traits and generics
   - Pattern matching

3. Concurrency and tooling
   - Threads and async
   - Cargo workflow

## Teaching approach
- Explain one borrow checker error.
- Refactor one unsafe pattern to safe Rust.
- Use one trait effectively.
