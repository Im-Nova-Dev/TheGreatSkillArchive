# Operating Systems Internals

## Core topics

1. Kernel architecture
   - Monolithic versus microkernel concept
   - System call interface

2. Memory and process management
   - Virtual memory internals
   - Scheduling

3. File systems and IO
   - VFS concept
   - Buffer cache and journaling

## Teaching approach
- Trace one system call path.
- Explain one scheduler decision.
- Review one file system operation.
