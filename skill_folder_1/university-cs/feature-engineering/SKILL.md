# Feature Engineering

## Core topics

1. Transformation
   - Normalization and encoding
   - Handling missing and outliers

2. Selection
   - Relevance versus redundancy
   - Feature importance

3. Production
   - Feature stores
   - Online and offline consistency

## Teaching approach
- Engineer features for one dataset.
- Compare encoding strategies.
- Identify one leakage risk.
