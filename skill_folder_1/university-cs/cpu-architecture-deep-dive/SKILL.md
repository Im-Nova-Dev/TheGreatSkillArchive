# CPU Architecture Deep Dive

## Core topics

1. Pipeline
   - Stages and hazards
   - Forwarding and stalls

2. Memory
   - Cache hierarchy
   - Coherence and consistency

3. SIMD
   - Vector instructions concept
   - Usage in data processing

## Teaching approach
- Trace one loop through pipeline stages.
- Compare scalar and vectorized versions.
- Measure cache behavior conceptually.
