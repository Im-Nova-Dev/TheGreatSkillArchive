# Incremental Delivery

## Core topics

1. Planning
   - Slicing value
   - Reducing initial scope

2. Release
   - Staged rollout
   - Monitoring and checks

3. Improvement
   - Feedback collection
   - Iterating on evidence

## Teaching approach
- Slice one feature plan into increments.
- Define release criteria.
- Design one safe rollback path.
