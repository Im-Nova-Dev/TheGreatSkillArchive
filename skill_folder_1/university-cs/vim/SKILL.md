# Vim

## Core topics

1. Modes and movement
   - Normal, insert, visual
   - Motion commands

2. Editing
   - Operators and text objects
   - Undo and repeat

3. Advanced usage
   - Macros and registers
   - Window management

## Teaching approach
- Perform one edit entirely in normal mode.
- Record and replay one macro.
- Organize one editing session with windows.
