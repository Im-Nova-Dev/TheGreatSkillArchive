# OSI Model

## Core topics

1. Layers
   - Physical to application
   - Layer responsibilities

2. Mapping
   - OSI and TCP/IP
   - Protocols per layer

3. Troubleshooting
   - Layer-based diagnosis
   - Common symptoms

## Teaching approach
- Trace one packet through the stack.
- Diagnose one issue layer by layer.
- Map one protocol to OSI layers.
