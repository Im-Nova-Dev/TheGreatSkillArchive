# Applied Cryptography

## Core topics

1. Symmetric cryptography
   - AES and modes of operation
   - Key management

2. Asymmetric cryptography
   - RSA, ECC, Diffie-Hellman
   - Signatures and verification

3. Hashing and integrity
   - SHA family
   - HMAC and KDFs

4. Transport security
   - TLS conceptually
   - Certificate validation

## Teaching approach
- Create one example protocol using symmetric and asymmetric parts.
- Emphasize why implementation often breaks even when math is sound.
