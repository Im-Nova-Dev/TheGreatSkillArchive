# Penetration Testing Fundamentals

## Core topics

1. Methodology
   - Reconnaissance
   - Scanning and exploitation

2. Practice
   - Reporting
   - Remediation guidance

3. Ethics and scope
   - Legal boundaries
   - Responsible conduct

## Teaching approach
- Define one scoped test.
- Review one finding for exploit and impact.
- Write one remediation recommendation.
