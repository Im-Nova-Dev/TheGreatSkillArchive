# Algorithmic Problem Solving

## Core topics

1. Problem framing
   - Restate in own words
   - Identify input, output, constraints

2. Pattern recognition
   - Searching versus transforming
   - Greedy versus exhaustive search
   - Dynamic programming cues

3. Solution design
   - Brute force then optimize
   - Data structure selection
   - Complexity target

4. Verification
   - Walkthrough example
   - Edge case checklist

## Teaching approach
- Solve one problem without typing code first.
- Write one brute force solution.
- Refactor to optimized version.
