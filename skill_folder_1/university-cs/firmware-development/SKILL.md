# Firmware Development

## Core topics

1. Hardware constraints
   - Memory model
   - Timing

2. Software
   - Startup and drivers
   - Communication

3. Reliability
   - Updates
   - Safe failures

## Teaching approach
- Design one firmware startup flow.
- Debug one interrupt issue.
- Evaluate one update mechanism.
