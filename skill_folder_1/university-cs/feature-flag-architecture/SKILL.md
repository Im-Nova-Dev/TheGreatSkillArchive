# Feature Flag Architecture

## Core topics

1. Design
   - Boolean versus percentage flags
   - Targeting rules

2. Release
   - Gradual rollout
   - Emergency control

3. Lifecycle
   - Debt and cleanup
   - Experimentation

## Teaching approach
- Design one flag strategy for one feature.
- Define one targeting rule.
- Propose one cleanup policy.
