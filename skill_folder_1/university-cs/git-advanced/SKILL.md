# Advanced Git

## Core topics

1. Rewriting
   - Interactive rebase
   - Commit amendment

2. Recovery
   - Reflog and bisect
   - Orphan branches

3. Advanced features
   - Worktrees and submodules
   - Hooks and rerere

## Teaching approach
- Practice one advanced move safely.
- Recover from one mistaken rewrite.
- Review one repository needing cleanup.
