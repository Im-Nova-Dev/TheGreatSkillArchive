# Debugging Techniques

## Core topics

1. Reproduction
   - Minimal case creation
   - Environment isolation

2. Inspection
   - Logging and tracing
   - Breakpoints and watchpoints
   - Memory and state dumps

3. Root cause analysis
   - Five whys
   - Change bisect
   - Hypothesis testing

4. Fix and prevent
   - Correct cause, not symptom
   - Regression test and note

## Teaching approach
- Debug one failing function without editor.
- Show why initial hypothesis was wrong.
- Add one permanent guard before final.
