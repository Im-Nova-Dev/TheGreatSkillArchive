# Linux Kernel Fundamentals

## Core topics

1. Kernel overview
   - Monolithic design
   - System call interface

2. Process and memory
   - Scheduling classes
   - Paging and caches

3. Filesystems and IO
   - VFS layer
   - Block and network filesystems

4. Modules
   - Building and loading
   - Debugging kernel code

## Teaching approach
- Read one kernel config option.
- Trace one system call with strace.
- Explain one scheduling event conceptually.
