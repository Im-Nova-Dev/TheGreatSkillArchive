# Programming Language Theory

## Core topics

1. Formal foundations
   - Syntax and semantics
   - Lambda calculus

2. Implementation
   - Parsing and interpretation
   - Compilation concepts

3. Design
   - Language tradeoffs
   - Feature selection

## Teaching approach
- Write one simple interpreter.
- Compare two language designs.
- Explain one semantic concept.
