# Perl Fundamentals

## Core topics

1. Data and syntax
   - Scalars and arrays
   - Hashes and references

2. Text
   - Regular expressions
   - Pattern matching

3. Practice
   - Modules and CPAN
   - Scripting patterns

## Teaching approach
- Parse one text file with regex.
- Refactor one script to use references.
- Explain one Perl data structure choice.
