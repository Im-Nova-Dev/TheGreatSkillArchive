# Instruction Pipelines

## Core topics

1. Stages
   - Fetch, decode, execute
   - Memory and write-back

2. Hazards
   - Data hazards
   - Control hazards

3. Advanced
   - Branch prediction
   - Out-of-order execution

## Teaching approach
- Trace one instruction through pipeline stages.
- Identify one hazard.
- Explain one stall versus forward choice.
