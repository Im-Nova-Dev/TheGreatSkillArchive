# Internet Protocols Fundamentals

## Core topics

1. Core protocols
   - IP and transport
   - Application protocols

2. Connection and security
   - TLS and QUIC
   - Handshakes and state

3. Local networking
   - Ethernet and Wi-Fi
   - ARP and DHCP

## Teaching approach
- Trace one web request across layers.
- Explain one protocol handshake.
- Review one local network scenario.
