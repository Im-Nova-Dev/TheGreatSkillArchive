# Code Review

## Core topics

1. Scope
   - What to review first
   - Review checklist use

2. Feedback
   - Tone and clarity
   - Specific and actionable comments

3. Practice
   - Reviewing design
   - Moving PRs forward

## Teaching approach
- Review one PR using a checklist.
- Rewrite one comment to be more Constructive.
- Discuss one review debate.
