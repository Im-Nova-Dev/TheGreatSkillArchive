# Low-Level Programming

## Core topics

1. Memory model
   - Layout and addresses
   - Pointers and references

2. Performance
   - Cache awareness
   - Instruction-level parallelism

3. Systems programming
   - System calls
   - Embedded constraints

## Teaching approach
- Trace one memory layout.
- Optimize one hot path.
- Explain one calling convention.
