# Schema Design

## Core topics

1. Table and key design
   - Columns and types
   - Primary and foreign keys

2. Integrity
   - Constraints
   - Referential rules

3. Evolution
   - Schema migrations
   - Versioning

## Teaching approach
- Design one schema from one brief.
- Add one safe migration.
- Review one schema for future change risk.
