# Legacy Maintenance

## Core topics

1. Understanding
   - Reading old systems
   - Recovering lost knowledge

2. Safety
   - Characterization tests
   - Change boundaries

3. Improvement
   - Strangler fig
   - Incremental modernization

## Teaching approach
- Inspect one legacy module and summarize behavior.
- Add one characterization test.
- Propose one safe modernization step.
