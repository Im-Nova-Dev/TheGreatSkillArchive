# State Machines

## Core topics

1. Model
   - States and transitions
   - Events and actions

2. Advanced
   - Hierarchical state
   - Guards and history

3. Use
   - Protocol modeling
   - UI and driver logic

## Teaching approach
- Model one system as a state machine.
- Draw one lifecycle and identify invalid moves.
- Implement one simple state machine.
