# Databases

## Core topics

1. Relational model
   - Tables and keys
   - Schema design

2. SQL
   - Queries and joins
   - Aggregation and subqueries

3. Performance
   - Indexes and plans
   - Normalization

4. NoSQL
   - Document and key-value tradeoffs

## Teaching approach
- Design one schema for one dataset.
- Write SQL queries step by step.
- Explain why one query is slow.
