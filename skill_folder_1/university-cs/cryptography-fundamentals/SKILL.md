# Cryptography Fundamentals

## Core topics

1. Symmetric cryptography
   - Stream ciphers and block ciphers
   - Modes of operation: ECB/CBC/GCM
   - Key length and security

2. Asymmetric cryptography
   - RSA and Diffie-Hellman
   - Elliptic curve intuition
   - Public key infrastructure

3. Hashes and integrity
   - Collision resistance
   - HMAC and key derivation

4. Protocols
   - TLS conceptually
   - SSH and certificate checks

## Teaching approach
- Start with one real attack story.
- Show control the protocol should have provided.
- Explain why weaker options fail.
