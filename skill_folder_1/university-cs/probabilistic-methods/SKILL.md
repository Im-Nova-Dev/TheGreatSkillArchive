# Probabilistic Methods

## Core topics

1. Basics
   - Linearity of expectation
   - Union and tail bounds

2. Existence proofs
   - Probabilistic existence
   - Alterations

3. Applications
   - Randomized rounding
   - Hashing and load balancing

## Teaching approach
- Show one proof using expectation.
- Simulate one randomized construction.
- Compare deterministic and probabilistic bounds.
