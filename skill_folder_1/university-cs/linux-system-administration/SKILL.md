# Linux System Administration

## Core topics

1. Users and permissions
   - Users, groups, sudo
   - File permissions and ownership
   - umask and ACLs

2. Processes
   - ps, top, htop
   - Signals and job control
   - Systemd units

3. Networking
   - ip and netstat
   - iptables basics
   - SSH and keys

4. Storage
   - Filesystem hierarchy
   - Mounts and fstab
   - Disk usage

5. Logs and troubleshooting
   - journald and syslog
   - grep, tail, less
   - Common boot issues

## Teaching approach
- Start with one real server or VM.
- Make one change and observe its effect.
- Keep a changelog during exercises.
