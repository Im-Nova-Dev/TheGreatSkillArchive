# Statistical Learning

## Core topics

1. Estimation
   - Bias-variance
   - Regularization concept

2. Models
   - Linear and nonlinear
   - Smoothing concepts

3. Validation
   - Cross-validation
   - Model selection

## Teaching approach
- Fit one simple linear model.
- Explain one bias-variance example.
- Compare one model selection decision.
