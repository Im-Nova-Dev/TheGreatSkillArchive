# Bioinformatics Fundamentals

## Core topics

1. Biological sequences
   - DNA, RNA, protein
   - Alphabet and encoding

2. Alignment
   - Global and local alignment
   - Scoring matrices

3. Databases and search
   - Sequence repositories
   - BLAST concept

4. Analysis
   - Gene finding concepts
   - Phylogenetics basics

## Teaching approach
- Align two short sequences by hand.
- Compare exact and heuristic search.
- Show one real dataset output.
