# Threat Modeling

## Core topics

1. Assets and adversaries
   - Critical assets
   - Threat actors

2. Surfaces
   - Entry points
   - Trust boundaries

3. Mitigation
   - Control design
   - Residual risk

## Teaching approach
- Map one threat surface for one system.
- Apply one threat modeling method.
- Propose one mitigation per threat.
