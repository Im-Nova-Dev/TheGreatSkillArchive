# Design Thinking

## Core topics

1. Framing
   - User and problem
   - Constraints

2. Ideation and prototyping
   - Sketching concepts
   - Rapid prototyping

3. Testing
   - Feedback
   - Iteration

## Teaching approach
- Reframe one problem from a user view.
- Sketch one solution concept.
- Run one quick prototype test.
