# Documentation Design

## Core topics

1. Structure
   - Audience and scope
   - Outline and flow

2. Content choices
   - Examples and diagrams
   - Task versus reference focus

3. Maintenance
   - Ownership and review
   - Versioning

## Teaching approach
- Redesign one document outline.
- Evaluate one example choice.
- Plan one document review cycle.
