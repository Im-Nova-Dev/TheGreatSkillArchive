# Shell Scripting and CLI

## Core topics

1. Shell concepts
   - Interactive versus script mode
   - Exit status and signals

2. Scripting
   - Variables and control flow
   - Functions and traps
   - Safe quoting

3. Tooling
   - grep, sed, awk, xargs
   - find, sort, uniq

## Teaching approach
- Automate one repeatable manual task.
- Compare two implementation choices.
- Debug one failing script end to end.
