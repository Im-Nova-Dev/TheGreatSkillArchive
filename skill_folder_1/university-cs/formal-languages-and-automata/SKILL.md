# Formal Languages and Automata

## Core topics

1. Regular languages
   - Finite automata and regex
   - Pumping lemma concept

2. Context-free languages
   - Pushdown automata
   - Parse table concept

3. Turing machines
   - Computation model
   - Universal machine

4. Decidability
   - Halting problem concept
   - Recognizer versus decider

## Teaching approach
- Convert one simple regex to NFA and DFA.
- Show one language that needs a stack.
- Explain undecidability with small examples.
