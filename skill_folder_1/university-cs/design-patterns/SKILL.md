# Design Patterns

## Core topics

1. Creational patterns
   - Builder, prototype, singleton
   - Factory method and abstract factory

2. Structural patterns
   - Adapter, decorator, facade
   - Composite and proxy

3. Behavioral patterns
   - Strategy, observer, command
   - Iterator and state

## Teaching approach
- Identify one pattern in a given snippet.
- Refactor one code region toward a pattern with a clear motivation.
- Discuss one case where abstraction adds complexity.
