# Compression

## Core topics

1. Lossless
   - Entropy concept
   - Run length and dictionary

2. Lossy
   - Transform coding idea
   - Perceptual criteria

3. Practice
   - Choosing algorithm

## Teaching approach
- Compare compression ratios for one dataset.
- Explain why one file resists compression.
- Choose one codec for one use case.
