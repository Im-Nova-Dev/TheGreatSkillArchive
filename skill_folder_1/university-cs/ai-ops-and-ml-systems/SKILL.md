# AI Ops and ML Systems

## Core topics

1. Pipelines
   - Data ingestion and transformation
   - Training and validation

2. Serving
   - Model deployment
   - Batch and online inference

3. Monitoring
   - Performance and drift

## Teaching approach
- Design one model deployment workflow.
- Identify one monitoring signal.
- Plan one canary release for model updates.
