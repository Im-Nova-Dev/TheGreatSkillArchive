# Reversible Computing

## Core topics

1. Principles
   - Logical reversibility
   - Landauer principle

2. Models
   - Reversible Turing machine
   - Circuit concepts

3. Practice
   - Energy implications
   - Design challenges

## Teaching approach
- Compare one reversible versus irreversible operation.
- Explain one thermodynamic bound.
- Review one practical challenge.
