# HPC Fundamentals

## Core topics

1. Architecture
   - Shared and distributed
   - Accelerators

2. Parallelism
   - Speedup and efficiency
   - Strong and weak scaling

3. Practice
   - Programming models basics
   - Profiling

## Teaching approach
- Estimate one speedup using Amdahl's law.
- Compare shared and distributed models.
- Profile one parallel kernel.
