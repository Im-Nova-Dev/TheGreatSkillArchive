# Accessibility and Inclusive Design

## Core topics

1. Principles
   - Perceivable, operable, understandable, robust
   - WCAG levels

2. Techniques
   - Semantic HTML
   - ARIA usage
   - Focus management

3. Testing
   - Automated and manual testing
   - Screen reader validation

## Teaching approach
- Audit one interface for accessibility issues.
- Fix one violation at a time.
- Compare before and after accessibility tree.
