# Concurrency

## Core topics

1. Execution
   - Threads and processes
   - Context switching

2. Synchronization
   - Locks and atomics
   - Barriers

3. Correctness
   - Race conditions
   - Deadlocks and livelocks

## Teaching approach
- Identify one race in example code.
- Repair one concurrency bug.
- Explain one memory ordering issue.
