# Compilers and Languages

## Core topics

1. Front end
   - Lexing and parsing
   - Semantic analysis and type checking

2. Middle and back
   - Intermediate representation
   - Optimization
   - Code generation

3. Runtime
   - Memory management
   - Exception and debugging support

## Teaching approach
- Build one tiny language incrementally.
- Explain each stage's responsibility.
- Debug one generated code example.
