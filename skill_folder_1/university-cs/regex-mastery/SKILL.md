# Regex Mastery

## Core topics

1. Engine behavior
   - Backtracking model
   - Greedy versus lazy quantifiers

2. Performance
   - Catastrophic backtracking
   - Possessive and atomic groups

3. Advanced patterns
   - Lookaround
   - Recursive patterns

4. Practical parsing
   - Structured text extraction
   - Code-driven generation

## Teaching approach
- Compare regex engines on one hard example.
- Rewrite backtracking regex to linear time.
- Teach reading regex as path, not magic.
