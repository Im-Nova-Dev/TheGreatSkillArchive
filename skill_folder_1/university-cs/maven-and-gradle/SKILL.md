# Maven and Gradle

## Core topics

1. Maven
   - Project and POM model
   - Lifecycle and plugins

2. Gradle
   - Build scripts
   - Incremental builds

3. Practice
   - Dependency management
   - Build optimization

## Teaching approach
- Analyze one build configuration.
- Compare Maven and Gradle for one project.
- Diagnose one build performance issue.
