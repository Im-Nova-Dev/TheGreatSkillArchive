# R Fundamentals

## Core topics

1. Data
   - Vectors and data frames
   - Data import

2. Analysis
   - Statistical testing
   - Modeling

3. Visualization
   - ggplot2 concepts

## Teaching approach
- Analyze one small dataset in R.
- Create one ggplot2 visualization.
- Perform one statistical test.
