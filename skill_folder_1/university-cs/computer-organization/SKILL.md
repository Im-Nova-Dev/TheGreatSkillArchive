# Computer Organization

## Core topics

1. Logic and datapath
   - ALU concept
   - Registers and buses

2. Memory mapping
   - RAM, ROM, and memory map
   - Access patterns

3. I/O
   - Interrupts and DMA
   - Device interface concepts

## Teaching approach
- Trace one data movement across registers and memory.
- Explain one DMA benefit.
- Review one hardware-software interface issue.
