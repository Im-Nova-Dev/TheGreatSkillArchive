# Machine Learning

## Core topics

1. Learning types
   - Supervised and unsupervised
   - Features and labels

2. Models
   - Linear models
   - Trees and ensembles

3. Evaluation
   - Train/validation/test
   - Bias and variance
   - Metrics

## Teaching approach
- Fit one baseline model first.
- Compare models on one validation set.
- Explain one error type clearly.
