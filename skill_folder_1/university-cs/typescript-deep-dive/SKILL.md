# TypeScript Deep Dive

## Core topics

1. Type system
   - Structural typing
   - Union and intersection
   - Type widening and narrowing

2. Advanced types
   - Generics and constraints
   - Conditional and mapped types
   - Template literals

3. Design
   - Discriminated unions
   - Type-safe APIs

## Teaching approach
- Refactor one JavaScript module to type-safe TypeScript.
- Replace hand checks with type constraints.
- Review compiler errors as design signals.
