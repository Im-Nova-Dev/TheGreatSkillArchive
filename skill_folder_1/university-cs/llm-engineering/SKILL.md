# LLM Engineering

## Core topics

1. Model behavior
   - Context and tokens
   - Latency and memory

2. System design
   - RAG and retrieval
   - Tool use and agents

3. Safety and policy
   - Guardrails
   - Output filtering

## Teaching approach
- Design one RAG flow.
- Identify one hallucination risk.
- Propose safety controls.
