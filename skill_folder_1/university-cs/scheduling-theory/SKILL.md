# Scheduling Theory

## Core topics

1. CPU scheduling
   - FCFS, SJF, RR, priority
   - Multilevel feedback

2. Real-time
   - RMS and EDF
   - Deadline misses

3. Practical
   - Linux CFS concept
   - Priority inversion avoidance

## Teaching approach
- Simulate one scheduling scenario.
- Compare algorithms on one workload.
- Measure latency impact of context switches.
