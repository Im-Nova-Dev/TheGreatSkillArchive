# Data Security

## Core topics

1. Classification and handling
   - Sensitive data
   - Storage and transit

2. Controls
   - Encryption
   - Access policy

3. Lifecycle
   - Retention
   - Disposal

## Teaching approach
- Classify one dataset.
- Review one data flow for exposure.
- Propose one encryption and access policy.
