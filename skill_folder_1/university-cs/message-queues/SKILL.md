# Message Queues

## Core topics

1. Models
   - Queues and topics
   - Fanout and routing

2. Delivery
   - At-least-once versus exactly-once
   - Ordering constraints

3. Operations
   - Dead-letter queues
   - Monitoring and replay

## Teaching approach
- Model one workload in queues.
- Compare delivery guarantees.
- Review one retry strategy.
