# Formal Methods

## Core topics

1. Specification
   - Contract-based specification

2. Verification
   - Model checking
   - Theorem proving concepts

3. Application

## Teaching approach
- Write one simple specification.
- Model check one small state machine.
- Explain one invariant concept.
