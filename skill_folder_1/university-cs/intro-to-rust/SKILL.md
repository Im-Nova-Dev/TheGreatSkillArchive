# Intro to Rust

## Core topics

1. Ownership and borrowing
   - Move semantics
   - References and lifetimes

2. Types and pattern matching
   - structs and enums
   - match and if let

3. Error handling
   - Result and Option
   - Combinators

4. Tooling
   - Cargo and crates
   - Clippy and rustfmt

## Teaching approach
- Start with one small CLI tool.
- Show one borrow error and fix.
- Compare Rust pattern to Python or C version.
