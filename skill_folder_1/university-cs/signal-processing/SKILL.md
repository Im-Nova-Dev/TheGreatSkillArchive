# Signal Processing

## Core topics

1. Signals and systems
   - Continuous and discrete
   - Sampling and reconstruction

2. Analysis
   - Fourier transform concept
   - Frequency domain

3. Filtering
   - FIR and IIR
   - Filter design

## Teaching approach
- Visualize one signal transformation.
- Design one simple filter.
- Explain one sampling theorem example.
