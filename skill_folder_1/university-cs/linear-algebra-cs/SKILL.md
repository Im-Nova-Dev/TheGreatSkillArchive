# Linear Algebra for CS

## Core topics

1. Vectors and matrices
   - Dimensions, transpose, addition, multiplication
   - Dot product and norms

2. Linear transformations
   - Matrix representation
   - Kernel and image

3. Systems of equations
   - Row reduction and Gaussian elimination
   - Existence and uniqueness

4. Eigenvalues and eigenvectors
   - Meaning and examples
   - Diagonalization concept

5. Applications
   - Graphics transforms
   - PageRank intuition
   - PCA and SVD in data

## Teaching approach
- Use small numeric examples.
- Emphasize geometric intuition.
- Show connection to code or systems.
