# Computational Complexity

## Core topics

1. Asymptotic notation
   - Big-O, Omega, Theta
   - Common complexity classes

2. Analysis methods
   - Recurrence relations
   - Amortized analysis
   - Probabilistic analysis

3. Complexity classes
   - P and NP intuition
   - NP-complete examples
   - Reductions conceptually

4. Intractability
   - When brute force is unavoidable
   - Approximation as escape

## Teaching approach
- Analyze one algorithm per session.
- Compare brute force versus optimized solution.
- Explain proof concept from textbook examples.
