# NoSQL Fundamentals

## Core topics

1. Models
   - Document and key-value
   - Graph and column stores

2. Consistency and distribution
   - CAP
   - Replication

3. Data modeling
   - Access-driven design
   - Query patterns

## Teaching approach
- Compare one SQL and one NoSQL design.
- Choose one NoSQL model for one use case.
- Evaluate one consistency tradeoff.
