# Computer Architecture

## Core topics

1. CPU design
   - Datapath and control
   - ISA overview

2. Memory
   - Hierarchy
   - Caching and virtual memory

3. Performance
   - Pipelining hazards
   - Metrics

## Teaching approach
- Trace one instruction through one pipeline stage.
- Explain one cache hit versus miss.
- Analyze one performance metric.
