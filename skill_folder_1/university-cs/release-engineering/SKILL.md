# Release Engineering

## Core topics

1. Build and artifacts
   - Reproducible builds
   - Artifact signing

2. Release process
   - Cadence and approval
   - Rollback readiness

3. Communication
   - Release notes
   - Change management

## Teaching approach
- Review one release pipeline.
- Identify one missing control.
- Draft one release checklist item.
