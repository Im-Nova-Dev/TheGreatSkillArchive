# Object-Oriented Design

## Core topics

1. Core concepts
   - Classes and objects
   - Inheritance versus composition

2. Principles
   - SOLID ideas
   - Interfaces and dependency direction

3. Practice
   - Cohesion and coupling
   - Refactoring examples

## Teaching approach
- Refactor one procedural module to OOP.
- Identify one inheritance misuse.
- Rewrite one tightly coupled example.
