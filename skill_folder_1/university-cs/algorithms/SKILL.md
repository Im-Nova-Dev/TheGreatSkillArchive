# Algorithms

## Core topics

1. Core techniques
   - Sorting and searching
   - Graph traversal

2. Strategies
   - Dynamic programming concept
   - Greedy choice

3. Analysis
   - Complexity
   - Correctness intuition

## Teaching approach
- Solve one problem using two approaches.
- Prove one greedy choice property.
- Optimize one brute force solution.
