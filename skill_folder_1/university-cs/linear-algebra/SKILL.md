# Linear Algebra

## Core topics

1. Vectors and matrices
   - Operations and shapes
   - Dot product and norms

2. Systems and spaces
   - Linear systems
   - Rank and span

3. Advanced topics
   - Eigenvalues and PCA idea
   - Singular value decomposition concept

## Teaching approach
- Compute one small matrix problem by hand.
- Explain one geometric intuition.
- Connect one concept to application.
