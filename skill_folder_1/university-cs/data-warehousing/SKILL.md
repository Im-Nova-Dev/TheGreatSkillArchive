# Data Warehousing

## Core topics

1. Modeling
   - Facts and dimensions
   - Star and snowflake schemas

2. Loading
   - ETL and ELT
   - Incremental loads

3. Operations
   - Partitioning and clustering
   - Performance tuning

## Teaching approach
- Design one star schema.
- Identify one dimension choice.
- Review one query plan.
