# Formal Verification Basics

## Core topics

1. Logic
   - Predicates and quantifiers
   - Preconditions and postconditions

2. Hoare logic
   - Triple notation
   - Assignment and composition rules

3. Invariants
   - Loop invariants
   - Induction

4. Properties
   - Safety and liveness
   - Termination

5. Tools
   - Model checking concept
   - Prover concept

## Teaching approach
- Verify one small program function.
- State invariants before loops.
- Compare testing and proving.
