# Software Design Principles

## Core topics

1. Principles
   - SOLID overview
   - Simplicity and reuse

2. Structure
   - Cohesion and coupling
   - Layering

3. Practice
   - Review one design
   - Refactor one module

## Teaching approach
- Review one module against SOLID.
- Refactor one smell toward simpler design.
- Explain one principle tradeoff.
