# Theory of Computation

## Core topics

1. Automata
   - Finite automata
   - Regular languages and regex

2. Formal languages
   - Pushdown automata
   - Parsing idea

3. Computability
   - Turing machines
   - Undecidability

4. Complexity
   - P and NP intuition
   - Reduction concept

## Teaching approach
- Convert one regex to automaton.
- Show one language needing a stack.
- Explain one impossibility result.
