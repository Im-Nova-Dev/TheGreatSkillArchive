# HTTP Protocol

## Core topics

1. Request and response
   - Methods and status codes
   - Headers and bodies

2. Versions
   - HTTP/1.1 and HTTP/2
   - HTTP/3 concepts

3. Caching and REST
   - Cache headers
   - Resource design

## Teaching approach
- Inspect one real HTTP request.
- Explain one status code family.
- Compare one resource design.
