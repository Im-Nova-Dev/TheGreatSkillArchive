# Technical Writing

## Core topics

1. Principles
   - Audience and scope
   - Clear explanations

2. Documentation types
   - README and guides
   - Reference and tutorials

3. Maintenance
   - Ownership and review
   - Keeping docs current

## Teaching approach
- Rewrite one document for clarity.
- Add one concrete example to one guide.
- Review one README for structure.
