# Network Fundamentals

## Core topics

1. Models and layers
   - OSI and TCP-IP
   - Encapsulation

2. Core protocols
   - IP, TCP, UDP, DNS
   - HTTP basics

3. Practice
   - Socket concepts
   - Troubleshooting

## Teaching approach
- Map one request to protocol layers.
- Inspect one packet capture excerpt.
- Diagnose one connectivity issue.
