# Scientific Computing

## Core topics

1. Numerical methods
   - Error and stability
   - Linear algebra

2. Solvers
   - ODEs and PDEs
   - Discretization

3. Practice
   - Libraries
   - Reproducibility

## Teaching approach
- Solve one simple numerical problem.
- Analyze one numerical error source.
- Compare numerical stability of two schemes.
