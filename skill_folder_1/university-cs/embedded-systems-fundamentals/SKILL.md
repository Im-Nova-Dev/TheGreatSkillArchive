# Embedded Systems Fundamentals

## Core topics

1. Hardware interface
   - MCU architecture
   - GPIO and timers

2. Software model
   - Interrupts
   - Communication interfaces

3. Constraints
   - Real-time requirements
   - Power management

## Teaching approach
- Diagram one simple embedded system.
- Trace one interrupt handler path.
- Evaluate one real-time requirement.
