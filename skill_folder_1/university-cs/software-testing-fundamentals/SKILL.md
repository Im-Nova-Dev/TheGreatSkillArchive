# Software Testing Fundamentals

## Core topics

1. Test levels
   - Unit through acceptance
   - Scope and isolation

2. Design
   - Equivalence and boundary
   - Combinatorial testing

3. Automation
   - Frameworks and CI
   - Test maintenance

## Teaching approach
- Write one unit test.
- Run one coverage report.
- Refactor one test for maintainability.
