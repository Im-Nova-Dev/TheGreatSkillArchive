# Data Governance

## Core topics

1. Ownership and policy
   - Ownership model
   - Access rules

2. Quality and catalog
   - Data quality
   - Metadata and lineage

3. Compliance
   - Regulations
   - Stewardship

## Teaching approach
- Map one data ownership model.
- Review one dataset for quality risks.
- Propose one governance rule.
