# Compiler Construction

## Core topics

1. Lexing
   - Regular expressions to lexer
   - Token streams

2. Parsing
   - Grammar and ambiguity
   - Recursive descent and table-driven parsers

3. Semantic analysis
   - Symbol tables
   - Type checking

4. Intermediate representation
   - ASTs and IRs
   - Static single assignment

5. Optimization
   - Constant folding
   - Dead code elimination

6. Code generation
   - Calling conventions
   - Register allocation

## Teaching approach
- Build a small language in phases.
- Show one clear transform per phase.
- Test each phase independently.
