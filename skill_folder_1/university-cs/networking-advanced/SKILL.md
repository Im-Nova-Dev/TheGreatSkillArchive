# Advanced Networking

## Core topics

1. Routing and switching
   - Static and dynamic routing
   - VLANs and trunks

2. Transport and application
   - HTTP/2 and gRPC
   - WebSockets and SSE

3. Performance and reliability
   - Load balancing algorithms
   - Retries and circuit breaking

4. Security
   - TLS termination
   - Firewall and proxy roles

5. Observability
   - Metrics and tracing
   - Logs and correlation

## Teaching approach
- Draw one request path with failure points.
- Add one optimization per iteration.
- Use packet captures to confirm behavior.
