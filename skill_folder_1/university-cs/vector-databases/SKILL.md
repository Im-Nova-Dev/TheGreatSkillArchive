# Vector Databases

## Core topics

1. Search

   - Distance metrics

2. Indexing

   - IVF

3. Operations

## Teaching approach
- Choose one index for one dataset.
- Explain one recall versus latency tradeoff.
- Design one hybrid search strategy.
