# Computer Engineering Fundamentals

## Core topics

1. Logic and architecture
   - Digital design basics
   - Processor concept

2. Memory and IO
   - Memory systems
   - Bus and I/O interfaces

3. Embedded systems
   - Constraints and tradeoffs
   - Codesign

## Teaching approach
- Review one small system design.
- Trace one memory access.
- Explain one embedded system constraint.
