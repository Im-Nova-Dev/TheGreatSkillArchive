# Machine Learning Fundamentals

## Core topics

1. Learning problems
   - Supervised, unsupervised, reinforcement
   - Features, labels, loss

2. Linear models
   - Linear regression
   - Logistic regression
   - Regularization: ridge and lasso

3. Trees and ensembles
   - Decision trees and overfitting
   - Random forests
   - Gradient boosting

4. Evaluation
   - Train/validation/test split
   - Cross-validation
   - Metrics: accuracy, precision, recall, AUC

5. Bias and variance
   - Underfitting and overfitting
   - Model selection

6. Deep learning basics
   - Neural network intuition
   - Activation and layers
   - Training loop concept

## Teaching approach
- Start with a dataset and a question.
- Fit a simple baseline first.
- Add complexity only when justified.
