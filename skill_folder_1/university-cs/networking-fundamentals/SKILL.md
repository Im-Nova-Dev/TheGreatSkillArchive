# Networking Fundamentals

## Core topics

1. Models
   - OSI and TCP/IP layers
   - Encapsulation

2. Protocols
   - IP, TCP, UDP
   - HTTP and DNS

3. Practice
   - Socket programming
   - Troubleshooting

## Teaching approach
- Trace one request across layers.
- Inspect one packet capture.
- Diagnose one connectivity issue.
