# Blockchain and Distributed Ledgers

## Core topics

1. Ledger structure
   - Blocks and chain
   - Hashing and immutability

2. Consensus
   - Proof of work and stake
   - Finality conceptually

3. Smart contracts
   - State transitions
   - Gas and costs

4. Tradeoffs
   - Decentralization vs throughput
   - Permissioned vs permissionless

## Teaching approach
- Draw one block and its links.
- Trace a transaction end to end.
- Compare permissioned and permissionless options.
