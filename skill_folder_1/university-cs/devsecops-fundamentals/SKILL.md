# DevSecOps Fundamentals

## Core topics

1. Shift left
   - Security requirements early
   - Secure defaults

2. Pipeline security
   - Dependency scanning
   - Build provenance

3. Runtime
   - Runtime monitoring
   - Incident response integration

## Teaching approach
- Review one pipeline for security gaps.
- Add one security check to one stage.
- Rewrite one requirement as security requirement.
