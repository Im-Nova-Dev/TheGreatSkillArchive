# NLP Engineering

## Core topics

1. Text processing
   - Tokenization and normalization
   - Feature extraction

2. Models
   - Word embeddings
   - Sequence models

3. Application
   - Text classification
   - NER and relation extraction

## Teaching approach
- Build one text classification pipeline.
- Compare two embedding approaches.
- Evaluate one model on one dataset.
