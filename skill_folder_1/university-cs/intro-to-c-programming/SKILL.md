# Intro to C Programming

## Core topics

1. Compilation and execution
   - Preprocessor, compile, link
   - Object files and headers

2. Memory model
   - Stack and heap
   - Pointers and addresses

3. Data structures
   - Arrays and strings
   - Structs and unions

4. Standard library
   - stdio and stdlib
   - Memory functions and error handling

## Teaching approach
- Trace one simple program in memory.
- Show segmentation fault causes.
- Build one linked list from scratch.
