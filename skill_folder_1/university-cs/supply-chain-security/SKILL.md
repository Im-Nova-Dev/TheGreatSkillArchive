# Supply Chain Security

## Core topics

1. Dependency risk
   - Third-party software
   - Vulnerability exposure

2. Integrity
   - Signing
   - Provenance

3. Practice
   - CI controls
   - Patching strategy

## Teaching approach
- Review one dependency graph for risk.
- Design one SBOM policy.
- Evaluate one third-party integration.
