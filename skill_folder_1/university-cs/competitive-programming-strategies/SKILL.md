# Competitive Programming Strategies

## Core topics

1. Patterns
   - Problem classification
   - Template usage

2. Implementation
   - Fast coding
   - Edge case handling

3. Strategy
   - Problem selection
   - Time management

## Teaching approach
- Solve one contest problem from scratch.
- Review one solution for correctness.
- Plan one timed strategy.
