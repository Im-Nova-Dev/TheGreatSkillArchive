# Terminal Productivity

## Core topics

1. Shell configuration
   - Dotfiles and rc files
   - Aliases and history

2. Multiplexing and sessions
   - tmux basics
   - Session management

3. Workflow
   - Prompt and tooling
   - Keyboard habits

## Teaching approach
- Refactor one shell session with aliases.
- Set up one tmux workflow.
- Review one dotfile for ergonomics.
