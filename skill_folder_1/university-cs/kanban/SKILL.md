# Kanban

## Core topics

1. Core model
   - Workflow stages
   - WIP limits

2. Flow
   - Cycle time
   - Throughput
   - Blockers

3. Policies
   - Definition of ready
   - Pull system

## Teaching approach
- Map one delivery process.
- Propose one WIP limit.
- Review one board and suggest one improvement.
