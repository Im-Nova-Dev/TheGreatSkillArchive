# Resilience and Site Reliability

## Core topics

1. Reliability model
   - SLI/SLO/SLA difference
   - Error budgets

2. Incident response
   - Triage and communication
   - Postmortem quality

3. Hardening
   - Retries and backoff
   - Circuit breakers

## Teaching approach
- Define SLOs for one small service.
- Write one realistic postmortem.
- Review one incident timeline.
