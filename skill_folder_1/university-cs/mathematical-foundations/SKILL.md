# Mathematical Foundations

## Core topics

1. Logic and proof
   - Induction and deduction
   - Proof techniques

2. Discrete structures
   - Sets and relations
   - Functions and counting

3. Probability and numbers
   - Basic probability
   - Integers and modular concept

## Teaching approach
- Write one simple proof by induction.
- Model one counting problem.
- Explain one discrete structure in a computing context.
