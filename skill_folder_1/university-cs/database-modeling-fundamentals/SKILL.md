# Database Modeling Fundamentals

## Core topics

1. Concepts
   - Entities and relationships
   - ER notation

2. Normalization
   - 1NF through BCNF
   - Anomalies

3. Schema design
   - Keys and constraints
   - Integrity rules

## Teaching approach
- Model one domain with an ER diagram.
- Normalize one sample schema.
- Critique one schema choice for integrity.
