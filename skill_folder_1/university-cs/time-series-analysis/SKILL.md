# Time Series Analysis

## Core topics

1. Concepts
   - Trend and seasonality
   - Stationarity

2. Models
   - ARMA/ARIMA intuition
   - Exponential smoothing

3. Evaluation
   - Forecast metrics
   - Resampling strategies

## Teaching approach
- Plot one series and identify components.
- Explain one stationarity requirement.
- Evaluate one forecast using error metrics.
