# Frontend Web Development

## Core topics

1. Layout and style
   - CSS box model
   - Responsive design

2. Interactivity
   - DOM and events
   - Component lifecycles

3. Engineering
   - Accessibility
   - Tooling

## Teaching approach
- Build one small UI component.
- Explain one layout concept.
- Review one interface for accessibility.
