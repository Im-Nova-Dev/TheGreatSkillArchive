# Domain Modeling

## Core topics

1. Building blocks
   - Entities and value objects
   - Aggregates

2. Contexts
   - Ubiquitous language
   - Bounded contexts

3. Mapping
   - To data models

## Teaching approach
- Model one familiar domain.
- Map one behavior to aggregate boundary.
- Rewrite one description in ubiquitous language.
