# Intro to Java

## Core topics

1. Language basics
   - Classes and objects
   - Access modifiers

2. OOP
   - Inheritance and interfaces
   - Composition over inheritance

3. Collections and generics
   - Lists, maps, sets
   - Type parameterization

4. Tooling
   - Build tools concept
   - Exception handling

## Teaching approach
- Map one Java concept to an equivalent in another language.
- Use one evolving project across exercises.
- Focus on naming and exception contracts.
