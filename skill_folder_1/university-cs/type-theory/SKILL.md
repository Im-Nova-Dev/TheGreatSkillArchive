# Type Theory

## Core topics

1. Type systems
   - Static and dynamic
   - Strong and weak typing

2. Polymorphism
   - Parametric and subtype
   - Type inference

3. Advanced types
   - Dependent types concept
   - Gradual typing

## Teaching approach
- Annotate one program with types.
- Explain one type error.
- Compare one static versus dynamic choice.
