# Intro to C++

## Core topics

1. Language model
   - Translation units and headers
   - ODR and linkage

2. Core concepts
   - Classes and constructors
   - Resource management

3. Standard library
   - Containers and algorithms
   - Smart pointers

4. Modern patterns
   - Move semantics
   - Lambdas and ranges

## Teaching approach
- Compare small program in C and C++.
- Show one resource leak and fix with RAII.
- Use STL before writing custom containers.
