# Strategic Planning

## Core topics

1. Direction
   - Vision
   - Goals

2. Roadmap
   - Initiatives
   - Sequencing

3. Execution
   - Allocations
   - Reviews

## Teaching approach
- Write one vision statement.
- Decompose one goal into objectives.
- Build one roadmap outline.
