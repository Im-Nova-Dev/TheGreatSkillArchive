# Scala Fundamentals

## Core topics

1. Language
   - Classes and objects
   - Case classes and pattern matching

2. Functional
   - Immutability
   - Higher-order functions

3. Advanced
   - Implicits and type classes
   - Async patterns

## Teaching approach
- Model one domain with case classes.
- Write one functional transformation.
- Explain one implicit usage.
