# Markdown and Technical Documentation

## Core topics

1. Markdown essentials
   - Headings, lists, emphasis
   - Code and fenced blocks
   - Tables and task lists

2. Extended markup
   - Admonitions and details
   - Diagrams and Mermaid
   - Frontmatter and metadata

3. Publishing
   - README conventions
   - Documentation generators
   - Versioning and changelog

4. Writing quality
   - Audience awareness
   - Example first style
   - Scannable structure

## Teaching approach
- Rewrite one README with progressive disclosure.
- Add one diagram to explain structure.
- Convert one email explainer to docs page.
