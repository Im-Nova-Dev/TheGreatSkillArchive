# Emacs

## Core topics

1. Editing model
   - Buffers and windows
   - Keybinding concept

2. Modes
   - Major and minor modes
   - Org and programming modes

3. Customization
   - Init file and Elisp basics
   - Package management

## Teaching approach
- Complete one editing task using Emacs shortcuts.
- Customize one setting in init file.
- Outline one workflow in Org mode.
