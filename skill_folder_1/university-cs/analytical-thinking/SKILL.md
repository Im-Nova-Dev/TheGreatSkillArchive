# Analytical Thinking

## Core topics

1. Framing
   - Problem decomposition
   - Assumptions and constraints

2. Reasoning
   - Evidence and uncertainty
   - Root cause analysis

3. Communication
   - Structured conclusions
   - Recommendations

## Teaching approach
- Break one messy problem into structured parts.
- Identify one hidden assumption.
- Present one recommendation clearly.
