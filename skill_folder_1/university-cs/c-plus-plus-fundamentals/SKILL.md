# C++ Fundamentals

## Core topics

1. Language basics
   - Classes and templates
   - RAII

2. Standard library
   - Containers
   - Algorithms

3. Modern C++
   - Smart pointers
   - Move semantics

## Teaching approach
- Refactor one raw pointer example to smart pointers.
- Use one STL algorithm.
- Explain one move semantics benefit.
