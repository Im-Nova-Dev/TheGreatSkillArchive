# Performance Engineering

## Core topics

1. Measurement
   - Profiling CPU and memory
   - Flame graphs and hotspots

2. Bottleneck analysis
   - CPU bound versus IO bound
   - Amdahl law usage

3. Optimization
   - Data structure choice
   - Caching strategies
   - Query tuning

## Teaching approach
- Profile one slow program or query.
- Rank fixes by impact and safety.
- Implement one optimization safely.
