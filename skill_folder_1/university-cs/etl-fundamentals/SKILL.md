# ETL Fundamentals

## Core topics

1. Extract and load
   - Source and target selection
   - Bulk and incremental

2. Transform
   - Cleansing and mapping
   - Business rules

3. Operations
   - Error handling
   - Monitoring and recovery

## Teaching approach
- Design one ETL job from one dataset.
- Choose one incremental load strategy.
- Plan one validation approach.
