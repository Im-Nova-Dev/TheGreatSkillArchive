# Programming Languages

## Core topics

1. Type systems
   - Static versus dynamic
   - Soundness and inference

2. Semantics
   - Operational semantics
   - Environment model

3. Runtime
   - Garbage collection
   - Stack versus heap

## Teaching approach
- Trace one small expression through environment model.
- Compare evaluation order in two languages.
