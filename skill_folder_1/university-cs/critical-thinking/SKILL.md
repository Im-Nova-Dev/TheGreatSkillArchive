# Critical Thinking

## Core topics

1. Reasoning
   - Arguments and evidence
   - Assumptions

2. Evaluation
   - Fallacies
   - Cognitive biases

3. Practice
   - Uncertainty
   - Structured critique

## Teaching approach
- Analyze one technical argument.
- Identify one hidden assumption.
- Evaluate one evidence claim.
