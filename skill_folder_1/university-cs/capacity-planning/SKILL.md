# Capacity Planning

## Core topics

1. Baselines
   - Current utilization
   - Utilization trends

2. Forecasting
   - Growth models
   - Seasonality and events

3. Decisions
   - Scale timing
   - Cost versus headroom

## Teaching approach
- Create one capacity summary for one service.
- Forecast one growth scenario.
- Propose one scale trigger.
