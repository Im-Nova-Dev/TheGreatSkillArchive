# Network Programming

## Core topics

1. Sockets
   - TCP and UDP
   - Client/server patterns

2. IO models
   - Blocking and nonblocking
   - Multiplexing

3. Practice
   - Protocol handling
   - Error handling

## Teaching approach
- Write one simple TCP server.
- Compare blocking and nonblocking handling.
- Review one connection error path.
