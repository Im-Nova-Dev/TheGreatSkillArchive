# Product Design for Engineers

## Core topics

1. Concept
   - Problem framing
   - User stories

2. Prototype
   - Lo-fi and hi-fi
   - Feedback collection

3. Delivery
   - MVP boundaries
   - Success metrics

## Teaching approach
- Pitch one product concept clearly.
- Sketch one MVP feature set.
- Define measurable success criteria.
