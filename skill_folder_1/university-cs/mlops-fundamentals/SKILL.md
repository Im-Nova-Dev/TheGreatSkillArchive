# MLOps Fundamentals

## Core topics

1. Pipeline
   - Data and training
   - Evaluation and validation

2. Operations
   - Model serving
   - Monitoring and alerts

3. Governance
   - Versioning
   - Compliance

## Teaching approach
- Design one training pipeline.
- Define one monitoring signal.
- Plan one rollback for model update.
