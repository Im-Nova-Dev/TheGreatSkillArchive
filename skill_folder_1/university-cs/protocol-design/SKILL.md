# Protocol Design

## Core topics

1. Structure
   - Layers and state
   - Message formats

2. Failure modes
   - Retry and timeout
   - Crash safety

3. Security
   - Authentication and replay protection
   - Idempotency

## Teaching approach
- Design one small protocol.
- Identify one failure and recovery path.
- Draw one state machine diagram.
