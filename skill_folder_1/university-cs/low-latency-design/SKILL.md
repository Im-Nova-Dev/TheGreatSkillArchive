# Low-Latency Design

## Core topics

1. Sources
   - Network and disk
   - Contention

2. Measurement
   - Percentiles and tails
   - Flame graphs

3. Mitigation
   - Caching and precomputation
   - Prioritization

## Teaching approach
- Measure one endpoint latency distribution.
- Propose one latency budget allocation.
- Rewrite one slow path reducing one waste.
