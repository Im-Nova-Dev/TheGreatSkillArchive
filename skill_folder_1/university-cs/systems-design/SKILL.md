# Systems Design

## Core topics

1. Requirements
   - Functional and nonfunctional
   - Constraints and scale

2. Caching and scaling
   - CDN and caches
   - Read versus write scaling

3. Reliability
   - Replication and redundancy
   - Consistency models

## Teaching approach
- Design one small service end to end.
- Identify one bottleneck and mitigation.
- Draw one architecture and explain tradeoffs.
