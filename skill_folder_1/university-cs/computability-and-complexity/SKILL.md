# Computability and Complexity

## Core topics

1. Computability
   - Turing machines
   - Decidability

2. Complexity classes
   - P, NP, NP-complete
   - Reductions

3. Practice
   - Recognizing hard problems
   - Using approximations

## Teaching approach
- Show one undecidable problem.
- Explain one reduction concept.
- Classify problems by complexity intuition.
