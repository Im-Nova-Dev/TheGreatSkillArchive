# Abstract Algebra Basics

## Core topics

1. Groups
   - Closure and inverses
   - Examples and properties

2. Rings and fields
   - Arithmetic modulo
   - Prime fields

3. Applications
   - Modular arithmetic in cryptography
   - Error correction

## Teaching approach
- Work with small numeric groups first.
- Show one cryptographic example.
