# Visual Thinking

## Core topics

1. Drawing
   - Simple shapes
   - Layout and emphasis

2. Diagrams
   - Flowcharts and sequences
   - State and system views

3. Communication
   - Explaining with drawing
   - Remote whiteboard usage

## Teaching approach
- Draw one system as one-pager diagram.
- Convert one explanation to picture.
- Review one diagram for clarity.
