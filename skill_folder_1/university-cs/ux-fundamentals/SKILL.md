# UX Fundamentals

## Core topics

1. Research
   - User interviews
   - Personas and scenarios

2. Design
   - Information architecture
   - Wireframes and prototypes

3. Evaluation
   - Usability testing
   - Accessibility reviews

## Teaching approach
- Interview one user about one workflow.
- Sketch one improved interface.
- Critique one current design against heuristics.
