# Edge Computing Fundamentals

1. Models
   - Edge and fog
   - Workload placement

2. Constraints
   - Latency
   - Compute and power

3. Operations

## Teaching approach
- Place one workload at edge or cloud.
- Evaluate one latency requirement.
- Review one data locality decision.
