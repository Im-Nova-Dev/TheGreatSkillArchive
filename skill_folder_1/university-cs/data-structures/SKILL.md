# Data Structures

## Core topics

1. Linear structures
   - Arrays and linked lists
   - Stacks and queues

2. Associative structures
   - Hash tables
   - Trees and heaps

3. Advanced structures
   - Tries and graphs
   - Union-find concept

## Teaching approach
- Compare two structures for one task.
- Implement one structure from scratch.
- Analyze one operation complexity.
