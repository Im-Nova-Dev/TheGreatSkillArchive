# Observability Fundamentals

## Core topics

1. Signals
   - Metrics and histograms
   - Logs and correlation IDs
   - Traces and spans

2. Instrumentation
   - Emitting key events
   - Dimensional labeling

3. Alerting
   - SLO-based alerts
   - Noise and fatigue reduction

4. Dashboards
   - Overview and drill-down
   - Golden signals

## Teaching approach
- Instrument one small service.
- Build one dashboard and one alert.
- Use one incident to practice tracing.
