# Programming Language Concepts

## Core topics

1. Syntax and semantics
   - Parsing and ASTs
   - Static vs dynamic semantics

2. Type systems
   - Static and dynamic typing
   - Type inference
   - Parametric and subtype polymorphism

3. Paradigms
   - Imperative and procedural
   - Object-oriented principles
   - Functional concepts

4. Runtime
   - Stack and heap
   - Garbage collection

## Teaching approach
- Compare same small program in three languages.
- Highlight where semantics differ.
- Emphasize tradeoffs over language wars.
