# ML Interpretability

## Core topics

1. Methods
   - Feature importance
   - SHAP and LIME

2. Explanation types
   - Global versus local
   - Counterfactuals

3. Practice
   - Debugging
   - Fairness review

## Teaching approach
- Explain one prediction for a human.
- Compare one global and one local method.
- Review one model for hidden bias.
