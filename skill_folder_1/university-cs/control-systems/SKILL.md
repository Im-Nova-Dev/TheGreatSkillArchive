# Control Systems

## Core topics

1. Basics
   - Open and closed loop
   - Feedback role

2. Controllers
   - PID concept
   - Tuning tradeoffs

3. Analysis
   - Stability idea
   - Transfer function

## Teaching approach
- Compare open and closed loop behavior.
- Diagram one feedback system.
- Simulate one PID response concept.
