# Geographic Information Systems

## Core topics

1. Foundations
   - Coordinate systems and projections
   - Spatial data models

2. Data
   - Raster and vector
   - Attributes and topology

3. Analysis
   - Spatial joins and queries
   - Buffers and interpolation

## Teaching approach
- Map one dataset to one coordinate system.
- Design one simple spatial query.
- Review one GIS layer and coordinate choice.
