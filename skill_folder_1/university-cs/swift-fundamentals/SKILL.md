# Swift Fundamentals

## Core topics

1. Language basics
   - Optionals
   - Optionals and control flow

2. Types and structures
   - Structs and classes
   - Protocols and generics

3. Modern Swift
   - Async/await
   - Memory management

## Teaching approach
- Explain one optional unwrapping decision.
- Model one domain with protocols.
- Compare value versus reference types.
