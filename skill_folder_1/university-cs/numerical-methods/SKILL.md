# Numerical Methods

## Core topics

1. Floating point
   - Representation and rounding
   - Precision loss

2. Root finding
   - Bisection and Newton
   - Convergence

3. Linear algebra
   - Direct and iterative solvers
   - Conditioning

4. Integration and ODEs
   - Quadrature rules
   - Euler and Runge-Kutta concept

5. Optimization
   - Gradient descent concept

## Teaching approach
- Compare methods on simple functions.
- Show when a method fails.
- Emphasize error and stability.
