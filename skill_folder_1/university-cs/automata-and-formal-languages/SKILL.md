# Automata and Formal Languages

## Core topics

1. Regular languages
   - DFA and NFA
   - Regular expressions

2. Context-free languages
   - PDA and grammars

3. Turing machines
   - Computation model
   - Decidability

## Teaching approach
- Convert one regex to automaton.
- Show one language needing stack.
- Explain one undecidability example.
