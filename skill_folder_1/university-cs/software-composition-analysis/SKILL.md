# Software Composition Analysis

## Core topics

1. Inventory
   - Dependency trees
   - SBOM

2. Risk
   - Vulnerability scanning
   - License compatibility

3. Response
   - Patching cadence
   - Exception handling

## Teaching approach
- Review one dependency tree.
- Generate and review one SBOM.
- Plan one safe dependency upgrade.
