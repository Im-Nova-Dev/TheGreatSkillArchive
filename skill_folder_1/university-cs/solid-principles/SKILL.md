# SOLID Principles

## Core topics

1. SRP and OCP
   - One reason to change
   - Extension over modification

2. LSP and ISP
   - Substitutability
   - Client-specific interfaces

3. DIP
   - Depend on abstractions
   - Inversion examples

## Teaching approach
- Identify one SOLID violation.
- Refactor one class toward SOLID.
- Discuss when strict SOLID hurts.
