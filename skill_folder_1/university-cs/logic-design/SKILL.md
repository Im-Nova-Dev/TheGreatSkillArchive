# Logic Design

## Core topics

1. Combinational
   - Boolean algebra
   - Minimization

2. Sequential
   - Flip-flops
   - Registers

3. State machines
   - FSM
   - Timing

## Teaching approach
- Design one combinational circuit.
- Simplify one logic expression.
- Draw one FSM state diagram.
