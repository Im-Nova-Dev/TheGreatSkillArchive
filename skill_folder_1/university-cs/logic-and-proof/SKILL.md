# Logic and Proof

## Core topics

1. Propositional logic
   - Connectives and truth tables
   - Logical equivalence

2. Predicate logic
   - Quantifiers
   - Inference rules

3. Proof techniques
   - Direct and contrapositive
   - Contradiction
   - Induction strong and weak

## Teaching approach
- Translate problem statements into logic.
- Write one proof per technique.
- Identify gaps in informal arguments.
