# Editor Workflow

## Core topics

1. Navigation
   - File and symbol jump
   - Search across project

2. Editing efficiency
   - Snippets and multi-cursor
   - Refactoring commands

3. Environment
   - LSP integration
   - Terminal and debugger

## Teaching approach
- Audit one workflow for wasted motion.
- Add one faster command replacing three slower ones.
- Measure time spent switching contexts.
