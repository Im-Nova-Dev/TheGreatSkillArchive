# Cloud Computing Fundamentals

## Core topics

1. Service models
   - IaaS, PaaS, SaaS
   - Service boundaries

2. Operations
   - Elasticity
   - Regions and zones

3. Cost and migration
   - Pricing considerations
   - Cloud adoption strategy

## Teaching approach
- Evaluate one workload for cloud fit.
- Estimate one cloud cost model.
- Design one cloud migration plan.
