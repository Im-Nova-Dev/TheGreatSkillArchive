# Compiler Optimizations

## Core topics

1. Basic optimizations
   - Constant folding and propagation
   - Dead code elimination

2. Loop and function
   - Unrolling and tiling
   - Inlining

3. Advanced
   - Vectorization
   - Interprocedural optimization

## Teaching approach
- Enable one optimization and inspect assembly.
- Explain one optimization benefit.
- Compare optimization versus readability.
