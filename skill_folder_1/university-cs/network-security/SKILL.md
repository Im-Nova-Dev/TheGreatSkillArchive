# Network Security

## Core topics

1. Perimeter and access
   - Firewalls
   - VPNs and segmentation

2. Encryption and PKI
   - TLS
   - Certificates

3. Detection
   - IDS/IPS
   - Zero-trust concepts

## Teaching approach
- Design one network zone model.
- Review one TLS configuration.
- Evaluate one detection rule.
