# Hardware-Software Co-Design

## Core topics

1. Hardware choices
   - CPU, GPU, FPGA, ASIC contexts
   - Power and performance tradeoffs

2. Software mapping
   - Parallelism and locality
   - Kernel versus user code

3. Design flow
   - Profiling first
   - Hotspot refinement

## Teaching approach
- Profile one workload and map hotspots.
- Propose one hardware-friendly refactor.
- Compare CPU versus GPU execution style.
