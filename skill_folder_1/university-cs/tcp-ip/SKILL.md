# TCP/IP

## Core topics

1. IP
   - Addressing and subnetting
   - Routing concept

2. Transport
   - TCP versus UDP
   - Connection lifecycle

3. Diagnostics
   - ICMP and ARP
   - Tools and interpretation

## Teaching approach
- Calculate one subnet range.
- Trace one TCP handshake.
- Explain one congestion control reaction.
