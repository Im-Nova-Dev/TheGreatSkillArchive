# Instruction Sets

## Core topics

1. Encodings
   - Fixed and variable length
   - Opcode layouts

2. Operands
   - Addressing modes
   - Register versus immediate usage

3. Design
   - Call conventions
   - ISA extensions

## Teaching approach
- Decode one example instruction stream.
- Compare two ISAs.
- Design one instruction format for one task.
