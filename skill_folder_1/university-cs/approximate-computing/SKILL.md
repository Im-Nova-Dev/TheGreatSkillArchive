# Approximate Computing

## Core topics

1. Accuracy
   - Error control
   - Quality metrics

2. Methods
   - Approximate arithmetic
   - Probabilistic structures

3. Application
   - Energy efficiency
   - Domain fit

## Teaching approach
- Evaluate one approximable workload.
- Compare one exact versus approximate approach.
- Design one error budget.
