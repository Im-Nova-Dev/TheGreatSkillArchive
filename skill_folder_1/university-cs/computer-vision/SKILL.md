# Computer Vision

## Core topics

1. Image processing
   - Filtering and transforms
   - Feature detection

2. Recognition
   - Object detection
   - Classification

3. Deep learning
   - CNNs for vision
   - Segmentation

## Teaching approach
- Implement one edge detection algorithm.
- Train one simple image classifier.
- Explain one CNN architecture choice.
