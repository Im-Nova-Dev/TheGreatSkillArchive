# Go Fundamentals

## Core topics

1. Language basics
   - Syntax and types
   - Methods and interfaces

2. Concurrency
   - Goroutines and channels
   - Select and patterns

3. Practice
   - Error handling
   - Module design

## Teaching approach
- Build one small Go service.
- Convert one threaded pattern to goroutines.
- Review one Go module for clarity.
