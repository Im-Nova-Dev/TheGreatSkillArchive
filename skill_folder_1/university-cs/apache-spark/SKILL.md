# Apache Spark

## Core topics

1. Architecture
   - Driver and executors
   - RDD concept

2. APIs
   - DataFrames and Datasets
   - Transformations

3. Performance
   - Shuffling and partitioning
   - Memory and serialization

## Teaching approach
- Design one Spark job.
- Explain one narrow versus wide dependency.
- Tune one performance bottleneck.
