# Refactoring Fundamentals

## Core topics

1. Smells
   - Long method, large class, duplicated logic
   - Feature envy and inappropriate intimacy

2. Safety
   - Tests and executable expectations
   - Incremental transformations

3. Common refactorings
   - Extract method, rename, move
   - Replace conditional with polymorphism

## Teaching approach
- Detect one smell in a small code sample.
- Convert one smell into one concrete refactoring step.
- Review one PR focusing on improved structure.
