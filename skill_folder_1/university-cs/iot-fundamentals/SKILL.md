# IoT Fundamentals

## Core topics

1. Device architecture
   - MCU and radio
   - Sensors and actuators

2. Connectivity
   - Protocols overview
   - Power considerations

3. Systems
   - Edge processing
   - Cloud integration

## Teaching approach
- Model one IoT product architecture.
- Compare one protocol choice for one use case.
- Evaluate one power budget.
