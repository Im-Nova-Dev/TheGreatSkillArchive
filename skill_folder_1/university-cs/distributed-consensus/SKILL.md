# Distributed Consensus

## Core topics

1. Fundamentals
   - Consensus problem
   - Safety and liveness

2. Algorithms
   - Paxos concept
   - Raft overview

3. Practice
   - Failure handling
   - Implementation tradeoffs

## Teaching approach
- Compare Paxos and Raft.
- Explain one safety requirement.
- Pick one consensus algorithm for one system.
