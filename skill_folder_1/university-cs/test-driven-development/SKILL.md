# Test Driven Development

## Core topics

1. TDD cycle
   - Red-green-refactor
   - Failure first mindset

2. Test types
   - Unit versus integration
   - Mocking

3. Quality
   - Coverage and design feedback
   - Maintainable tests

## Teaching approach
- Apply red-green-refactor to one feature.
- Refactor one test suite.
- Explain when TDD adds value versus overhead.
