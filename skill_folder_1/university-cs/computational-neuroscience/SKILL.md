# Computational Neuroscience

## Core topics

1. Models

   - Network models

2. Coding
   - Neural representations
   - Population codes

3. Dynamics

## Teaching approach
- Simulate one simple neuron model.
- Explain one neural coding concept.
- Evaluate one network dynamics result.
