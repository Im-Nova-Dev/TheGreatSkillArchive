# Memory Hierarchy

## Core topics

1. Levels
   - Registers and cache
   - RAM and storage

2. Locality
   - Temporal and spatial
   - Access patterns

3. Caches
   - Lines and mapping
   - Replacement policies

## Teaching approach
- Analyze one memory access pattern.
- Estimate one cache miss rate.
- Optimize one data layout for locality.
