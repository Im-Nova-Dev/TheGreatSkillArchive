# Cloud-Native Architectures

## Core topics

1. Architecture
   - Microservices idea
   - Container orchestration

2. Resilience and observability
   - Health checks and retries
   - Metrics and tracing

3. Operations
   - Scaling and cost
   - Migration approach

## Teaching approach
- Design one cloud-native system.
- Identify one resilience gap.
- Propose one monitoring plan.
