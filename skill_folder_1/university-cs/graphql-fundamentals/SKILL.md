# GraphQL Fundamentals

## Core topics

1. Schema
   - Types and relationships
   - Query shape

2. Execution
   - Resolvers
   - Data loaders

3. Practice
   - Client usage
   - Error handling

## Teaching approach
- Design one schema for one domain.
- Implement one resolver flow.
- Resolve one N+1 performance issue.
