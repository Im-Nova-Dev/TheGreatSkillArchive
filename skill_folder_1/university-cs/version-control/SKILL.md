# Version Control

## Core topics

1. Concepts
   - Snapshots and changes
   - DAG model

2. Workflows
   - Branching strategies
   - Merge and rebase

3. History
   - Blame and bisect
   - Rewriting history safely

## Teaching approach
- Inspect one repository history.
- Choose one branching strategy.
- Practice one conflict resolution.
