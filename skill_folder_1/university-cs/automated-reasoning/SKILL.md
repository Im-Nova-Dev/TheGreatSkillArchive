# Automated Reasoning

## Core topics

1. Logic
   - Propositional and predicate
   - Normal forms

2. Methods
   - Resolution
   - SAT and SMT concept

3. Applications
   - Verification idea
   - Tool usage concept

## Teaching approach
- Translate one claim to logic.
- Apply one inference rule.
- Review one verification scenario.
