# Clustering

## Core topics

1. Similarity
   - Distance measures
   - Feature scaling

2. Methods
   - Partitioning and hierarchical
   - Density methods

3. Evaluation
   - Internal and external
   - Labeling

## Teaching approach
- Cluster one small dataset.
- Compare two distance measures.
- Evaluate one clustering result.
