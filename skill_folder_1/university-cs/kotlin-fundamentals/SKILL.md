# Kotlin Fundamentals

## Core topics

1. Language
   - Null safety
   - Data classes

2. Functional style
   - Lambdas and receivers
   - Collections

3. Concurrency and interop
   - Coroutines
   - Java interop

## Teaching approach
- Convert one nullable Java API to Kotlin style.
- Refactor one class into a data class.
- Explain one coroutine builder choice.
