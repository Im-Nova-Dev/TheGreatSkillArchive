# Technical Problem Solving

## Core topics

1. Clarification
   - Problem restatement
   - Constraint identification

2. Design
   - Decomposition
   - Pattern selection

3. Analysis
   - Tradeoffs
   - Complexity

## Teaching approach
- Clarify one ambiguous problem.
- Propose one decomposition.
- Evaluate one solution tradeoff.
