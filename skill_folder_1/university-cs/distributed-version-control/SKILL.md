# Distributed Version Control

## Core topics

1. Model
   - Repository copies
   - Peer-to-peer synchronization

2. History
   - DAG model
   - Branching strategies

3. Workflow
   - Merge strategies
   - Divergence resolution

## Teaching approach
- Diagram one distributed collaboration flow.
- Compare merge strategies.
- Recover from one history divergence.
