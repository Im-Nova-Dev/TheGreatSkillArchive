# Process Synchronization

## Core topics

1. Critical section problem
   - Mutual exclusion
   - Progress and bounded waiting

2. Primitives
   - Mutex and semaphore
   - Condition variables

3. Problems
   - Deadlock avoidance and detection
   - Priority inversion

## Teaching approach
- Identify one race in example code.
- Apply one synchronization fix.
- Explain one deadlock scenario.
