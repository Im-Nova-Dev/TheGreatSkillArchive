# Application Security

## Core topics

1. Insecure inputs
   - Injection
   - Validation and encoding

2. Authentication and authorization
   - Session and access control flaws
   - Secure defaults

3. Monitoring
   - Logging
   - Incident response integration

## Teaching approach
- Audit one code path for risks.
- Fix one vulnerable example.
- Add one logging improvement.
