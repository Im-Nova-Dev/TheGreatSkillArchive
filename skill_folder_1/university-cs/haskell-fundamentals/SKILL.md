# Haskell Fundamentals

## Core topics

1. Language basics
   - Syntax and types
   - Pattern matching

2. Advanced types
   - Typeclasses
   - Polymorphism

3. Functional design
   - Monads and functors
   - Lazy evaluation

## Teaching approach
- Write one pure function in Haskell.
- Explain one typeclass.
- Compare eager versus lazy evaluation.
