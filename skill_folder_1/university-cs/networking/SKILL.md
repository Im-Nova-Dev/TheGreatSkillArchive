# Networking

## Core topics

1. Models and protocols
   - TCP/IP layers
   - Common protocols

2. Programming
   - Sockets and ports
   - Client/server examples

3. Operations
   - Packet capture
   - Path analysis

## Teaching approach
- Capture one web request in Wireshark.
- Write one simple client/server socket program.
- Map one delivery path with one failure scenario.
