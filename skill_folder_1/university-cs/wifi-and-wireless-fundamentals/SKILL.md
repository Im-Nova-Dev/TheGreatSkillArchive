# Wi-Fi and Wireless Fundamentals

## Core topics

1. Radio basics
   - Spectrum and channels
   - Interference sources

2. Standards
   - 802.11 families
   - Band and width choices

3. Security
   - WPA3 and WPA2
   - Open networks

## Teaching approach
- Review one site survey result.
- Map channel overlap in a given area.
- Recommend one placement and channel plan.
