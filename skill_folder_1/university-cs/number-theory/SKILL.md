# Number Theory

## Core topics

1. Divisibility
   - Primes and factorization
   - GCD and LCM

2. Modular arithmetic
   - Congruences
   - Inverses

3. Applications
   - Primality tests
   - Cryptography connections

## Teaching approach
- Compute one modular inverse.
- Explain one primality test idea.
- Connect one theorem to a practical use case.
