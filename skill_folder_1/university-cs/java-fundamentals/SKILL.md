# Java Fundamentals

## Core topics

1. Language and types
   - Classes and interfaces
   - Generics

2. APIs
   - Collections
   - Streams

3. Runtime
   - Exception handling
   - JVM overview

## Teaching approach
- Model one domain with classes.
- Stream-transform one collection.
- Explain one checked versus unchecked exception choice.
