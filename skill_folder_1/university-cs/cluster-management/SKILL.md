# Cluster Management

## Core topics

1. Scheduling
   - Bin packing
   - Affinity and taints

2. Lifecycle
   - Upgrades and rollbacks
   - Draining nodes

3. Observability
   - Node health and utilization
   - Workload quotas

## Teaching approach
- Review one cluster manifest.
- Propose one upgrade plan.
- Simulate one node failure response.
