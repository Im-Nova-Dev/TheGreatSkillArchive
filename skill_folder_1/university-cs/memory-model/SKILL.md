# Memory Model

## Core topics

1. Consistency
   - Sequential consistency
   - Relaxed ordering

2. Synchronization
   - Acquire and release
   - Barriers

3. Correctness
   - Happens-before

## Teaching approach
- Explain one reordering example.
- Use one ordering primitive correctly.
- Diagnose one memory ordering bug.
