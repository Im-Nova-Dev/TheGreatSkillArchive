# Game Theory

## Core topics

1. Games
   - Players and strategies
   - Payoffs and equilibrium

2. Analysis
   - Nash equilibrium
   - Dominant strategies concept

3. Applications
   - Cooperation and mechanism design
   - Interaction design

## Teaching approach
- Analyze one two-player game.
- Find one equilibrium.
- Explain one cooperation versus defection outcome.
