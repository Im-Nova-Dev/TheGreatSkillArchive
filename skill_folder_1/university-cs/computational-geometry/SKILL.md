# Computational Geometry

## Core topics

1. Primitives
   - Points, lines, polygons
   - Orientation and predicates

2. Structures
   - Convex hull
   - Triangulation concept

3. Applications
   - Mesh processing
   - Geographic queries

## Teaching approach
- Implement one geometric predicate.
- Explain one orientation test.
- Apply one geometry primitive in a small task.
