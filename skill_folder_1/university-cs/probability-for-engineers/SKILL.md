# Probability for Engineers

## Core topics

1. Random variables
   - Discrete and continuous
   - Distributions

2. Joint and conditional
   - Independence
   - Bayes theorem

3. Models
   - Markov chains
   - Reliability models

## Teaching approach
- Model one system failure probability.
- Explain one Bayes theorem use case.
- Compare one distribution to observed data.
