# Testing Strategies

## Core topics

1. Levels
   - Unit, integration, E2E
   - Scope and speed

2. Quality
   - Flaky tests and repair
   - Data and environment

3. Maintenance
   - Suite design
   - Selection and removal

## Teaching approach
- Classify one test by purpose and scope.
- Replace one flaky test with stable design.
- Review one suite for coverage misuse.
