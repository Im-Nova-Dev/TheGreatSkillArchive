# Security Auditing

## Core topics

1. Planning
   - Scope and criteria
   - Audit objectives

2. Review
   - Access and configuration
   - Policy enforcement

3. Reporting
   - Evidence and findings
   - Remediation

## Teaching approach
- Plan one security audit.
- Review one access list.
- Draft one audit finding.
