# Device Drivers

## Core topics

1. Kernel interface
   - Module model
   - Device registration

2. Hardware interaction
   - Interrupts and DMA
   - Memory access

3. Practice
   - Debugging
   - Power management

## Teaching approach
- Trace one interrupt path.
- Review one simple driver skeleton.
- Explain one device tree concept.
