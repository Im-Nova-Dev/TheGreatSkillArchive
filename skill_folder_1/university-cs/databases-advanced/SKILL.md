# Advanced Database Topics

## Core topics

1. Query optimization
   - Execution plans
   - Index selection and hints

2. Transactions
   - Isolation anomalies
   - Locking modes

3. Replication and sharding
   - Consistency choices
   - Partitioning strategies

## Teaching approach
- Inspect one query plan and rewrite.
- Compare isolation level tradeoffs.
- Design one sharding key for sample dataset.
