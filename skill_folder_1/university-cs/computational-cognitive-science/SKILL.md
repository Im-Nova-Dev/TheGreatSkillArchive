# Computational Cognitive Science

## Core topics

1. Modeling
   - Cognitive architecture
   - Representation and computation

2. Learning
   - Concept learning
   - Categorization models

3. Application
   - Decision making
   - Language processing

## Teaching approach
- Model one cognitive task computationally.
- Compare two learning models.
- Evaluate one reasoning approach.
