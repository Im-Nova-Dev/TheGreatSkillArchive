# Clojure Fundamentals

## Core topics

1. Data and functions
   - Immutable collections
   - Sequence operations

2. Language
   - Syntax and destructuring
   - Macros concept

3. Concurrency
   - State management
   - Reactive patterns

## Teaching approach
- Transform one collection functionally.
- Explain one persistent data structure.
- Use the REPL to explore one problem.
