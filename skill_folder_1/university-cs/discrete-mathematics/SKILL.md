# Discrete Mathematics

## Core topics

1. Logic and proofs
   - Propositional and predicate logic
   - Direct, contradiction, induction

2. Combinatorics
   - Counting and pigeonhole
   - Inclusion-exclusion

3. Graphs and trees
   - Paths and cycles
   - Representing data

4. Recurrences
   - Linear recurrences
   - Master theorem concept

## Teaching approach
- Convert one proof from informal to formal.
- Model one combinatorics problem.
- Solve one recurrence together.
