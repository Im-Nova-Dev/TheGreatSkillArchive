# Git and GitHub

## Core topics

1. Version control basics
   - Repository lifecycle
   - Commits and history

2. Branching and merging
   - Branch workflow
   - Conflict resolution

3. Collaboration
   - Remotes and PRs
   - GitHub features

## Teaching approach
- Explain Git through one real workflow.
- Practice resolving one merge conflict.
- Demonstrate one GitHub review flow.
