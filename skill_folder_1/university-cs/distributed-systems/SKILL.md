# Distributed Systems

## Core topics

1. Concepts
   - Time and ordering
   - Consistency and availability

2. Consensus
   - Leader election intuition
   - Failure recovery

3. Messaging
   - Queues and events
   - Ordering guarantees

## Teaching approach
- Model one multi-node failure.
- Define one desired invariant.
- Compare one strong vs eventual choice.
