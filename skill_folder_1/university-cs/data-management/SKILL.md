# Data Management

## Core topics

1. Governance
   - Ownership and policy
   - Data lifecycle

2. Quality
   - Validation and cleansing
   - Monitoring

3. Architecture
   - Domains and standards
   - Metadata usage

## Teaching approach
- Review one dataset for quality issues.
- Define one data quality rule.
- Document one lineage step.
