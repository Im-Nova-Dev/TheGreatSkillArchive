# Security and Cryptography

## Core topics

1. Threat model
   - Assets and adversaries
   - Attack surfaces

2. Controls
   - Authentication and authorization
   - Encryption and integrity

3. Application security
   - Input validation
   - Injection and escaping

## Teaching approach
- Design one secure login flow.
- Review one code path for risk.
- Explain one common attack story.
