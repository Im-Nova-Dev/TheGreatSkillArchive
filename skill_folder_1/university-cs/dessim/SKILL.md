# Discrete Event Simulation

## Core topics

1. Model elements
   - Entities and resources
   - Event lists and queues

2. Execution
   - Clock and events
   - Randomness and replication

3. Analysis
   - Confidence intervals
   - Verification and validation

## Teaching approach
- Design one simple queue model.
- Run one simulation conceptually.
- Compare analytical and simulated results.
