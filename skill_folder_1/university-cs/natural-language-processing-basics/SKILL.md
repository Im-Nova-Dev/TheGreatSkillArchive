# Natural Language Processing Basics

## Core topics

1. Text processing
   - Tokenization
   - Normalization and cleaning

2. Features and models
   - Bag of words and TF-IDF
   - Word embeddings

3. Deep learning
   - RNNs and transformers
   - Attention mechanism

## Teaching approach
- Build one text classification pipeline.
- Explain one attention mechanism.
- Compare traditional versus deep learning NLP.
