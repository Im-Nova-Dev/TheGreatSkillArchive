# Time Management

## Core topics

1. Planning
   - Prioritization
   - Estimation

2. Execution
   - Focus and context
   - Interrupt handling

3. Reflection
   - Weekly review
   - System tuning

## Teaching approach
- Plan one week realistically.
- Estimate one task set.
- Review one schedule for interruptions.
