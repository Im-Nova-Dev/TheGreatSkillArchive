# Big Data Fundamentals

## Core topics

1. Storage
   - Distributed file systems concept
   - Object storage concept

2. Processing
   - Batch and stream tradeoffs
   - Partitioning and shuffling

3. Querying
   - Warehouse versus lake
   - Schema-on-read versus schema-on-write

## Teaching approach
- Design one pipeline for one dataset shape.
- Compare batch and streaming ingestion.
- Estimate job cost and runtime.
