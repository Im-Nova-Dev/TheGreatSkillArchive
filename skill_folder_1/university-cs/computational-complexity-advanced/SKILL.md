# Advanced Computational Complexity

## Core topics

1. Advanced analysis
   - Amortized analysis
   - Dynamic programming complexity

2. Hardness
   - NP-hard examples
   - Reductions

3. Algorithms
   - Approximation heuristics
   - Parameterized algorithms concept

4. Meta algorithms
   - Meet-in-the-middle concept
   - Branch and bound concept

## Teaching approach
- Analyze one data structure amortized cost.
- Show one reduction between problems.
- Explain why one problem feels harder.
