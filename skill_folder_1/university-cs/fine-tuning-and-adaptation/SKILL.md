# Fine-tuning and Adaptation

## Core topics

1. Preparation
   - Dataset cleaning and format
   - Base model choice

2. Training
   - Hyperparameters
   - Adapter methods

3. Evaluation
   - Generalization checks
   - Drift concept

## Teaching approach
- Prepare one small dataset for one task.
- Choose one technique and explain tradeoffs.
- Define one evaluation protocol.
