# Godot 3D Game Development

## Core topics

1. Scene
   - Meshes
   - Lighting

2. Physics
   - Collision
   - Rigid bodies

3. Presentation
   - Camera
   - UI

## Teaching approach
- Build one simple 3D scene.
- Add one interaction.
- Polish one visual element.
