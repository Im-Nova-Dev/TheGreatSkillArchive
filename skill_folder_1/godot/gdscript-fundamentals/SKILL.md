# GDScript Fundamentals

## Core topics

1. Basics
   - Variables and types
   - Functions

2. Control flow
   - Conditionals
   - Loops

3. Data structures
   - Arrays
   - Dictionaries

## Teaching approach
- Write one small script from scratch.
- Explain one keyword per example.
- Use one game-like context.
