# Causal Inference

## Core topics

1. Framing
   - Causal questions
   - Confounding

2. Methods
   - Design
   - Analysis

3. Practice
   - DAGs
   - Limitations

## Teaching approach
- Distinguish correlation from causation in one example.
- Draw one simple DAG.
- Propose one causal study design.
