# Statistics for Data Science

## Core topics

1. Foundations
   - Probability
   - Distributions

2. Testing
   - Hypothesis tests
   - Confidence intervals

3. Application
   - A/B testing
   - Regression intuition

## Teaching approach
- Formulate one testable question.
- Run one hypothesis check.
- Explain one p-value pitfall.
