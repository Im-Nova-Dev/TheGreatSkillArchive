# Data Ethics and Privacy

## Core topics

1. Principles
   - Fairness
   - Consent

2. Risks
   - Bias
   - Privacy

3. Practice
   - Reviews
   - Documentation

## Teaching approach
- Audit one dataset for bias.
- Review one model for fairness risk.
- Draft one data ethics statement.
