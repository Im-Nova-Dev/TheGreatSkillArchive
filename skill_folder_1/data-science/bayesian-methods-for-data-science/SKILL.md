# Bayesian Methods for Data Science

## Core topics

1. Basics
   - Priors and posteriors
   - Updating

2. Methods
   - Conjugate priors
   - MCMC intuition

3. Practice
   - Credible intervals
   - Model comparison

## Teaching approach
- Update one belief with data.
- Compare one prior to posterior.
- Design one Bayesian A/B test.
