# Big Data Tools for Data Science

## Core topics

1. Concepts
   - Distributed data
   - Partitioning

2. Tools
   - PySpark
   - Dask

3. Practice
   - Sampling
   - Performance

## Teaching approach
- Run one PySpark job on sample data.
- Compare pandas versus Dask for one task.
- Design one sampling strategy.
