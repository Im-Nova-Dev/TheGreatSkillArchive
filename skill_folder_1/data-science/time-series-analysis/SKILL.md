# Time Series Analysis

## Core topics

1. Basics
   - Trends and seasonality
   - Stationarity

2. Modeling
   - Decomposition
   - Forecasting

3. Evaluation
   - Metrics
   - Baseline selection

## Teaching approach
- Plot one time series and identify components.
- Build one baseline forecast.
- Evaluate one forecast against actuals.
