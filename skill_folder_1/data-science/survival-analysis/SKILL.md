# Survival Analysis

## Core topics

1. Foundations
   - Event and censoring
   - Survival functions

2. Methods
   - Kaplan-Meier
   - Cox model

3. Practice
   - Assumptions
   - Interpretation

## Teaching approach
- Define one event and censoring rule.
- Build one Kaplan-Meier estimate.
- Interpret one hazard ratio.
