# Airtag-Free Local Presence

## Options
1. Home Assistant app tracker: Wi-Fi/Bluetooth/GPS lle.
2. ESP32 BLE snap: advertise or sniff known phones.
3. Zigbee presence snippets via sensor-connected rooms.

## Privacy
- Store only current zone, not history.
- Avoid MAC logging; app-controlled identifiers only.

## Teaching exercise
Implement two zones: home and away using only local signals.
