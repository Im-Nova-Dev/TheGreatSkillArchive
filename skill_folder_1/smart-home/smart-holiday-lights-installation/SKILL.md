# Smart Holiday Lights Installation Mastery

## Core concept
Smart holiday lights use addressable RGB or RGBW LEDs with individual pixel control. Installation differs from dumb string lights because of power injection, controller placement, and data-direction rules.

## Product landscape (2025-2026)
  - Twinkly Strings Multicolor (app effects library, per-LED control)
  - Govee String Lights 2 / Smart Christmas String Lights (Matter, translucent cable)
  - Nanoleaf Matter Smart String Lights (300 LEDs, 60 zones)
  - Philips Hue Festavia (Hue-integrated per-LED patterns, requires Hue Bridge)

## Pre-installation
- Outdoor vs indoor: do not mix. Outdoor sets have molded IP65 ends; indoor sets can fail in rain.
- Plan the run: estimate length, then add 10% extra for looping. Each string set has fixed length with male/female end.
- Power location: keep controller within Wi-Fi range. For large trees, plan from trunk up rather than from top down.

## Power and data wiring
- Controller output: some systems have separate power and data ports; wire power to first LED in line.
- Data direction: almost all addressable strings have arrow or DI/DO (data-in/data-out). Install in the marked direction or patterns fail.
- Power injection: runs over 100 LEDs or longer than 50ft benefit from injection on V+ and GND at midpoint.

## Routing through trees/outdoor
- Avoid sharp kinks; use liberal clips rated for tree limbs.
- UV cable: outdoor cable rated UV-resistant lasts seasons vs indoor cable fading in summer.
- Do not staple through cable. Use zip ties with sheath sliders.

## Seasonal automation
- Schedule in app to turn on/off at sunset/sunrise.
- Music/camera sync: reply requires microphone and camera permissions.
- Storage: coil without clamp damage; store dry; label run.

## Safety
- Only use listed ETL/UL power supplies.
- Do not exceed max length; overvoltage from chaining beyond spec damages LEDs.
- Wet locations: keep controller above grade.

## Teaching drills
- Drill 1: apprentice identifies data direction on an unlabeled string and tests continuity on DI/DO lines.
- Drill 2: simulate run for a 12ft tree; apprentice calculates injection points and clip count.
- Drill 3: seasonal mock; apprentice sets schedule to trigger in HomeKit with voice scene.

## Post-install
- [ ] End-to-end test with all effects before climbing ladder.
- [ ] Data direction finalized by reading result, not just arrow.
- [ ] Outdoor ends after last string sealed with covers.
- [ ] App name matches room/year: “Front Porch Tree — 2025 Christmas”。
