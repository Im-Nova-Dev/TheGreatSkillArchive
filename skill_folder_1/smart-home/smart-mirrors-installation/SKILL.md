# Smart Mirror Installation Mastery

## Core concept
Mirror category divides: (1) backlit LED mirrors (dimmable light), (2) smart mirrors with anti-fog + altitude/weather sensors, and true third category: display mirrors with touchscreen. Skills below focus on 1 and 2 because display mirrors are rare in 2026 market.

## Product examples
  - Keonjinn Anti-Fog LED Bathroom Mirror
  - Kohler Verdera Smart Mirror (HomeKit)
  - Hauschen / Dimo frameless LED mirrors
  - RR line, Elitlife, and generic Amazon backlit LED mirrors

## Pre-installation checks
- Wall type: drywall over studs vs tile backerboard. Need molly bolts or masonry anchors.
- GFCI protection: outlet inside mirror cavity or nearby must be GFCI.
- Height: center of mirror 60in-72in from floor (dependent on household height); allow bottom clearance above vanity backsplash.
- Junction box location: most hardwired mirrors need mounting at stud with 1-gang box. Surface-mount kits avoid remodeling.

## Installation flow: hardwired mirror
1. Turn off breaker. Verify dead.
2. Mark stud locations; mark box center at desired height.
3. Cut opening for old-work box; run romex from GFCI-protected circuit.
4. Connect mirror leads: hot, neutral, ground. Some mirrors add separate defogger pad wire (often 12-24V low voltage).
5. Tuck wires; mount box; hang mirror on hooks or French cleat.
6. Seal edges with silicone if in shower wet zone.
7. Restore power.

## Installation flow: plug-in / corded
1. Verify outlet location behind mirror (chase cable through wall with low-voltage bushing if hidden-cord desired).
2. Plug in and route cord.
3. Mount mirror.

## Smart features (Kohler Verdera class)
- Pair via HomeKit or vendor app. Requires always-hot power.
- Anti-fog timer: usually auto-run on power-up; can be disabled via physical switch or app.
- Dimmable light: use wall switch with dimmer only if mirror states it supports trailing-edge; otherwise use app-only dimming.

## Safety
- Shatter risk: tempered glass only. If mirror arrives with cracked edge, do not install.
- Steam condensation: mirror must be rated for wet zone if inside shower.

## Teaching drills
- Drill 1: apprentice marks correct height on a 60in mirror mock and predicts clearance on a 36in vanity countertop.
- Drill 2: mock junction-box wiring; apprentice connects hot/neutral/ground plus separate low-voltage defog pad in under 10 minutes.
- Drill 3: explain GFCI test procedure and why tape-insulation is not a fix.

## Post-install
- [ ] Mirror level.
- [ ] Anti-fog pad warms within 60 seconds.
- [ ] Light edges evenly illuminated (no dark band from poorly seated diffuser).
- [ ] Smart pairing confirmed if Bluetooth/Wi-Fi module present.
