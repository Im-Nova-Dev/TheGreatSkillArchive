# Smart Pendant & Ceiling Lights Installation Mastery

## Core concept
“Pendant” and “flush mount” differ mainly by drop length. Smart ceiling lights almost always integrate LED driver + controller into the fixture body.

## Pre-installation
- Weight: most smart ceiling fixtures are 3-8 lbs. Standard octagon box is rated 50 lbs when secured to framing. Surface-mount pancake boxes only 10-15 lbs; verify before hanging heavy units.
- Box location: confirm box is secured to framing or has old-work bar hanger for retrofit.
- Height: 28-36in above counter/island for pendants; 7ft clearance above floor for hallways.
- Alignment: for multi-pendant arrays, use laser level; 12in spacing standard.

## Hardwired install flow
1. Power off breaker; verify.
2. Remove old fixture, disconnect hot/neutral/ground.
3. If old fixture is heavy: inspect box mounting; upgrade to fan-rated box if loose.
4. Feed fixture wires through new canopy; connect:
   - Line/hot to fixture hot (black/red).
   - Neutral to fixture neutral (white).
   - Ground to fixture ground wire and box.
5. Tuck wires; mount canopy to box.
6. Restore power.
7. App discovery: Hue Ceiling / Govee Ceiling show up in app when powered.
8. Name by location + function: “Kitchen Island - Pendant Center”.

## Corded pendant with plug-in base
- Plug into switched or always-on outlet depending on use.
- Cord routing: keep away from cooktop heat; use cord cover if across floor.

## Driver heat and IP
- Keep driver in ventilated space, not inside insulation.
- For enclosed ceiling boxes: IC-rated driver required.

## Troubleshooting
  Symptom                         | Cause                                    | Fix
  --------------------------------|------------------------------------------|-----------------------------------------
  Flickers after install           | Loose neutral or ground in j-box          | Reseat wire nuts
  Hue ceiling never shows in app   | Requires Hue Bridge unless Bluetooth     | Add Bridge or use app directly
  Uneven light between 2-bracket array | Misaligned heights                    | Adjust rod lengths before tightening set screws

## Teaching drills
- Drill 1: reading fixture spec sheet and matching to ceiling box rating.
- Drill 2: apprentice heights drop-length adjustment with measuring tape and level.
- Drill 3: pair, name, and group; apprentice builds scene with 3 pendant lights in room.

## Post-install
- [ ] Box not pulling away from ceiling.
- [ ] No exposed copper.
- [ ] App sees all segments individually.
- [ ] Dimming from 1-100% is flicker-free.
