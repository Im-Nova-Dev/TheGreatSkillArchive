# Smart Mirror Refresher + Teaching Script

## Beginner confusion to resolve
Users often mistake backlit LED bath mirrors for fully network-controlled displays. The typical expectation gap:
- What they think: touchscreen + weather + Spotify mirror.
- What they get: dimmable LED backlight + anti-fog pad.
- Teaching line: "This mirror is for light and defog, not for apps. Smart control is via the companion app if present."

## Height and clearance
- Ideal center height 64-72in below 84in ceiling.
- Mirror bottom above backsplash 4-6in.

## Wiring diagram
```
Line  ----> Mirror hot
Neutral --> Mirror neutral
Ground --> Mirror ground + box
Optional 12-24V -> Defog pad
```

## Teaching sequence
1. Begin with GFCI test.
2. Show mounting plate position.
3. Wire and test before full mirror mount.
4. Explain anti-fog timer usually runs on power-up.

## Common failure causes
- GFCI outlet behind mirror trips immediately → shared neutral miswire.
- Light banding → diffuser not seated fully against backplate.
- Steam on non-wetrated mirror in shower zone → returns for replacement.

## Post-install checklist
- [ ] GFCI test is green.
- [ ] Light edges evenly bright.
- [ ] Defogger warms in under 90 seconds.
