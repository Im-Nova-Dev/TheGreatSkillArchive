# Smart Vanity & Bathroom Lighting Installation Mastery

## Core concept
Bathrooms require moisture-safe *and* smart lighting. The installation priority order is: safety, then illumination quality, then smart features.

## Product types
  - Mirror-integrated LED backlight: Keonjinn, Kohler Verdera, Dimo, Hauschen, Novi Home.
  - Separate light bars/sconces: Govee mirror lights, Nanoleaf, Hue Adore collection.
  - Smart bath fan with light: Broan/Legrand fan + Hue bulb compatible.

## Rating thresholds
  - Damp-rated: IP44; acceptable inside bathroom away from shower.
  - Wet-rated: IP65; inside shower zone required by code in many jurisdictions.
  - GFCI: all bathroom outlets must be GFCI-protected; provide test button.

## Install flow: integrated LED mirror
1. Turn off GFCI breaker; verify dead with non-contact tester.
2. Mount backing at correct height (mirror center usually 64-72in).
3. Wire hot/neutral; run separate wire if GFCI outlet is outside mirror cavity.
4. Hang mirror; seal edges if in tub/shower surround.
5. Restore power; anti-fog and dimming function tested.

## Install flow: light bar added to existing mirror
1. Use adhesive or drill mounts per instructions.
2. Route hidden wire from GFCI outlet behind mirror.
3. Connect 12V LED driver only if hardwired variant.

## Smart behavior design
  - Morning: 4000K, full brightness.
  - Evening: 2700K, 60%.
  - Night low: motion-triggered 2700K at 3% through floor LED strips.
  - Auto-off after 20 min if no motion prevents bathroom all-night lights.

## Lighting mistakes to avoid
  - Light above mirror casts facial shadows. Always include front-facing perimeter light or side sconces.
  - Downlights alone = poor task lighting near vanity.

## Post-install
- [ ] GFCI tested; trips.
- [ ] Mirror anti-fog activates within 60s.
- [ ] Dimmable without flicker at 1-100%.
- [ ] Scene tested for morning and evening use.
