# Smart Lamps Installation Mastery

## Core concept
Smart lamps ship three ways: (1) base-only + buy smart bulb, (2) integrated smart bulb + proprietary driver, (3) lamp-as-platform (RGB, speaker, air quality).

## Category selection
  - Base-only: Philips Hue pedestal, LIFX Table Capsule, IKEA Tradfri lamp base. Allows bulb upgrade path.
  - Integrated: Govee Floor Lamp 3, LIFX Luna, Govee Table Lamp 2 Pro x JBL. Ships with proprietary driver and bulb.
  - Platform lamps: Govee floor lamp with speaker + air-quality sensor; adds complexity beyond lighting.

## Pre-installation cues
- Clear 6in radius around base; many bases are wider than traditional lamps for driver housing.
- Floor lamp cord: prefer routed behind furniture or under rug cord cover; do not leave across walkways (trip hazard).
- Outlet under load: some floor lamps draw 15-20W; fine on any 15A circuit, but stacked with high-wattage space heaters may trip breaker.

## Installation paths

### Base-only + smart bulb
1. Unpack base, assemble shaft (two-piece shafts twist-lock).
2. Place bulb into socket (confirm max wattage rating; usually 10W LED max).
3. If bulb is Hue: pair bulb first before placing in lamp (easier to reach link button on base).
4. If bulb is Wi-Fi: join Wi-Fi via app while bulb on desk.

### Integrated lamp (Govee / LIFX)
1. Remove transit packaging from shade/head.
2. Connect power adapter or plug cable; turn on switch on cord or base.
3. App discovery: ensure Bluetooth/Wi-Fi not congested; many units broadcast BLE for first-pair.
4. Name “Living Room - Reading Floor Lamp”; assign room.

### Lamp with speaker or extra features
1. Two-stage pairing: audio/BLE handshake followed by Wi-Fi credentials.
2. Keep phone near lamp during pairing; some sonic pairing requires <1m.
3. Verify firmware after pairing; treat features like scenes, sound-reactive modes separately.

## Safety and stability
- Center of gravity: floor lamps with heavy heads need wide base or sandbag. Keep from bedroom edges where children/pets can pull.
- Heat: upper enclosure should not exceed 50°C; if driver housing gets too hot to touch after 30 min, place on fire-safe surface.

## Teaching hooks
- Ask the user: “Would you still want ‘smart’ features if Wi-Fi dies?” Answer: base-only + Pico remote is most resilient; integrated lamp becomes dumb.
- Compare upgradeability : base-only wins over 3 years; integrated lamp needs full replacement.
- Walkway test: run a 1-hour walk test with lamp powered; if anyone trips over cord, relocate outlet.

## Post-install
- [ ] Lamp stable at max height without wobble.
- [ ] Cord routed out of walkway or covered.
- [ ] Brightness, CCT, scenes set.
- [ ] Voice assistant tested: “Set Living Room lamp to reading.”
- [ ] Physical backup control (Pico remote or wall switch) documented.
