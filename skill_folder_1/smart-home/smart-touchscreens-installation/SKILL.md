# Smart Touchscreen Installation Mastery

## Core concept
Wall-mounted touchscreens combine app, scene, camera, and intercom control. They are expensive. Installation is mostly mechanical: mount, power, network, and content layout.

## Product families
  - Brilliant Control Gen 1/Gen 2: replaces 2-gang wall plate; built-in Hue/Lutron integration.
  - Control4 / Elan / Savant panels: unified AV + lighting + shade.
  - iPad mounted in wall dock (dumb terminal approach).
  - Smart displays (Echo Show, Nest Hub Max): desk or wall shelf mount; not flush.

## Installation flow: Brilliant Control
1. Power off breaker; verify.
2. Confirm 2-gang box depth and clearance; Brilliant is thick.
3. Connect hot/neutral/ground per manual.
4. Mount; restore power.
5. Connect via Wi-Fi or Ethernet adapter if supported.
6. Pair in-app with ecosystems (Hue, Lutron, Sonos etc.).

## Mount height
- 44-56in center from finished floor is standard for accessibility.
- Avoid direct sunlight on screen (causes washout and ghost touches).
- Keep away from cooktop grease and bathroom steam.

## Reliability tips
- Use PoE or UPS-backed outlet.
- Create dedicated guest network if family uses panel for Wi-Fi.

## Teaching drills
- Drill 1: apprentice measures clearance behind proposed site and approves or rejects.
- Drill 2: mock pairing session; apprentice links Hue + Lutron in app and builds buttons.
- Drill 3: resilience test; apprentice explains what still works when internet drops.

## Post-install
- [ ] Mount level; bezel flush to wall.
- [ ] Network stable; no dropouts.
- [ ] Buttons map to scenes, not raw bulbs.
- [ ] Guest mode or restricted access configured.
