# Smart Sconces Installation Mastery

## Core concept
Sconces mount to walls and provide up/down wash lighting. Smart sconces come in two forms: (1) hardwired smart fixture, (2) adhesive-backed wall panels used as sconces.

## Installation flow: hardwired sconce
1. Turn breaker off; verify.
2. Remove old sconce; note wire positions.
3. Connect new smart sconce:
   - Hot, neutral, ground.
   - If dimmer required: keep switch always-on and set minimum in app.
4. Mount canopy and shade according to instructions.
5. Restore power; pair.

## Installation flow: Nanoleaf Elements / wall panel sconce
1. Clean surface thoroughly (alcohol wipe; dry).
2. Plan layout and orientation.
3. Press panels into place with included clips; test adhesion.
4. Pair with app via Thread/Zigbee/Wi-Fi bridge.
5. Use hue/brightness scenes to create wall-wash effect.

## Bath sconces
- GFCI required.
- Vapor-tight rated (IP44+).
- Anti-fog aids separately controlled if available.
- Keep switches outside shower wet zones unless rated.

## Coaching cues
- Explain “up and down light” concept: sconces should not cast glare directly at eye level in hallways.
- Teaching point: swapping sconces in historic houses often reveals knob-and-tube; stop and upgrade wiring.

## Post-install
- [ ] Sconces even brightness left/right.
- [ ] Wall/power switch left always-on.
- [ ] Scene activated suits task lighting or ambiance.
