# Smart Keypads Installation Mastery

## Core concept
A keypad replaces several standard switches with one programmable panel. Good for homeowners who want one-touch scenes without app dependency.

## Key product families
  - Lutron Palladiom / seeTouch / Alisse: wireless or wired; works with Caseta or RA3; supports 1-6+ buttons and LED backlighting.
  - Control4 Wireless Keypads: Zigbee; deep integration with Control4 ecosystem.
  - Brilliant Multi-Touch: replaces 2-gang wall plate; controls lights, shades, audio.

## Pros and cons versus remotes
- Pros: multiple buttons visible, no wallet hunting, can include fan, shade, and audio.
- Cons: larger gang box; harder retrofit; cost can exceed per-remote.

## Installation: Lutron Palladiom (example)
1. Turn breaker off.
2. Confirm box depth: Lutron keypads need deeper box (often double-gang or deeper single-gang).
3. Disconnect existing switch; Lutron keypad may need hot  + neutral.
4. Mount keypad; restore power.
5. Pair in Lutron app; name each button.

## Installation: wireless keypad (battery)
1. Mounting plate at 4-5ft height.
2. Keypad snaps or magnets onto plate.
3. Pair to hub.

## Programming
- Assign scenes to buttons with LED feedback:
  - Green = scene on
  - Off = scene off
  - Amber = partial
- Label clearly. Labels fade; use engraved or printed vinyl labels.
- Test all buttons from installation height (not while standing over the keypad).

## Teaching drills
- Drill 1: apprentice maps 5-button keypad to 5 scenes from user requirements.
- Drill 2: label design review; apprentice drafts labels that a guest can use without instruction.
- Drill 3: battery check procedure; apprentice demonstrates replacement on wireless unit.

## Post-install
- [ ] All buttons map to intended scenes.
- [ ] Labels legible from chair height.
- [ ] Backlight level adjustable.
- [ ] Local operation confirmed without phone.
