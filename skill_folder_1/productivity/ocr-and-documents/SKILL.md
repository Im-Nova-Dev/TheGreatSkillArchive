# OCR and Documents

## Core topics

1. OCR workflow
   - Image selection and preprocessing
   - Text extraction and cleanup

2. Layout
   - Tables and forms
   - Multi-column reading

3. Quality
   - Accuracy checks
   - Manual verification

## Teaching approach
- Extract text from one document.
- Rebuild one table from extraction.
- Compare two OCR results.
