# Notion

## Core topics

1. Workspace basics
   - Pages and blocks
   - Databases and properties
   - Relations and rollups

2. Workflows
   - Task trackers with databases
   - Meeting notes with linked tasks
   - Documentation structure

3. Collaboration
   - Sharing and permissions
   - Comments and mentions

## Teaching approach
- Build one project tracker in Notion.
- Convert one note structure to database.
- Compare page vs database task tracking.
