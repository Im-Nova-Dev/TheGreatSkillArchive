# 2026 OSS Bounty & Sponsorship Platform Survey: Exact Numbers

**Date:** 2026-06-05  
**Coverage:** Opire, Immunefi, Algora, Code4rena/Codecking, GitHub Issue Sponsors  
**Primary signal types:** Market demand + compensation range  

## Opire
- Active bounty range in early 2026: **$30–$2,780 USD** per GitHub issue.
- Typical languages: TypeScript, Rust, Python, C++.
- Platform fee: 4% + Stripe processing only on payout.
- Who pays: OSS maintainers, companies with dependency issues, individuals.
- Competitive environment: mostly medium-activity projects; top-tier DeFi protocols do not use Opire as primary channel.

## Immunefi
- Primary focus: smart-contract and Web3 protocols with TVL.
- Critical bounty payouts: common range $10k–$100k USDC for top protocols; some protocols raise ceiling to $500k in legacy-tier programs.
- Code4rena's prior clients are migrating here; competition for submissions is still high, but Immunefi's reputation system replaces contest-style ranking.
- Code4rena had $6M raised in 2023; payout records confirm multiple $50k–$100k critical bounties.

## Codecking / Codecking-style successors
- Total developer fund commitment: **$100 million** (announced).
- Operating model resembles Code4rena competition variant; payouts similar range but centralization is higher.

## Algora
- Focus: competitive open-source engineering hiring, often wrapping bounties around viral GitHub issues.
- Notable example in 2025–2026: $15k bounty posted to hire a Rust engineer via GitHub issue.
- Crowdsource bounties often tied to blockchain/crypto projects but not exclusively.

## GitHub Issue Sponsors + Open Collective
- Open Collective: thousands of community-funded projects with transparent budgets; typical per-project monthly spend in the hundreds of dollars.
- GitHub Sponsors: per-organization per-month tiers; highest earners often have full-time maintainers funded.
- ElevenLabs OSS Engineers Fund (Q1 2026): total $22k / 6 months, distributed via tiered monthly contributions ($300, $200/$100) to nominated OSS projects.

## Enterprise & Consortium Programs
- NGI Zero Commons Fund: €20M+ over multiple rounds; next application window Jan 15, 2026.
- Jenkins, Trusted Firmware (Arm), and Microsoft bug bounty programs: global scope; payouts vary; typically not accessible to individual solvers without reputation.
- HeroDevs: earns via enterprise support on end-of-life OSS projects rather than direct bug bounties; ideal for long-tail framework specialists (AngularJS, etc.).
