# AI Oversight + OSS Bounty Dual-Income

## When to Use

- You are facing junior-role compression, generic freelance rate decline, or AI displacement anxiety.
- Want near-term revenue without a long credentialing cycle.
- Willing to sell oversight, review, and governance rather than copy-paste implementation.

## Offer 1: AI-Code QA Retainer

**Problem to solve:** Companies adopted Copilot / Codex / Claude Code but lack architectural oversight, leading to drift, test gaps, and secrets leaks.

**Default terms:**
- Weekly architecture review + PR review for AI-generated code.
- Deliverables: architecture risk memo, PR review report, test coverage audit.
- Price: **$5k–$10k/mo** for 10–15 billed hours/week.
- Minimum term: 3 months.

**Pricing calibrations:**
- Company just raised Series A/B: anchor at $7.5k–$10k/mo.
- Team of 3–5 engineers with no dedicated tech lead: anchor at $6k–$8k/mo.
- EU client needing US-hour overlap: +15–25% premium.

**Qualification signals:**
- Public mentions of AI coding agents in job posts, repos, or LinkedIn.
- No dedicated code-review lead.
- Engineering headcount cuts but continued release cadence.

## Offer 2: Targeted OSS / Security Bounty Stack

**Goal:** Replace 1 low-value freelance gig/quarter with high-value bounty or sponsorship income.

**Rate expectation:** $174–$300+ effective hourly for focused fixes.

**Platform tier for 2026:**
- Low entry, good volume: Opire ($30–$2,780 general OSS bounties)
- High reward, security required: YesWeHack, Jenkins Bug Bounty, Microsoft MSRC
- Project-level grants: NGI Zero Commons Fund (next window Jan 15, 2026), ElevenLabs OSS Engineers Fund ($22k/6mo, tiered corporate contributions)

**Skill-value stack order:**
1. Rust / C++ systems work (thin labor supply)
2. TypeScript / Rust AI tooling
3. Go / Terraform infrastructure
4. Python security scripting

**Spillover monetization:**
- Each bounty win → 1-line pitch to companies using that repo as a dependency.
- Target convert rate: 1 retainer per 4–6 bounty wins.

## 90-Day Launch Sequence

**Week 1**
1. Register on Opire, create a 1-sentence "AI supervision" service page.
2. Pick one bounty-eligible repo you already know and attempt 1 smaller issue to build feedback loop.

**Week 2–3**
1. Identify 5 companies from recent "Who's Hiring" lists that mention agentic stacks.
2. Send 5 outreach DMs with case study: "AI Code QA review delivers X% fewer regressions."

**Week 4**
1. Submit 1 larger bounty ($1k+) in Rust/C++/TS AI infra.
2. If win, pitch 5 companies using the repo as a dependency for continued support.

**Month 2–3**
1. Target NGI Zero Commons or ElevenLabs-style sponsorship with project proposal.
2. Convert the strongest inbound sponsor into a $100–$300/month retainer.

## Pipeline Management

Maintain a startup.json-style record with:
- Source, bounty/opportunity title, reward, language, deadline, sponsor, payout probability.
- Weekly review: remove closed items, upscore hackathons and audit cycles.
- Track outreach-to-meeting ratio; if below 10%, narrow vertical to AI/security.

## Anti-Patterns

- Selling prompt engineering as a standalone service.
- Competing on price without adding review/governance layer.
- Chasing low-complexity implementation work that AI agents will commoditize first.

## Success Metrics

- First 90 days: ≥1 bounty or sponsorship ≥$500; 1 AI-review meeting booked.
- 6 months: ≥$1.5k/month total from bounty + sponsorship + retainer mix.
- 12 months: convert domain reputation into ≥$3k/month retainer base.
