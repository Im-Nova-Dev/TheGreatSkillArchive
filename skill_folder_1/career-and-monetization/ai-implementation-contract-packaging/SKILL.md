# AI Implementation Contract Packaging
*Skill type: career-and-monetization*

## When to Use This Skill
- You have Python/JS/TS/Cloud/MCP exposure and want contract/fractional AI work.
- You want to move from "AI Engineer" search profile to "AI Implementation Specialist" position.
- You need to price, scope, or sell AI integration retainers.

## 2. Positioning Statement Templates
Pick one and test with 5 prospects:

### A. Integration Contractor
> "I integrate LLMs, RAG pipelines, and AI agents into your existing stack — so your team gets production behavior, not another pilot."

### B. Retainer
> "Ongoing AI system execution: integrations, evals, monitoring, and handoff for $4k–$7k/mo."

### C. Fractional CTO / AI lead
> "Fractional AI engineering lead — I run your AI implementation sprints, hire, and get systems to prod."

## 3. Offer Anatomy

### Project Skeleton (1–4 week engagements)
| Phase | Deliverable |
|-------|-------------|
| Discovery | Workflow map + current-state stack |
| Integration | RAG / agent or automation deployed |
| Quality | Eval framework + red-team runbook |
| Handoff | Runbook + knowledge transfer session |

### Retainer Skeleton (monthly)
| Deliverable | Frequency |
|-------------|-----------|
| Integrations / evals / fixes | Weekly |
| Status + roadmap review | Weekly or biweekly |
| On-call review window | Async + 2h/week burst |

### Price Brackets (2026 evidence-based)
- Small: $3k–$7k project
- Medium: $10k–$25k project
- Large: $25k–$50k project
- Retainer: $3k–$8k/mo
- Hourly (independent): $75–$180/hr (NO-commissions), $40–$100/hr staffing-style
- Fractional CTO: $6k–$12k/mo

## 5. Conversion Artifacts to Publish
Publish *one* of these within 2 weeks:

1. **Case study one-pager** — End-to-end AI integration → metrics, architecture, outcomes.
2. **RAG/AI integration audit** — 3-5 question survey/checklist; offer 20-min audit as lead magnet.
3. **MCP integration starter** — Open-source or template repo showing server wiring.
4. **Retainer offer page** — Clear scope + price + SLA on personal site or LinkedIn.

---

## 6. Outreach Copy Patterns

### Cold DM to Buyer Trigger
> "Saw your posting about [goal]. Last month I shipped a similar [RAG/agent/automation] for [type hint]. Happy to share what worked. Open to a 10-min compare?"

### Proposal Opening
> "Deliverables in 30 days: 1) deploy [integration], 2) eval framework, 3) handoff package. Price: $X. Milestones: agree specs week 1, deploy week 2, validate/retain week 3-4."

### Retainer Pitch
> "Instead of hourly firefighting, I offer a fixed monthly retainer covering [list] with [SLA]. Most clients stay 3+ months once production is stable."

## 7. Skill Stack Checklist
Ensure you can demo 3+ of these:
- Python + OAuth/API integrations
- LangChain/LlamaIndex/agent frameworks
- MCP server wiring or Model Context Protocol
- RAG: vector DB + chunking + retrieval tuning
- Evals/observability: prompt eval, latency, cost logging
- Cloud deployment: AWS/GCP/Azure + container basics

If missing, prioritize MCP/agent wiring and evals — they are the biggest pricing lift in 2026 contracts.

## 10. Anti-Patterns
| Avoid | Instead |
|-------|---------|
| Calling yourself only "AI engineer" | Use "AI Implementation Specialist" + integration evidence |
| Hourly-only retainers | Mix project + retainer to increase LTV |
| Generic prompt-engineering offers | Specific integration domain + vertical |

| Modeling training as entry skill | Lead with deployment and operations |
