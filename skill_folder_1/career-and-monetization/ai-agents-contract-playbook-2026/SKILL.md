## Teaching Flow (4 Sessions × 90 min)

### Session 1: Market Positioning & Rate Anchoring (90 min)

**Objective:** Define your AI agent specialist niche and set evidence-based rate floor.

**Content:**
1. **Market Data Review** (15 min) — Walk through IT Jobs Watch + Upwork numbers
2. **Niche Selection Workshop** (30 min) — Pick 1 of 3 proven profiles:
   - **Profile A: AI Integration Engineer** — Embed LLMs/RAG into existing systems (Upwork +178%)
   - **Profile B: RAG/Agent Implementation Specialist** — Build production agent workflows (23% co-occurrence)
   - **Profile C: AI Agent Platform Engineer** — MLOps + LangGraph + Cloud (Azure 31%, AWS 24%, Bedrock 10%)
3. **Rate Card Creation** (30 min) — Build 3-tier pricing:
   - Discovery/Scope: £1,500 fixed (2-3 days)
   - Implementation: £650-750/day (contract) or £150-200/hr (freelance)
   - Retainer/Advisory: £3,000-5,000/mo (fractional CTO/AI lead)
4. **Homework:** Draft 3 case study one-pagers for portfolio

**Deliverable:** `rate-card.md` + `niche-statement.md`

### Session 2: Portfolio & Evidence Assembly (90 min)

**Objective:** Produce 2-3 portfolio pieces that signal "I ship AI agents to production."

**Content:**
1. **Case Study Template** (20 min) — Problem → Architecture → Tools → Metrics → Business Outcome
2. **Live Build Demo** (45 min) — Ship a minimal agent in session:
   - LangGraph + RAG + Observability (LangSmith/Logfire)
   - Deploy to Railway/Render/Fly.io (free tier)
   - Add eval harness (RAGAS or custom)
3. **GitHub Polish** (25 min) — README, architecture diagram, run instructions, eval results

**Deliverable:** 2 polished GitHub repos + case study PDFs

### Session 3: Contract Acquisition Mechanics (90 min)

**Objective:** Build repeatable pipeline for £600+/day contracts.

**Content:**
1. **Channel Strategy** (20 min) — Prioritized sourcing:
   - Tier 1: Direct outreach to UK SME CTOs (100-500 employees, series A/B)
   - Tier 2: Specialist recruiters (Oscar Technology, Burns Sheehan, Understanding Recruitment)
   - Tier 3: Upwork (AI Integration category), Toptal, Y Combinator job board
   - Tier 4: LinkedIn "Open to Work" + AI agent content posting
2. **Proposal System** (35 min) — Templates:
   - Discovery call script (15 min → scope → fixed-price discovery)
   - SOW template with AI-specific clauses (model costs, eval criteria, rollback)
   - Rate negotiation anchor: "Market median £600, my floor £650 given [specific evidence]"
3. **Pipeline Tracking** (20 min) — Notion/Airtable CRM with stages: Lead → Discovery → Proposal → Negotiation → Signed
4. **WFH Leverage** (15 min) — Position remote-first; cite £625 median + 13.6% YoY growth

**Deliverable:** `pipeline-template.notion` + `proposal-templates/` + `outreach-sequences/`

### Session 4: Delivery Excellence & Expansion (90 min)

**Objective:** Convert first contract into recurring revenue and referrals.

**Content:**
1. **Delivery Framework** (30 min) — 4-phase model:
   - Phase 1: Discovery & Architecture (week 1-2, fixed £3-5K)
   - Phase 2: MVP Agent (week 3-6, daily rate)
   - Phase 3: Eval/Obs/Hardening (week 7-8, daily rate)
   - Phase 4: Handoff + 30-day support (included)
2. **Upsell Patterns** (25 min) —
   - Multi-agent orchestration → +30-50% scope
   - Eval/observability stack → £2-3K fixed
   - Team training/workshop → £1.5-2K/day
   - Fractional AI lead retainer → £3-5K/mo
3. **Referral Engine** (20 min) — Ask template at delivery + quarterly check-ins
4. **Rate Escalation Calendar** (15 min) — +10% every 6 months; +20% on niche specialization

**Deliverable:** `delivery-checklist.md` + `upsell-menu.md` + `referral-email-templates.md`
